== Virtualization and Lustre == 

Virtualization enables the running of multiple machines on a single physical host and can provide an efficient and affordable way to learn the basics of Lustre or as a means to run simple sanity tests prior to moving to more expensive configurations. The following pages describe how different virtualization techniques on Linux can be used to create small Lustre file systems and the trade-offs each method offers.

* [https://wiki.lustre.org/KVM_Quick_Start_Guide Lustre with KVM]
* [https://wiki.lustre.org/Create_a_Virtual_HPC_Storage_Cluster_with_Vagrant Lustre with Vagrant]
* [https://wiki.lustre.org/Using_Xen_with_Lustre Lustre with Xen]
