=== Introduction === 

Wireshark is a free and open source packet analyzer. A Lustre plugin for Wireshark is available in the "lustre/contrib/wireshark" directory of the Lustre source tree.

* [https://www.wireshark.org/ Wireshark Site]
* [https://www.wireshark.org/docs/ Wireshark Documentation]

=== Building the Lustre Dissector ===

The Lustre Wireshark dissector plugin source is included in the Lustre source tree at <code>lustre/contrib/wireshark</code>. Refer to the README in that directory for build instructions. Wireshark 3.x or later is recommended for current Lustre protocol dissection, as older Wireshark versions may not support the plugin API used by the Lustre dissector.
