=== Introduction === 

The Lustre Monitoring Tool (LMT) is an open-source tool for the capture and display of Lustre file system activity. Data is stored in a MySQL database. Command line tools are available for live monitoring.

* [https://github.com/LLNL/lmt Main LMT Github]
* [http://github.com/LLNL/lmt/wiki LMT Wiki]

=== Obsolete ===

Unmaintained GUI (java-based) client tools are available which operate on data stored in the SQL database.  If you fork lmt-gui and take over maintenance, please update the URL here.  The google group is unused now, but some discussion is still relevant.

* [https://github.com/chaos/lmt-gui LMT GUI Github]
* [https://groups.google.com/forum/#!forum/lmt-discuss LMT Mailing List]
