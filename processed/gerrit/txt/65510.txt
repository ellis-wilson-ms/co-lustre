LU-11850 obd: fixes for Netlink stats

Bugs have been found in the Netlink stats code that
need to handled:

- We were accessing prev stats structure after being freed.
- Ensure stats lc_name never exceeded 32 bytes. Fix debugfs
  to handle lc_name larger then 25.
- Correct the elapsed-time calculation (was negative)
- Some misspellings.

Note since we allocate the needed size using cb_min_dump_alloc
the return values for the wrapper of nla_put() are not needed.
