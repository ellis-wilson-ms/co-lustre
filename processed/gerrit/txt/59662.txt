--- Commit Message ---
LU-19095 target: LNet callbacks for network changes

Patch adds functionality for target to be aware about
LNet changes in network configuration and be able to
notify MGS about these changes.
Patch does the following:
- LNet register/unregister/manage callbacks to notify
  about NID updates
- register callback for LNet to get updates if any target
  is started
- organize workqueue to process them
- organize list of targets watching updates
- put work in workqueue to handle update and send RPC in
  behalf of each target involved in
Patch doesn't implement RPC sending itself and MGS part.
