LU-19242 test: recovery-small 26b failure

The failure occured because as only a new RPC comes for a new export
its exp_deadline is set, calculated using the current AT, which may be
pretty large. A later setting a small AT_MAX will have effect on AT,
but not on exp_deadline. Thus, AT_MAX is to be set before mount.
