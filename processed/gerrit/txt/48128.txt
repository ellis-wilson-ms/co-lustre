LU-16035 kfilnd: Fix checking of peer field

The transaction debug and error print macros only check the
transaction peer field if it is NULL or not. There is a third state
the peer field can be which is error. Define the KFILND_TN_PEER_VALID
macro which can be used to verify the peer field is valid or not. It
accounts for all three states.
