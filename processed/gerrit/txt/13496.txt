--- Commit Message ---
LU-5565 osd-ldiskfs: handle nonexisting objects

In osd_xattr_get() check that the object exists before handling
XATTR_NAME_VERSION. In osd_xattr_set() add an early -ENOENT return if
the supplied object does not exist.
