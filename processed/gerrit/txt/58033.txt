LU-18700 tests: use /dev/urandom for sanityn test 36

The latest ZFS is using data compression by default, so reading
from /dev/zero results in files not consuming the expected amount of
space. Instead, read from /dev/urandom for ZFS to write files for
sanityn test 36.
