LU-8998 llapi: remove llapi_layout_comp_use_id()

The llapi_layout_comp_use_id() function is mostly the same as
llapi_layout_comp_use(), combine the two into a single function.

Add LLAPI_LAYOUT_COMP_USE_PREV to the llapi_layout_comp_use() API,
and test for same in llapi_layout_test.c.

Use LCME_ID_INVAL instead of comparing component ID to 0 directly.

Define LCME_FL_NOT_ID to make it clear that this flag is already
used instead of just a comment at the lcme_id declaration.  Rename
LCME_ID_NONE to LCME_ID_NOT_ID to clarify that this is a flag.

Reserve 4095 component IDs for future use.
