--- Commit Message ---
LU-19929 ec: Enable optimized erasure code for RISC-V

Enable the 64-bit optimized erasure coding path for RISC-V
architecture. RISC-V64 is 64-bit little-endian, which matches
the requirements for the optimized implementation.

This improves erasure coding performance on RISC-V systems.
Future optimizations may leverage RISC-V Vector Extension (RVV)
for even better performance.
