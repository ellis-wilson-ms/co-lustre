--- Commit Message ---
LU-19860 mdc: fix missing mutex_unlock in chlg_obd_get()

chlg_obd_get() acquires chlg_registered_dev_lock but fails
to release it when ced_obds list is empty, returning NULL
with the mutex still held. This causes a deadlock for any
subsequent caller attempting to acquire the same mutex.

Add the missing mutex_unlock() call before returning NULL.
