LU-19921 ptlrpc: add GSSIAM security context client

The GSSIAM Security Flavor is a new, lightweight sptlrpc
authentication mechanism designed to integrate Lustre
with external enterprise Identity and Access Management
(IAM) systems.

This patch adds the client side packing GSSIAM context
RPC process, including security context instantiation,
lookups, and buffer allocations.
