LU-11691 llapi: PFL layout size limit

The PFL code does not correctly check for overflowing the
layout size limits by putting many stripes in multiple
components.

Add a check that limits the total stripes in a PFL file to
1.5 times MAX_STRIPE_COUNT.  The extra 50% allows the last
comp of a PFL file to have MAX_STRIPE_COUNT, and makes it
possible to mirror a file using MAX_STRIPE_COUNT to a
smaller (but still large) number of OSTs.
