LU-19073 gss: add ASCII-encoded SSK key support

Add support for ASCII-encoded SSK keys to improve usability for
web-based key distribution and copy-paste operations.

- Add -a|--ascii option to lgss_sk for ASCII key generation
- Implement transparent format detection via "Lustre SSK v1.0" header
- Use base64url encoding for safe text transmission

ASCII keys can be more easily processed by user tools
(e.g. copy-paste from EMF SSK generator web interface or similar).

Sanity-sec test 101 is added to exercise ASCII SSK keys

Test-Parameters: trivial
Test-Parameters: testgroup=review-dne-selinux-ssk-part-2
Test-Parameters: testlist=sanity-sec env=ONLY=101,ONLY_MINUTES=30,SHARED_KEY=true
