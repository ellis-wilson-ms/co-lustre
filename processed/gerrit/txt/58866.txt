LU-18932 tcu: implement user command lfs trash move|mv

Implement user command "lfs trash move|mv $src $dst" to move a
file or a directory tree in Trash can to a destination path.

The file or path can be moved into destination path via POSIX
->rename() interface. However, it still needs to clear the
LUSTRE_UNRM_FL flag for the moved files.

Add sanityn/test_117j to verify this command works as expected.
