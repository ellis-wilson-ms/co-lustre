LU-12668 tests: add EC recovery tests

Add EC recovery tests covering different configurations,
I/O patterns, file sizes, failure modes, layouts, and
stress scenarios.

Tests added:
- test_40b-d: Recovery with 2+1, 2+2, 4+1 EC configs
- test_41a-e: Partial reads, spanning raid sets,
  different offsets, mmap, Direct I/O with recovery
- test_42a-d: Recovery with partial stripes, incomplete
  raid sets, PFL component boundaries
- test_43a-d: Single, multiple, excessive, and
  progressive OST failures
- test_44a-d: Error handling, parity vs data failures,
  read errors, invalid configurations
- test_46a-d: Recovery with 64K, 256K, 4M stripes and
  mixed stripe sizes in PFL layout
- test_47a-d: Multiple EC components in PFL, mixed
  EC/non-EC components, different stripe sizes per
  component, reads spanning component boundaries
- test_48a-d: Stale parity recovery, partial resync,
  overwrite scenarios
- test_49a-d: Concurrent reads, writes, mirror resync,
  and OST reactivation during recovery
- test_50a-b: Multi-mount recovery scenarios

All tests use 4+2 EC configuration with 8 OSTs.
