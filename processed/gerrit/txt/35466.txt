Patch Set 1:

patch gave us another ~25% performance improvements with PPN=24.

Without patch
Max Read:  21937.44 MiB/sec (23003.07 MB/sec)

With patch
Max Read:  27294.21 MiB/sec (28620.05 MB/sec)

Now, the biggest challenge is how to avoid performance drops at reclaiming page caches.

For instance, if file size is larger than client's memory (or max_cached_mb), it cause performance impact a lot.
if file size is less than max_cached_mb, we see better performance.
actual file size(1.35TB) Max Read:  27156.70 MiB/sec (28475.86 MB/sec)
1/2 of file size Max Read:  29831.10 MiB/sec (31280.18 MB/sec)
1/4 of file size Max Read:  33500.20 MiB/sec (35127.51 MB/sec)

Another evidence with large max_cached_mb is here. 
PPN=24, llite.*.max_cached_mb=90000
1/2 of file size Max Read:  35019.28 MiB/sec (36720.37 MB/sec)

no page cache reclaiming becouse of 90Gb x 10 client > 1.35TB / 2
