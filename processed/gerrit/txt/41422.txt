LU-14380 statahead: refine per process statahead control

This patch separates the statahead control out from the data
struct @ll_statahead_info, which is now used as per process
statahead contoller. In the meanwhile, a new data structure
@ll_sa_context is defined to store the information and context
about statahed for each directory inode.
