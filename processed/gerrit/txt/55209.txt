--- Commit Message ---
LU-17880 libcfs: set cpu_npartition in a multiple NUMA system.

Set cpu_npartition in a multiple NUMA system, otherwise it will
always use default partition to create ptlrpc service, which may
cause some performance issue if there are too many cores in each
partition.

With this patch, it will put 4 cores in each partition by default,
instead of assigning all cores from one NUMA in one partition.
