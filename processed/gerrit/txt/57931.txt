LU-17961 obdclass: fix compile error on Ubuntu24.04

e826e8bb77 causes build failures on Ubuntu24.04
(kernel 6.8.0-49-generic).
Because in kernel 6.7 and above, it has been changed from
```
struct group_info {
        atomic_t        usage;
        int             ngroups;
        kgid_t          gid[];
} __randomize_layout;
```
to
```
struct group_info {
        refcount_t      usage;
        int             ngroups;
        kgid_t          gid[];
} __randomize_layout;
```

Fixes: e826e8bb77 ("LU-17961 sec: support supplementary groups from client")
