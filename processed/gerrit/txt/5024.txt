--- Commit Message ---
LU-2603 obdclass: remove a race between ioctl and release dev

class_handle_ioctl can touch an obd_device which is now being released
by class_release_dev. And class_release_dev can release an obd_device
which is now being handled by class_handle_ioctl.

This patch removes the race by adding proper locking logic
