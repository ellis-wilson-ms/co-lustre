LU-19167 lnet: add corrupt fault injection

Add a new fault injection mechanism that can be added/removed and
listed using "lctl net_corrup_add/del/list" commands. It randomly
corrupts network message payloads at specified offsets/sizes,
enabling testing of error handling and data integrity mechanisms.
Example Usage:
lctl net_corrupt_add -s <src> -d <dst> -r <rate> -f <offset> -z <size>
lctl net_corrupt_list
lctl net_corrupt_del -a
