--- Commit Message ---
LU-17044 ptlrpc: convert TBF module params to tunables

Convert tbf_jobid_cache_size, tbf_rate, and tbf_depth from module
parameters to regular tunables so they can be configured globally from
the MGS via lctl conf_param.

Each parameter is now defined as a global variable in obdclass with a
sysfs attribute under /sys/fs/lustre/ and a PARAM_* define recognized
by mgs_write_log_sys(). The deprecated module parameters are retained
for backward compatibility.
