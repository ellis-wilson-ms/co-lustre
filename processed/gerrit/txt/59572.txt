--- Commit Message ---
LU-13643 flr: add immediate mirror extend/delete/split

Ensure that lfs mirror delete/split/extend operations work
correctly on files with the immediate mirror flag set. This
required fixing the mirror mode detection and validation
logic to properly handle mirror delete operations.

Changes:
- Extended mirror_mode detection to include SO_MIRROR_DELETE
  and SO_MIRROR_SPLIT operations so component IDs are allowed
- Fixed validation logic to allow component IDs with mirror
  delete operations (removed restriction that prevented -I
  flag usage with mirror operations)

Test-Parameters: trivial
