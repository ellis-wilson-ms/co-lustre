LU-19296 changelog: Add user-specific changelog filtering

This patch implements changelog filtering based on username and masks.
- Define a new API llapi_changelog_start_user() to get changelog
  records based on username and mask.
- Use chlg_ioctl() with OBD_IOC_CHANGELOG_FILTER to set changelog
  filter and call new function mdc_changelog_get_user_info() to
  get user info from MDT by KEY_CHANGELOG_USER MDS_GET_INFO RPC.
- Add options --user and --mask to "lfs changelog" command to
  support user-specific changelog reading.
- Add man page llapi_changelog_start_user.3 redirect.
- Update llapi_changelog_start.3 and lfs-changelog.1 for changes.
- Add sanity.sh/test_160v to verify this patch.
- Add missing man page for llapi_convert_str2mask().
- Fix up related man pages to reference new function, add examples.
