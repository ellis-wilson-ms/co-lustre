LU-19929 contrib: Add RISC-V to crash debug tools

Add RISC-V64 architecture support to crash module building script.
This enables debugging support for Lustre on RISC-V systems.

The checkstack.pl script already has RISC-V support (added by
Wadim Mueller), but the crash module builder needed updating.
