--- Commit Message ---
LU-930 doc: update lctl-set_param with more details

Be more specific about the semantics of which parameters the
"lctl set_param" command will be affecting when it is run with
different arguments.  It wasn't very clear from the previous
text that using '-P' on the MGS was the primary way of setting
the parameters for the whole filesystem.

Add a section about bash-completion so that users can learn
about this very useful functionality for set_param, get_param,
list_param, and lfs find.

Fix some minor checkpatch-man warnings.
