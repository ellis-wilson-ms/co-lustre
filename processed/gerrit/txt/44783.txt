LU-14974: improve the repartition of requests between copytools

Distribute requests from unsupported archive_ids to coopytools with
archive_id 0.

Since we now distribute requests evenly between copytools for each
archive_id, if a request with a new archive_id comes in, it will
be placed in a single list and subsequent requests with the same
archive_id will be put there.

If we have several copytools supporting all archive_ids (e.g.
archive_id 0), we can distribute the requests between them
instead.
