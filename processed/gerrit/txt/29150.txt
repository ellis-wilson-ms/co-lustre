--- Commit Message ---
LU-9789 obdfilter: add cluster ID to JobID

Use nodemap to specify the cluster where client
is sending job data from.

If a nodemap exists, jobids will be saved in the
form "nodename-jobid" where jobid is the value
sent from the client and nodename is the value
calculated via the nodemap.

If the client is not part of a nodemap, only
jobid will be saved.
