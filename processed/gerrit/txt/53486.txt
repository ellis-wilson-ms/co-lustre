--- Commit Message ---
LU-16502 lutf: improving logging

Print logging messages in real time rather
than all at once at the end. Output logging
to both a file and stdout.

Many messages are marked at too high of a log level.
Every message has been downgraded to DEBUG log level.
A few select messages have been made INFO. The
default log level has been set to INFO.

These changes will make it easier to get live logging
when running test suites.
