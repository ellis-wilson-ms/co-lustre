LU-5614 build: use %kernel_module_package in rpm spec

This patch is for Morrone's build testing only.  Not meant to land.

Replace module packages definitions with a call to the vendor defined
%kernel_module_package macro.

The files that contain the file listing for each %kernel_module_package
call are called out with their own Source<N>: lines in the spec file
to make them available to the spec file in advance of %prep unpacking
the tarball.
