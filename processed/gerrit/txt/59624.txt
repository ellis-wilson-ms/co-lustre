--- Commit Message ---
LU-13643 llite: add mirror detection to llite

This patch introduces a new ll_get_mirror_info() function that:

- Provides a clean interface taking inode as input and returning
  mirror count and immediate flag via pointers

The function is called from ll_file_io_generic() to detect
mirror properties for each file I/O operation. Debug output
is added to show the detected mirror count and immediate
flag status.

TODO: This function uses XATTR_SIZE_MAX buffer and needs to be
made smarter.
