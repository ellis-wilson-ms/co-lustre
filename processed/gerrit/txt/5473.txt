LU-2808 osp: tolerate truncates on striped objects

If a client lacks lsm for a file then it may send setattr from
truncate to the MDS even if the file already has stripes.  Tolerate
this in osp_declare_attr_set()/osp_object_assign_fid() by not LBUGing.
