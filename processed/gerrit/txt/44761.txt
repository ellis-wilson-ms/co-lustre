LU-14969 llite: DIO fall back to buffered

Allow unaligned DIO by falling back to buffered I/O.
The kernel handles this automatically when we return 0 in
direct I/O.

We do not allow unaligned AIO because AIO is expecting DIO
behavior and it is extra work to make fallback work for
AIO.
