--- Commit Message ---
LU-6960 utils: erase mgsnode param with tunefs.lustre

When we need to change MGS node, "tunefs.lustre --mgsnode" would only
append to existing mgsnode params.

This patch add a "--erase-mgsnode" parameter to wipe existing mgsnode
and append mgsnode parameters afterwards.
