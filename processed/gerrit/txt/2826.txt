--- Commit Message ---
LU-549 xattr: packaged xattr in single RPC reply

Currently, MDS returns ACL together with other object's attributes
when replies getattr/lookup RPC. As for stating directory object,
client will send additional RPC for default ACL.

As an improvement, MDS will return packaged xattr to contain both
posix ACL and default ACL (for directory). For remote client, the
remote permission is also returned through packaged xattr.

The packaged xattr framework supports to return more other xattrs
by single RPC.
