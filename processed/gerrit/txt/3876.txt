--- Commit Message ---
LU-1823 kernel: enable kernel debug options

This patch enables the following debug options:
CONFIG_DEBUG_SLAB
CONFIG_DEBUG_SLAB_LEAK
CONFIG_DEBUG_SPINLOCK
CONFIG_DEBUG_SPINLOCK_SLEEP
CONFIG_DEBUG_STACK_USAGE

It also adds a check_slab_corruption() function into the
test framework to detect the slab corruption error.

Test-Parameters: envdefinitions=SLOW=yes testlist=sanity
