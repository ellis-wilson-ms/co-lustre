LU-10499 llite: Add new DLM mode PCCRO for readonly PCC

Currently Readonly mode of PCC uses grouplock to protect the data
from being written or even normal read from other clients, even
if there is still a copy on the original OSTs.
This patch introduces a new kind of LDLM lock mode PCCRO, which is
compatible with PR lock mode, so when the file is PCC-fetched and
attached on a client, the client will obtain PCCRO lock on the
file which allows the read access for the file data on original
OSTs from other clients, but no write lock is allowed.
