--- Message (Andreas Dilger, 2011-11-10 12:18:24.000000000) ---
Patch Set 1: I would prefer that you didn't submit this

I don't think it should move into /proc/sys, since we have generally stopped locating new variables there.  Instead, it can be moved up one level to /proc/fs/lustre.

At some point, I'd like to duplicate all of the values in /proc/sys over to /proc/fs/lustre, and later remove them from /proc/sys entirely.  This would at least allow the Lustre code to avoid having to deal with two different types of system files internally.
