Patch Set 4: Verified

Functionality works as expected. HSM subdirectory structure is visible on MDT mount, even if hsm_control is set to stopped. In addition, parameters can be changed and saved persistently with "set_param -P".

set_param -P works when hsm_control == "stopped":

[root@m64-1 ~]# df -ht lustre
Filesystem            Size  Used Avail Use% Mounted on
/dev/sdb             1008M   35M  923M   4% /lustre/mgs
/dev/sdc              7.5G  446M  6.6G   7% /lustre/scratch/mdt0
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm_control
mdt.scratch-MDT0000.hsm_control=stopped
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=1
[root@m64-1 ~]# lctl set_param -P mdt.scratch-MDT*.hsm.default_archive_id=2
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=2
[root@m64-1 ~]# umount /lustre/scratch/mdt0
[root@m64-1 ~]# umount /lustre/mgs
[root@m64-1 ~]# lustre_rmmod 
[root@m64-1 ~]# lsmod | grep lnet
[root@m64-1 ~]# lsmod | grep lustre
[root@m64-1 ~]# mount -t lustre /dev/sdb /lustre/mgs
[root@m64-1 ~]# mount -t lustre /dev/sdc /lustre/scratch/mdt0
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=2


set_param -P works when hsm_control == "enabled":

[root@m64-1 ~]# lctl set_param -P mdt.scratch-MDT*.hsm_control=enabled
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm_control
mdt.scratch-MDT0000.hsm_control=enabled
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=2
[root@m64-1 ~]# lctl set_param -P mdt.scratch-MDT*.hsm.default_archive_id=3
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=3
[root@m64-1 ~]# umount /lustre/scratch/mdt0
[root@m64-1 ~]# umount /lustre/mgs
[root@m64-1 ~]# lustre_rmmod 
[root@m64-1 ~]# mount -t lustre /dev/sdb /lustre/mgs
[root@m64-1 ~]# mount -t lustre /dev/sdc /lustre/scratch/mdt0
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm_control
mdt.scratch-MDT0000.hsm_control=enabled
[root@m64-1 ~]# lctl get_param mdt.scratch-MDT*.hsm.default_archive_id
mdt.scratch-MDT0000.hsm.default_archive_id=3


conf_param does not work under any circumstances:

[root@m64-1 ~]# lctl conf_param scratch-MDT*.mdt.hsm.default_archive_id=5
