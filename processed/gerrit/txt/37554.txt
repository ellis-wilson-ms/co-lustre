Note that if you find some wonky issues with QUOTA with this patch is
is due to the bug in the ext4-data-in-dirent.patch
In ext4.h it *should* be:
  #define EXT4_MOUNT_DIRDATA	0x60000	/* Data in directory entries*/

But the current patch is using:
  #define EXT4_MOUNT_DIRDATA	0x40000	/* Data in directory entries*/

Will be pushing a fix RSN
