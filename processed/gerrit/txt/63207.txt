LU-19179 build: remove RHEL7 handling of fscrypt

With the removal of RHEL7 we no longer need the LC_IS_ENCRYPTED
autoconf test so remove it. Rework the autoconf handling for
fscrypt. We handle the case of a kernel turning off fscrypt and
the case of building only the tools for the Linux native client.
