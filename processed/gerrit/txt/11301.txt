LU-4701 osp: OSP object attributes cache cleanup

1) Some adjustment for OSP object attributes cache maintainig to make
   the logic more clear and reasonable.

     osp_declare_attr_set().

     in the osp_declare_xattr_set().
1.3) Drop the cached extended attribute in osp_xattr_del().

2) Typo fixing and code cleanup.
