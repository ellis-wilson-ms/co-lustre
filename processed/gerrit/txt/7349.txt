--- Commit Message ---
LU-3727 nfs: Fix ll_get_parent() LBUG caused by permission

When ll_get_parent() is trying to get the attributes of parent
directory, it will hit LBUG if NFS daemon user does not have the
execute permission of the child directory.

This patch fixes the problem by set the UID/GID to root for
ll_get_parent().
