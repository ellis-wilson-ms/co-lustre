--- Commit Message ---
LU-12348 llite: enable "flock" mount option by default

By default, flock()/lockf()/fcntl() functions that try to lock files
would return failure because "-o flock" option is not enabled by
default. This would cause failure to applications which are being
migrated from other file systems to Lustre.

This patch enables "flock" option by default.
