LU-13353 quota: quickly determine if quota exceeded

Use llapi_quotactl() to retrieve the lqe_edquot
flag which indicates whether or not some entity
has exceeded a quota. This can be done without
querying the QSDs which can improve performance.
Add -e flag to lfs-quota to get this information.

Test-Parameters: fortestonly testlist=sanity-quota
