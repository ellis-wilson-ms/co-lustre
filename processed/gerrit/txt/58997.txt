LU-18456 tcu: update LinkEA when move file into Trash Can

Update LinkEA data accordingly when moving a file into Trash Can.
As rename() operation also needs to update LinkEA, to avoid
conflict, we define a seperate LinkEA data and buffer for the
update of LinkEA of the Trash Object.

Add a test case sanityn/test_117n to verify it works as expected.
