LU-1606 api: Package posix-wordsize.h for user space.

Include the libcfs/posix/posix-wordsize.h in the packaging
for user space, to appear as $(includedir)/libcfs/posix/posix-wordsize.h.

lustre_user.h then includes posix-wordsize.h in user space only,
which defines LPU64 and its other variants, which have crept into
the user space headers over time, making lustre_user.h unbuildable
in user space on Linux.
