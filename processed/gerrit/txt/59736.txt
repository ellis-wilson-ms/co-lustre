--- Commit Message ---
LU-13643 flr: store per-mirror errors in cl_sub_dio_end

Store negative return codes (errors) in the mirror-specific error
array when sub-DIO operations complete, but only if the current
slot value is 0 (no previous error stored).

This enables FLR immediate mirror writes to track per-mirror
I/O results for better error handling and mirror management.

These errors will be communicated to the server in the LVB for
the AW lock.
