LU-18818 utils: use libext2fs in ldiskfs_write_ldd()

Replace temporary filesystem mounting with direct libext2fs
operations in ldiskfs_write_ldd().

The function now uses libext2fs to:
- Open filesystem with write access using ext2fs_open()
- Create CONFIGS directory using ext2fs_new_inode() and ext2fs_link()
- Create mountdata file using ext2fs_new_inode() and ext2fs_link()
- Write LDD data using ext2fs_file_open() and ext2fs_file_write()
- Ensure all filesystem changes are persisted using ext2fs_flush()

Also removes the e2label system call in ldisk_read_ldd.
