--- Commit Message ---
LU-6081 lib: add a full fledge JSON parser to liblustreapi

The migration data will be passed between lfs and the copytool in JSON
format. However Lustre has no decoder for JSON. liblustreapi has a
JSON emitter but not a parser. cYAML also exists but is too limited;
it cannot parse JSON nor handle arrays.

Import the CCAN json parser/generator as-is
(http://ccodearchive.net/info/json.html).
