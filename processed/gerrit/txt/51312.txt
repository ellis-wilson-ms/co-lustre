LU-16893 libcfs: cfs_force_sig deadklock on send_sig

When force_sig HAVE_FORCE_SIG_WITH_TASK is not available
cfs_force_sig uses send_sig to duplicate the functionality,
however send_sig also locks siglock, which can trigger
a deadlock and an rcu stall.

cfs_force_sig(sig, task)
  -> spin_lock_irqsave(&task->sighand->siglock, flags);
    -> send_sig(.., task,..)
     -> send_sig_info(.., task)
       -> do_send_sig_info(..., task)
         -> lock_task_sighand(task)
           -> __lock_task_sighand(task, flags)
              {
                  sighand = rcu_dereference(task->sighand);
                  spin_lock_irqsave(&sighand->siglock, *flags);
                  ...
              } 

cfs_force_sig must drop the lock on siglock before calling
send_sig.
