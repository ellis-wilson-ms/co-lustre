LU-9533 test: update how to get version in obdfilter-survey

The routine get_lustre_version() in lustre-iokit/obdfilter-survey
expects the string "lustre" in the return value of 'lctl get_param
-n version'. Unfortunately, "lustre" is no longer returned as part
of the version string.

Remove the code looking for the string "lustre" from the
return value of 'lctl get_param -n version'.
