EX-8038 csdc: reserve compression info in LMA EA/filter_fid

Store compression information as part of layout information in
OST-object's LMA EA, filter_fid, and PFID, lfsck could use it to
recover the MDT-object layout EA from orphan OST-object(s).

For the purpose of compatibility with the existing OST-objects and
wire data protocol, this patch append the compression information in
corresponding structures, and rename the existing layout member
to layout_base, and name the compression info member as layout_compr.
The structures have been changed is as such:

lu_orphan_rec_v3        orphan OST-object
obdo                    general attribute in wire data to server
filter_fid              self/parent FID stored on server objects
