LU-18456 mdd: server-side fid2path using by recycle bin

This patch implements server-side fid2path().
It is similar to the client-side implementation, MDD layer issues
the fid2path() call to LOD layer which determines whether obtain
path locally via mdt_fid2path_local() for local FID, or direct to
the next OSP layer to issue an GETINFO RPC for remote FID.
