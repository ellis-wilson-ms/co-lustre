(1 comment)

Looks like there ought to be a 'snprintf() -> scnprintf()' cleanup as there are a few places where the snprintf() pattern is being used.

We can write a trivial wrapper around *snprintf() to get *scnprintf():

int scnprintf(...,sz,...) {
    int rc = snprintf(..., sz,...);
    return (sz > rc) ? rc : sz;
}
