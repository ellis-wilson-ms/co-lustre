--- Commit Message ---
LU-7357 lmv: add layout lock to striped directory

During unlink striped directory, master MDT needs to
invalidate the layout on the client cache, so client
will try to refresh the layout, and then be notified
the object has been unlinked.

1. Client will set Layout to be valid, if getattr or
open return layout lock.

2. If client check layout has been invalid, it needs to
refresh(enqueue the layout lock) the layout before using
it.

3. Add a few tests to check the process.
