--- Commit Message ---
LU-6931 kernel: fix kernel to be compiled with o0

Gcov requires code to be compiled without optimization (o0).
Lustre get some parts of kernel's code to compile ldiskfs.
There is kernel function that is used by Lustre and can't be
compiled without optimization.

This patch fix change operand type, so this function
can be compiled without optimization.
