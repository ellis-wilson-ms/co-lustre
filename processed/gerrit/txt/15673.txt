--- Commit Message ---
LU-7897 utils: an infrastructure to benchmark and test OSD

using requests in OUT format. with this infrastructure it's
possible to develop synthetic benchmarks to learn performance
of specific OSD updates and combinations. also, it's possible
to build funcional tests to automatically verify correctness
of OSD API and implementations (ldiskfs-osd, zfs-osd, osp,,,)

 - run with OSP
