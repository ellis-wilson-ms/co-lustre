--- Commit Message ---
LU-18222 quota: add LQA lfs setquota/quota commands

Add --lqa option to lfs setquota/quota to set and read quotas for
requested LQA. To specify the quota type, this option should be used
with one of the short keys -U/-G/-P. Example:

 lfs setquota -U --lqa lqa_name -B100M -b200M /mnt/lustre
 lfs quota -P --lqa lqa_name /mnt/lustre

lfs quota -U/-G/-P --lqa doesn't return LQA's granted space for
specified quota type. This will be implemented later in a separate
patch.
