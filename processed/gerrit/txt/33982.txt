--- Commit Message ---
LU-10092 pcc: Add RW-PCC support for non-root user

When auto RW-PCC cache for a newly created file, the client needs
to set the file with HSM exists, archived, released state.
However, non-root users are forbidden to set these flags which are
not defined in HSM_USER_MASK.
This patch sloves this problem by allowing the non-root user to
set these flags for auto RW-PCC caching newly create files.
