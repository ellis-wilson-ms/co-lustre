--- Commit Message ---
LU-19129 utils: message for lctl-nodemap on non-MGS node

In case an lctl-nodemap command fails because it was launched on the
wrong node, the error message should explicitly inform the user that
the command must be executed on the MGS node.
Make nodemap handling code in the kernel return -ENXIO in this case,
so that userspace can print the appropriate message.
