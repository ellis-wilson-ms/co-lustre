--- Commit Message ---
LU-19157 nodemap: allow ban list on default nodemap

Allow the default nodemap to have a ban list. As the default nodemap
does not have any NID ranges, the constraint on its ban list is that
it cannot conflict with any NID range defined on any other nodemap.

Enhance sanity-sec test_81a to test ban list on the default nodemap.
