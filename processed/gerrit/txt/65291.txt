LU-20048 tcu: encryption XATTR handling for virtual striped dir

This patch fixes the error in encryption XATTR handling for
virtual Trash striped directory which is mainly used by the
virtual directory  ".Trash".
With this fix, users can access the encrypted files in the virtual
Trash directory via ".Trash" under a striped directory.

Add a new sanityn/test_118x to verify the fix.
