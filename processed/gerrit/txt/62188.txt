LU-19469 utils: set/clear LUSTRE_SPARSE_FL on a regular file

This patch adds a new LUSTRE_SPARSE_FL to indicate that the file
marked with this flag is sparse.
Add a command to set/clear the flag:
	$LFS chattr [+|-s] [+|-sparse] $FILE ...
Add a command to display the FSFLAGS on a file:
	$FLS lsattr $FILE...

Add sanity/test_855g to test this flag.
