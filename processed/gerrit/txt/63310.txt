LU-19772 build: Improve o2iblnd path detection

config/lustre-lnet.m4 provides a unified way to detect o2iblnd
path but it cannot retrieve o2iblnd path in some cases (e.g
mlnx-ofa_kernel-source and mlnx-ofa_kernel-dksm) so improve
it by adding a default path.

Also remove the duplicated check in lustre-dkms_pre-build.sh.
