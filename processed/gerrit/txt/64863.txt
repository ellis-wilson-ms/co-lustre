--- Commit Message ---
LU-19768 mdd: add changelog users netlink

Add a netlink handler for streaming changelog index and
registered changelog users from all MDD targets. The data is
collected via a callback registered by mdd.ko into obdclass.ko
(changelog_nl_collect_fn) to avoid a module dependency.

Each entry carries the MDT device name, current changelog index,
and a list of users with their last-consumed index, idle time,
and event mask.
