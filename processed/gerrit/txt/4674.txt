LU-2311 mdd: clean up orphan code style

Rename externally-visible "orph_*" functions to "mdd_orphan_*":
    orph_index_{name,init,fini} -> mdd_orphan_index_{name, init,fini}
    orph_declare_index_insert ()-> mdd_orphan_declare_index_insert()
    orph_declare_index_delete() -> mdd_orphan_declare_index_delete()
Remove extraneous "__" prefix from mdd_orphan_{add,del,cleanup}()
Remove extraneous function wrappers and call underlying code directly:
    __mdd_orphan_{add,del,cleanup}()
    mdd_orphan_{insert,delete}_obj()
    mdd_orphan_ref_{add,del}()
Remove unnecessary "rc" and "is_dir" variables in mdd_finish_unlink()
Fix tab indenting around any modified code.
