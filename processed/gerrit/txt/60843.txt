LU-18667 kernel: RHEL 10.0 server support

This patch makes changes to support RHEL 10.0 release
with kernel 6.12.0-55.43.1.el10_0 for Lustre server.

It also adds %changelog section into lustre.spec.in to
fix the following RPM build warning:
  source_date_epoch_from_changelog set but %changelog is missing
