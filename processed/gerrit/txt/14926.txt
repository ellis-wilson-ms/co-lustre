--- Commit Message ---
LU-6584 osd: add more debug to check buffer size

Patch adds more debug to address LU-6584 issue with buffers
larger than page.

Adds ofd_check_remotebuf() to check buffer received from client,
return -EPROTO to client if something wrong happens during the
network transfer.

Adds tgt_check_localbuf() to check adjusted buffer to be used for
read, return -EIO to client if the local buffer is insane.

These two functions will print the whole remote/local buffer with
D_BUFFS/buffs debug message level.
