--- Commit Message ---
LU-15938 llog: Fix chunk re-read case in llog_process_thread

The patch 1a24dcd changed the return code (-EIO -> -EBADR) of
llog_osd_next_block() when it reachs the end of a llog file.
This break the re-read logic of the last chunk if new record had been
added during the first read.

Fixes: 1a24dcd ("LU-15938 lod: prevent endless retry in recovery thread")
