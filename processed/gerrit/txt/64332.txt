LU-14708 tests: improve recovery-small test_155

This test is failing intermittently in Janitor testing (though not
in Autotest), but it is difficult to see why it fails.

Write the 'ls' output to temporary files and use 'diff' to compare
them, so that it is easier to see what the differences are, instead
of just comparing the output strings and printing both of them in
the error message.

It appears it fails due to a previous subtest file changing:
- -rw-r--r-- 1 root root     512 Mar  2 18:52 f131.recovery-small
+ -rw-r--r-- 1 root root       0 Mar  2 18:52 f131.recovery-small

Flush client before the test to avoid any issues with old files.
