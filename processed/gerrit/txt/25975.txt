LU-9211 tests: fix ost-pool non-zero exit status

Tests ost-pools/7[abc] passed but return exit code != 0.
ost_pools_init and cleanup_pools called at the start and
end of the ost-pool test case execution, when do_facet
called to list non-existing pool, it returns with exit
status 2.

As this is not test specific error, fix to redirect this
error to /dev/null and return status 0.
