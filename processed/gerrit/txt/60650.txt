--- Commit Message ---
LU-0000 utils: allow stripe size on EC parity components

Drop the parity guard in layout_stripe_size_set() so parity
components can inherit and accept stripe-size values, while
still forbidding it for foreign layout pattern.

tests: sanity-ec: assert parity stripe size when -S is used

Add optional expected stripe size to verify_ec_layout() and
use it in 3c (4+2, -S 1M) to check both data and parity stripe
sizes via getstripe.
