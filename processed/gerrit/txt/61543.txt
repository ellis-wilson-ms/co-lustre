LU-19424 lmv: expose qos_rr_index parameter

Add a new lmv.*.qos_rr_index parameter that prints or sets
the current qos_rr_index value modulo the number of MDTs.

This can be used to determine if the round-robin index is
being initialized as expected, or can be set to ensure a
uniform distribution of clients across MDTs for testing.
