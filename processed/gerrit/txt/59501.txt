LU-19074 ec: add loop unrolling for common source counts

Furhter optimizations from Claude - significant speedup
from loop unrolling.

Add specialized loop unrolling optimizations for common erasure
coding configurations (4, 6, and 8 data disks) to eliminate
loop overhead and improve register utilization.

The optimization unrolls the inner source loop for the most
common EC configurations, hoisting table pointer calculations
and reducing branch overhead in the critical encoding path.

Performance improvements from loop unrolling:
- Small blocks (4KB, 4+2 EC): 1.66x speedup, 121->201 MB/s
- Medium blocks (64KB, 4+2 EC): 1.66x speedup, 120->199 MB/s
- Large blocks (1MB, 8+4 EC): 3.09x speedup, 11->34 MB/s

Combined with previous addressing optimizations, this provides
substantial performance improvements for Lustre erasure coding
operations across common configurations.
