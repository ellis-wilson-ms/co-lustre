LU-19303 tcu: Handle symlinks upon the last unlink

Move symlinks into Trash Can upon the last unlink just like
regular files.
It needs to do special handling to clear LUSTRE_UNRM_FL on symlink
as the normal open() syscall for the IOCTL FS_IOC_SETFLAGS will
open the target file or directory the symlink points.
Thus we make a specical LL_IOC_TRASH (which is like LL_IOC_PROJECT
ioctl) on the parent directory to set/get/clear the LUSTRE_UNRM_FL
on files.

Add test cases sanityn/test_117y{a,b,c}.
