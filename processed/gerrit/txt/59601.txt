LU-12345 mdt: add server-side IT_WRITE_LEASE support

This patch implements server-side support for write lease
intents in the MDT layer. The server now properly handles
IT_WRITE_LEASE intents from clients and grants appropriate
write lease locks.

The implementation adds:
- IT_WRITE_LEASE case in mdt_intent_opc() routing
- mdt_intent_write_lease() handler function
- Proper lock acquisition with MDS_INODELOCK_WRITE_LEASE
- Request format and disposition handling

When a client sends an IT_WRITE_LEASE intent, the server
now grants an exclusive write lease lock using the
MDS_INODELOCK_WRITE_LEASE lock bit, providing coordination
between multiple clients for write operations.

This completes the write lease implementation started in
the previous client-side patch, enabling full write lease
functionality in Lustre.
