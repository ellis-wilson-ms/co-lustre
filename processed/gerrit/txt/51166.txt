LU-14639 tests: remove disable-lru-resize check

A number of tests check for the client/server connection
not having lru-resize support.  It has been some years
since we had servers without lru-resize support by default,
and the previous patch removes the ability to compile out
lru-resize support, so we can remove these checks.

Trivial should be OK since that still runs sanity.
