LU-11170 tests: add delay sync for sanity test 415

For a Lustre file system with ZFS targets and DNE configured,
renaming files, sanity test 415, takes more time than expected.
We should set delayed sync for ZFS for this test to see if
this has an impact on the amount of time to rename files.

Test-Parameters: trivial mdscount=2 mdtcount=2 fstype=zfs testlist=sanity
