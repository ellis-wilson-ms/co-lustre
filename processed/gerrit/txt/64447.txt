LU-19859 scripts: fix bash path for developer scripts

Hard-coding /usr/bin/bash prevents editing files on MacOS with
the Git commit hooks enabled, since MacOS still uses /bin/bash.
Change them over to "/usr/bin/env bash" so they run everywhere.
