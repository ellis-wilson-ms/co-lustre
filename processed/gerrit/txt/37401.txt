--- Commit Message ---
LU-11510 lfs: create a comp file correctly when migrating

In function migrate_open_files(), "layout" pointer should be used
instead of "param" pointer to tell whether a comp file should be
created or not, because "param" pointer is always not null and thus
the comp file will never be created.
