--- Commit Message ---
LU-9441 pltrpc: don't wakeup on 1 second intervals

If there are no requests on the set there is no need
to wake up every second. The thread is woken up when a
request is added to the set or when the STOP bit is set.
So it is sufficient to only wake up when there are requests
on the set to worry about.
