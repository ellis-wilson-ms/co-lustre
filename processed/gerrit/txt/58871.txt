LU-00000 obdfilter: Debug/test patch

Data is processed in a loop and (page(s) memory) are
handled per loop. After each loop the context is
refreshed to a clean-state before starting another
round of processing.

There are few places where the context is not refreshed
before starting a new loop or exiting the loop.

This is a test patch to correctly refresh in
context in case processing loop in broken mid-way

Note: env context are not holders/owners of actual
large memory
