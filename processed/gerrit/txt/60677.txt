--- Commit Message ---
LU-19248 tests: Wait for fake intf ipv6 dup detection

IPv6 addresses undergo duplicate address detection during which they
are flagged as "tentative" in the output of ip addr show. Some test
cases do not work correctly until this flag is cleared on the fake
interface that is created in setup_fakeif(). Modify this function to
wait for the flag to be cleared before returning.

Test-Parameters: trivial
