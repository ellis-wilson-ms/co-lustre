--- Commit Message ---
LU-7642 build: Allow Lustre to build in absence of git

Lustre currently refuses to build if it is not in the presence
of a git working directory.  Subsequent tarballs generated
from "make dist" under a git working directory, on the other
hand, can build without the presence of a git working directory.

This patch is a quick hack to allow Lustre to build the _first_
time through without a git working directory.  This is especially
useful if one wants to grab a Lustre source tree using
"git archive --remote", for instance from inside an automated
build system's scripts.
