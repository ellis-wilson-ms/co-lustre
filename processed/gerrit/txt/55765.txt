--- Commit Message ---
LU-18038 llog: change llog_log_hdr format

The patch moves llh_tail from the end of a structure to
a middle, before a bitmap. This allows to skip one dt_write
during llog_add record. Old format is supported too. And a new
one only applies to a changelog and llog tests.

Also patch adds test11 to llog_test. It creates kernels threads
which in loop add records to a catalog and measure performance.
The test simulates changelog under havy metadata IO.

The patch speedup changelog a bit - 6.5% base on test11.
