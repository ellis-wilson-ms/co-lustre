LU-19947 quota: dump transaction info in error path

When we exceed QUOTA_MAX_TRANSIDS, Lustre will print an error
message. But we should dump more info about the quota
transaction so we can better root cause the issue. Otherwise,
it's not obvious if we are facing a legitimate bug.
