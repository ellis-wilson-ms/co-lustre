--- Commit Message ---
OSF-52 osp: add osp fid support

Add OST object FID support on OSP, OSP(MDT) requests FID sequence
from OST, and OST requests super FID sequence from FID controller
MDT0.

FID will be used to track precreate objects, and add lov_seqs file
to remember current seq of the precreation.

During precreation, OSP will preallocate FIDs first, then send these
FIDs to OST, and OST will precreate these objects with the FID.
