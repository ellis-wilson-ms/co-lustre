LU-11327 mdd: do not link file to PENDING on migrate

migrate does not apply for open objects. Then if an object is linked
to PENDING directory, there will be no way to unlink it from there
because there will be no close.

Do not link objects to PENDING on migrate.

Test illustrating the issue is added.
