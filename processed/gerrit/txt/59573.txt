LU-13643 flr: add tests for extend/split with immediate flags

Add comprehensive tests to verify that mirror extend and split
operations work correctly with immediate mirror files.

Test 212d verifies that:
- Mirror extend works on immediate mirror files
- The immediate flag is preserved after extending

Test 212e verifies that:
- Mirror split works on immediate mirror files
- The immediate flag is preserved on the remaining file when
  it still has multiple mirrors
- Split files do not inherit the immediate flag (they become
  regular files)
- The immediate flag is removed when a file becomes non-mirrored
  (only one mirror remaining after split)

These tests ensure that all major mirror layout operations
(create, extend, delete, split) work correctly with the
immediate mirror flag.

Test-Parameters: trivial
Test-Parameters: fortestonly
