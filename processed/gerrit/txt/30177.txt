--- Commit Message ---
LU-7058 utils: Allow lfs migrate to operate on directories

When directory names are provided to the lfs migrate command,
the contents of the directory are migrated instead of returning
-EISDIR. A new sanity test verifies that directory contents are
recursively migrated.

The help text for the migrate command is cleaned up, and now uses
a macro for options that are common with the setstripe command.

The --component-end|-E and --layout|-L options were missing from
the lfs migrate man page, and have been added.

Test-Parameters: trivial
