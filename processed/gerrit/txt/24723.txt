LU-5170 lfs: Standardize error messages in lfs utility

This patch makes usability improvements to the error messages
displayed by the lfs utility when the user provides invalid
arguments.  Messages are now prefixed with the name of the utility
to improve readability when run from scripts.  Message text is now
consistent when the same error type occurs.  String values provided
by the user are now delimited by single quotes to separate them
from the rest of the message text.

Subsequent patches will provide additional improvments, such as the
'--list-commands' option.
