--- Commit Message ---
LU-0000 lfs: model EC read/write on mirror read/write

Update lfs_ec_read() and lfs_ec_write() to closely
follow the implementation of lfs_mirror_read() and
lfs_mirror_write(), including:

- Proper file I/O with buffer allocation and management
- Use of leases for write operations (LL_LEASE_RESYNC)
- Actual data reading/writing loops using llapi_ec_read()
  and llapi_ec_write()
- Output file handling for reads (with O_EXCL)
- Input file handling for writes (with same-file check)
- Truncation handling for non-page-aligned writes
- Proper error handling and cleanup

The underlying llapi_ec_read() and llapi_ec_write()
functions are still stubs, but the lfs command structure
is now complete and ready for when the IOCTLs are fully
implemented.
