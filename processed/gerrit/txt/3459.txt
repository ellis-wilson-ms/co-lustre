--- Commit Message ---
LU-1672 obdclass: Set correct ldd_fsname when using osd-zfs

When mounting a ZFS-based OST, lsi_prepare() always set ldd_fsname to
"lustre".  This causes the OST to register to an incorrect Lustre file
system if the user has designated it to a file system with a different
name.  This patch fills ldd_fsname according to ldd_svname instead.
