LU-4367 open: fetch open lock by default

enable MDS_OPEN_LOCK by default, this can improve performance
a lot for certain situations, because upon close locks on this
file won't be canceled.

open also fetches MDS_UPDATE_LOCK/MDS_PERM_LOCK, along with
above change cached lock is more efficient.
