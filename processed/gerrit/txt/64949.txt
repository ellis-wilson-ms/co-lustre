--- Commit Message ---
LU-18951 utils: declare argv[argc] as NULL

The C standard requires that argv[argc] is NULL, but lsnapshot.c was
creating argument lists with only argc elements in it.  This could
cause command execution to crash in some cases.

Declare argv[] to have enough elements and initialize it to NULL.

Minor updates to related man pages.
