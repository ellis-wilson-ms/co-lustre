--- Commit Message ---
LU-822 osd: decrease OI files number to 32

Test results (attached on LU-822) show that 32 OI files can give
same performance as 64 OI files over ldiskfs. Also, 32 OI files can
give better performance than 64 OIs over ZFS, so we decrease default
number of OI files from 32 to 64.
