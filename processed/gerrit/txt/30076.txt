--- Commit Message ---
LU-10214 lnet: Add support for gateway range

Currently both lctl and lnetctl command does not
support range for gateway parameter while adding routes.

lnet_routes.conf file specifies example to add routes
using range for gateway parameter but the feature doesnt
exist. Adding the feature with this patch for both lctl
and lnetctl command to allow ranges for gateway
parameter while adding routes.
