--- Commit Message ---
LU-20094 lnetctl: overflow if import with old api

lnetctl import could lead to:
*** buffer overflow detected ***: terminated
Aborted (core dumped)

The buffer used when reading YAML file with fgets in old API is 1 byte
too small since we need to store '0'.

Also wrong length is given to fgets because strlen is called on
modified buffer instead of the old one.
