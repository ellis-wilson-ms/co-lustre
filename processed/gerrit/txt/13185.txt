LU-6070 libcfs: provide separate buffers for libcfs_*2str()

Provide duplicates with separate buffers for libcfs_*2str() functions.

Replace libcfs_nid2str() with libcfs_nid2str_r() function in critical
places.

Provide buffer size for nf_addr2str functions.

Use __u32 as nf_type always

Why not use libcfs_nid2str_r() to implement libcfs_nid2str()?

 
char *libcfs_nid2str(lnet_nid_t nid)
{
        return libcfs_nid2str_r(nid, libcfs_next_nidstring(), LNET_NIDSTR_SIZE);
}

Then assume that libcfs_nid2str_r() is always passed a non-NULL buffer. This avoids the duplication of logic between these pairs of functions.
