LU-8066 sysfs: properly handle failure of sysfs/procfs initialization

While preparing for the move to sysfs I noticed that
initialization of procfs didn't handle most error
cases. So this patch address those issues. Also the
upcoming sysfs code will setup default attrs which
will come in three different flavors so add a flag
so the proper struct attribute_group is used.
