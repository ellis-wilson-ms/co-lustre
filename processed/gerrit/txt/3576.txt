Patch Set 1: I would prefer that you didn't submit this

Instead of having inline #ifdef code, and to make it easier to submit code upstream with minimal changes, please define a macro like:

 #ifdef HAVE_VOID_MAKE_REQUEST_FN
 #define ll_mrf_ret void
 #define LL_MRF_RETURN(rc) EXIT
 #else
 #define ll_mrf_ret int
 #define LL_MRF_RETURN(rc) RETURN(rc)
 #endif

These can be used online in the code, and can easily be replaced with sed for upstream submission. 

For GOTO, the typical usage is of the form:

                GOTO(label, rc = PTR_ERR(ptr));
