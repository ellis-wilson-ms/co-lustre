--- Commit Message ---
LU-20100 obdclass: obdclass: max_dirty_mb module parameter

Add module parameter to lustre to allow default setting of
max_dirty_mb

Use max_dirty_mb to get the initial default for
  lctl get_param max_dirty_mb

Or optionally update it with:
  lctl set_param [-P] max_dirty_mb=6500

It can also be set at module load time

Ex: /etc/modprobe.d/obdclass.conf contains:
options obdclass max_dirty_mb=6173
