LU-19929 build: Add RISC-V to SUBARCH mapping

Add RISC-V architecture support (riscv64) to the SUBARCH
mapping in build configuration. This refactors the existing
sed-based approach to use case statements for better code
organization and maintainability.

The SUBARCH variable is used throughout the build system to
determine architecture-specific kernel paths and settings.
