LU-0000 lfs: add lfs ec read and lfs ec write commands

Add stub implementations of 'lfs ec read' and 'lfs ec
write' commands to read and write EC (Erasure Coding)
components of a specified mirror.

These commands are modeled after 'lfs mirror read' and
'lfs mirror write', accepting the same --mirror-id|-N
parameter to specify which mirror to operate on.

The stub implementations verify that:
- The specified mirror ID exists in the file
- The mirror contains EC (parity) components

A new helper function check_mirror_has_ec() iterates
through all components in the specified mirror to find
parity components (identified by the LCME_FL_PARITY
flag).

Man pages for both commands are included.
