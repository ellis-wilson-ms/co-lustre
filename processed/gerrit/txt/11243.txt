--- Commit Message ---
LU-4053 mdt: return -ENOENT in mdt_intent_layout()

mdt_intent_layout should return -ENOENT for non-existing object,
otherwise, layout lock could be acquired on client for an unlinked
object, and that result in the inode cache for the unlinked object
can't be purged.
