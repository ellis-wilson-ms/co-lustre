LU-9698 osd-ldiskfs: See if kallsyms_lookup_name is exported

kallsyms_lookup_name is not exported prior to 2.6.33 kernels,
so rhel6 kernels cannot use ldiskfs anymore which annoys
people still using it even though it is not supported.
So add a configure check for it.
