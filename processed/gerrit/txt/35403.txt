LU-12501 utils: stop showdf() endless loop

The function showdf() in lfs.c copies the struct obd_statfs
os_state field and then loops over each bit to test if it is
set. If the bit is set then its masked out. The loop then exit
once all the bits are masked out. This does not happen for the
OS_STAT_NONROT case. Since the OS_STAT_NONROT is never masked
the loop never breaks. Always mask out OS_STAT_NONROT instead
which ensures we still skip this value and allows all bits to
be masked to break the loop.
