LU-3176 kernel: md/raid6 fix read-error correction

md/raid6 is not doing correct read-error when single disk failed
and going to be degraded mode for one more drive.
The original code only considered raid5, but didn't work for raid6.
This is fixed in mainstream linux kernel.
