LU-19149 build: make lbuild recognize Rocky Linux

Some Rocky Linux versions return RedHatEnterprise as
a distro name, others return Rocky or RockyLinux.
Map all these names to "rhel" in autodetect_distro().

Make the distro names in autodetect_target() consistently *not*
have a dash in them (regardless of how they are specified).

name=$(lsb_release -s -i)
-> name=RedHatEnterprise
