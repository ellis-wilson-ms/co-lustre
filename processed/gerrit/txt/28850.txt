LU-9943 lnet: fix queue size in ENOMEM case.

Changing a WR count without queue size change caused
a IB error
  mlx5_ib_post_send:4184:(pid 26272): Failed to prepare WQE
due situation when o2iblnd code allocate a more tx's when
QP settings.
Lets adjust a connection queue depth to reflect a WR
queue size changes.

Fixes a 4083806828a94ee09c2dadf2cca8c224547d5ebc
