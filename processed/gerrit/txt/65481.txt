--- Commit Message ---
LU-20178 lov: layout support for compression fields

Add LOV and LOD layer support for storing and retrieving
per-component compression settings in composite layouts:

- LOV: unpack compression type and chunk size from
  lov_comp_md_entry_v1 into lsm_component_entry, and
  repack them when converting back to on-disk format
- LOD: propagate compression parameters through the
  layout pipeline (lod_object.c, lod_qos.c, lod_lov.c)
- Internal header updates to expose compression fields
  in lov_stripe_md_entry
