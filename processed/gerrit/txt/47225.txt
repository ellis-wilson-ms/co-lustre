--- Message (Patrick Farrell, 2022-05-18 15:53:50.000000000) ---
Patch Set 5:

Shadow,

The valid flag is only used for attr writes, it's not used by attr_get in Lustre.  attr_get doesn't set the valid flags.  It never has.

This issue occurs on the first read of a file, when we don't know the size, so the KMS shows as zero.

We calculate the read offset as 0 - pos = start + count - 1; so that also shows as zero.

The ">" check in prep_size means that single byte reads at KMS don't trigger the size check.  That's wrong.

Shaun's patch fixes that issue.
