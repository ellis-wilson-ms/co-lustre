Currently Lustre 2.1 does not work for kernels < 2.6.18 which by extension NFS over Lustre also doesn't work.This was also reported in LU-1411/LU-1646 which is fixed by this patch. This patch enables one to mount lustre for these older kernels but NFS will not still mount for kernels < 2.6.18. NFS should work for modern kernels. With the patch for LU-1718 for b2_1 it will restore NFS to modern as well as older kernels.
