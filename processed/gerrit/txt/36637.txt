LU-12924 ofd: scale ofd.precreate_batch with OST size

Scale OFD object precreate_batch size with the OST size, since the
object precreate performance can limit metadata create rates when
there are fewer OSTs for a given amount of space.  There is a
limit on the precreate_batch size due to the maximum transaction
size, which is tuned by mkfs.lustre by default, and also a protocol
limit on the batch size based on OST_MAX_PRECREATE.
