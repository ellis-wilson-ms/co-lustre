--- Commit Message ---
LU-13643 flr: implement immediate mirror writes for DIO

NB: This does not implement any error handling and only works
with DIO

Implement sequential mirror iteration for immediate mirror files
during Direct I/O write operations. When writing to a file with
the immediate mirror flag, iterate through all mirrors writing
the same data to each mirror sequentially.

Key changes:
- Add mirror iteration loop in cl_io_loop() for DIO writes
- Reset iov_iter state between mirror iterations in VVP layer
- Update LOV layer to use mirror index from iteration logic
- Acquire/release ACTIVE_WRITER locks during immediate mirror
writes

This provides immediate data replication for DIO writes to
immediate mirror files, eliminating resync requirements.
