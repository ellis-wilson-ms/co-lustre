Patch Set 2: I would prefer that you didn't submit this

If ldiskfs patches need to be changed, you should very likely be copying those patches to a new rhel6.5 subdirectory and changing them there.  Then make a new series file for 6.5 that is selected from the autoconf stuff if the correct kernel version number is detected.
