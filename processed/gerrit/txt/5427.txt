LU-2806 build: Enable download of OFED release candidates

lbuild doesn't allow download of OFED release candidates.
RHEL/CentOS 6.3 OFED support is available for OFED version 3.5-rc{1,2,3,4,5,6}
only. To test software builds and functional testing using infiniband as
interconnect the change enables the download of version 3.5-rc{1,2,3,4,5,6}.
Note: This change will never land in any branch. It will be removed as
soon as a stable version of OFED-3.5 (or later) has been released.
