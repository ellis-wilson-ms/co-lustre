LU-17077 tests: User correct group id in RUNAS

sanity-quota/63 RUNAS was using the same id for
gid. This patch adds new RUNAS3 global variable
which uses the correct gid.

For quota_usr:
$ getent passwd quota_usr
quota_usr:x:1001:1002::/home/quota_usr:/bin/bash

Debug run logs original code:
runas -u 1001 -g 1001 ...

Debug run logs changed code:
runas -u 1001 -g 1002 ...
