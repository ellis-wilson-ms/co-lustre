LU-13020 o2iblnd: timeout is now obsolete

The LND timeout is now determined as a function of the
lnet_transaction_timeout module parameter:

if lnet_retry_count > 0 &&
   lnet_transaction_timeout > lnet_retry_count
   lnd_timeout = lnet_transaction_timeout / lnet_retry_count
else
   lnd_timeout = lnet_transaction_timeout

For more details please look at:
  api-ni.c:retry_count_set()
and
  api-ni.c:transaction_to_set()
