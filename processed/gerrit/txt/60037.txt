LU-19143 tcu: lfs df --trash command to show with compensation

By default "lfs df" shows the MDT/OST block and inode usage with
the @trash_can_projid usage subtracted from used space and inodes.

A new option is added "lfs df --trash" that shows the free or
available blocks and inodes without the adjusted @trash_can_projid
usage, so that it is possible to see the actual filesystem usage
for diagnostic reasons.

Add sanityn/test_117ub.
