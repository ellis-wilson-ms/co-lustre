--- Commit Message ---
LU-19179 compat: use directly native Xarray support.

RHEL7 was missing Xarray support so we used a copy from a newer
kernel. Now that RHEL7 support is dropped we can remove our
Xarray copy. A few newer kernel issues exist so now our xarray.h
just contains the work around. Latestly test if Xarray multi-tier
support is enabled. Since its used by device zone handling which
is used by many drivers the changes are small it will be disabled.
Add a test just for that rare case.
