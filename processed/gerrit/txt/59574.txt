LU-13643 tests: add immediate mirror data operation tests

Add comprehensive test coverage for immediate mirror files
including data operations like read, write, copy, resync,
and verify. These tests validate that immediate mirror
files work correctly with all standard FLR operations.

Tests added:
- test_212f: mirror read/write operations
- test_212g: mirror copy operations
- test_212h: mirror resync operations
- test_212i: mirror verify operations (basic functionality)

The tests handle the fact that immediate mirror files may
have different behavior than regular mirror files for some
operations, particularly direct writes and verify operations.
Tests use mirror write commands instead of direct file writes
to avoid kernel crashes, and gracefully handle operations
that may not be fully supported yet.

These tests complement the existing immediate mirror tests
(212a-212e) to provide complete coverage of immediate mirror
functionality.

Test-Parameters: trivial
Test-Parameters: fortestonly
