--- Commit Message ---
LU-3071 obdfilter: call cfs_cap_raise() before journal start

cfs_cap_raise() may allocate memory and result in page flush on
different fs, which could probably trigger a nested journal
start on different journal.

This patch moves cfs_cap_raise() before journal start.
