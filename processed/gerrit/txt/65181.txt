--- Commit Message ---
LU-19066 obdfilter: add failure_domain parameter

Add a failure_domain parameter to the device.

This can be set on a per-device basis using:
  lctl set_param obdfilter.lustrewt-OST0000.failure_domain=55
and will be used to allow FLR stripe allocation
to become failure_domain aware.

Later patches will store this persistently on each OST itself
and will not be needed to be stored in the persistent MGS config
with "set_param -P".
