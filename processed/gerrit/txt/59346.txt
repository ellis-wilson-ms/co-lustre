--- Commit Message ---
LU-18844 mdt:  restrict hard linking of zero-link count files.

Add safety checks to osd_ref_add() to only allow hard-linking of
i_nlink==0 files when:
- File has VOLATILE/LINKABLE flag set in oo_lma_flags
- Parent is PENDING or TRASH directory
- Otherwise return -ENOENT to prevent inadvertent resurrection of
  orphaned files

This complements mdt_reint_link() which clears VOLATILE flag after
successful linking.
