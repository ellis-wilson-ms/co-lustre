--- Commit Message ---
LU-18791 tests: speed up sanity test_51b/test_51c

Both sanity test_51b and test_51c can take a very long time to
run in Janitor testing (20-30 and 40-60 minutes respectively).

Speed this up by running createmany and unlinkmany in parallel
when creating directories of the required size for the test.

Sanity test_834 is also over 20 minutes, but no easy way to
speed it up at this point.  It still runs in Autotest sessions.

Fix a mostly-harmless typo in sanity/51c (mtdidx -> mdtidx).
Add slow subtests to the EXCEPT_SLOW list for Janitor testing.

Clean up test script style issues in affected subtests.
