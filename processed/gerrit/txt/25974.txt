--- Commit Message ---
LU-0000 mgs: remove skip records from config file

This is debug patch for only_mgs_is_running()
verification.

Configuration logs are append-only files of limited size.
Over the course of time the logs may grow over the limit size.
Usually, configuration logs keep needless records marked
as SKIP. The new lctl command "clear_conf" is added to
allow administartors to clear configuration files by
removing mentioned SKIP records. lctl man page is updated.
conf-sanity test is added to test the new command.
