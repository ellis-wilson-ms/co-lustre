--- Commit Message ---
LU-333: read_bytes in llite/*/stats should report bytes read.

Requesting a large number of bytes to be read from a small or empty
file causes inflated values in llite/*/read_bytes.  The problem is
that the LPROC_LL_READ_BYTES stat is tallied by the umber of bytes
requested rather than the number of bytes actually read.
