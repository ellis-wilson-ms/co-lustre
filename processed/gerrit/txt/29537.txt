--- Commit Message ---
LU-10107 clio: truncate PFL file at component boundary

Truncate I/O used to be sent to overlapping components of
[size, eof), however this is problematic when the file is
truncated up to component boundary and if the next component
is not instantiated yet.

This patch will change to send truncate RPC to overlapping
components by [size - 1, eof).
