--- Commit Message ---
LU-723 External lustre-ldiskfs package integration

Add a configure option --with-ldiskfs-devel to build Lustre against an
externally maintained ldiskfs package. This package maintains ldiskfs
as a standalone code base. The intention is to make the ldiskfs API
and dependencies explicit and to have a clear layering between ldiskfs
and Lustre. With the addition of alternate OSDs building ldiskfs
should become optional.
