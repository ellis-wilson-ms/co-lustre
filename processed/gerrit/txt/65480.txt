LU-20178 osc: wire protocol and UAPI definitions for CSDC

Add wire protocol structures and UAPI definitions for client-side
data compression (CSDC). This includes:

- New compression type and chunk size fields in struct
  lov_comp_md_entry_v1 for per-component compression settings
- OBD_CONNECT2_COMPRESS connect flag for client/server
  negotiation of compression capability
- UAPI header lustre_compr.h defining supported compression
  algorithms (lz4, lz4hc, lzo), compression levels, and
  chunk sizes (64KB to 4MB)
- Wire protocol test updates in wirecheck.c and wiretest.c
  to validate the new fields
