LU-12452 lnet: allow to set IP ToS value per-NI

This patch allows to set the IP "Type of Service" value per network
interface to use IP QoS on TCP or RoCE network.

e.g:
	$ lnetctl add --net tcp2 --if eth1 --tos 104
