LU-18803 class: add namelen_max and maxbytes params

Add parameters for all OSD/OBD devices to print the statfs
os_namelen and os_maxbytes values from the respective devices.
This allows checking these values more easily from test scripts.

Update test_79 to print the values (for functionality) and test
that the llite.*.namelen_max matches the value of sbi->ll_statfs.

It might be possible to make a function-generating macro for them, but they are otherwise just a few lines and not hard to write. I'm not sure that having a macro like:

 DT_STATFS_FN(kbytesfree);
 :
 DT_STATFS_FN(namelen_max);
 DT_STATFS_FN(maxbytes);

is a huge improvement to readability, and a meta-macro like:

 CREATE_STATFS_FNS(DT_STATFS_FN);

that instantiates each of the "kbytesfree", ..., "namelen_max", variants of the passed macro would make that even more obscure when trying to find these functions.
