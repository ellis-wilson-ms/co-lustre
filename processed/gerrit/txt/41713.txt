LU-14459 mdt: autosplit directory during create

Auto-split directories over available MDTs at creation time so
that performance and capacity scales as the directory grows,
rather than waiting until the directory is accessed since not
all workloads read the directory during file creation, and that
results in a much large number of remote directory entries.

Separate out the logic for directory split so that it can be
used in the create path.
