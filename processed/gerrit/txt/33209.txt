LU-11380 llite: add LL_IOC_FID2PARENT and fid2parent

Change ll_getparent(file, arg) to ll_getparent(sbi, inode, fid, arg)
so it can be used to get the parent of any file by FID in addition to
the current file.  Use the refactored ll_getparent() to add a new
ioctl LL_IOC_FID2PARENT to allow getting parent by FID, along with
llapi_lufid2parent(), llapi_fd2parent(), and llapi_path2parent()
to access it from applications.

Add "lfs fid2parent" to directly list a FID's parent directories.

Add man pages for the new llapi functions and lfs sub-command.
