Patch Set 4: I would prefer that you didn't submit this

Sorry for the confusion here.  This patch is now the mashup of the first approach (using errno to pass the status to llapi_err()) which was very messy and hard to maintain, and the second approach (using "rc" from llapi_*() functions and passing this directly to llapi_error()).

Please remove the changes that were setting "errno" before returning this same value as "rc", and just use "rc" to pass to llapi_error().  I've pointed out some of the places in the liblustreapi.c patch that should be changed, but there are many more.
