--- Commit Message ---
LU-0000 utils: allow mirror rw of mirror 0

EC will require specifying designated IO to direct IO to
the parity components.  The easiest way to do this for
non-mirrored files is to allow using mirror read/write to
mirror ID 0.

NB:
The read implementation is fine, but the write
implementation changes the lease type from RESYNC to a
plain lease - this is incorrect, since we'll want EC
comp write to be a resync just like a normal mirror
write.

This requires fixing up some of the layout implementation
first.
