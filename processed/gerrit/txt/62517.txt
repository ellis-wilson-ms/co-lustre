--- Commit Message ---
LU-12187 lod: set parity component pattern

When creating a component, set parity component pattern
(LOV_PATTERN_PARITY) if the component has been set LCME_FL_PARITY
flag, i.e. it's a parity component.

This ensures that older clients will not try to read this
component as file data.
