--- Commit Message ---
LU-18400 libcfs: add bounds checking to proc_dobitmasks()

Fail writes larger than PAGE_SIZE in proc_dobitmasks().
Otherwise, privileged users can exhaust system memory
by passing a sufficiently large write length - since the
write length is pass unchanged to memdup_user_nul().

Fix some style issues at the same time.
