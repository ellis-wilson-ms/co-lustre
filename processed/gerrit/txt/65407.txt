LU-20090 nrs: add dynamic granularity classifier for TBF

In the original NRS TBF scheduler, the classifier is static, which
means the classifier type is fixed when the policy is started. And
the classifier type can not be changed until the policy is stopped
and started again.
The adding rules with different classifier types are not allowed.

This patch adds dynamic granularity classifier for TBF scheduler,
which means the classifier type can be changed dynamically for
different rules. And the classifier type is determined by the
rule.
The class type for the dynamic classifier can be automatically
determined by the matching condition fields of a newly adding
rule.

Add sanityn/77tb to verify that the dynamic classifier works as
expected.
