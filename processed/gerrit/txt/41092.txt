--- Commit Message ---
LU-8130 obdclass: convert nid_stats to rhashtable

Linux has a resizeable hashtable implementation in lib,
so we should use that instead of having one in libcfs.
The Linux version has better performance.

This patch converts the nid stats hash to use rhashtable.
