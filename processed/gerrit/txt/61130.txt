--- Commit Message ---
LU-19229 lnet: change LNetAddPeer() error handling

Patch modifies LNetAddPeer() to unify error handling,
reporting and returning.

- make LNetAddPeer() void as its return code was never
  checked but just used in debug message.
- move all debug to LNetAddPeer() itself and add more
  messages for different cases
- don't stop processing of incoming NIDs in case of error
- consider -EEXIST and -EALREADY as appropriate result
- fix wrong pnid reset in case of -EEXIST

Fixes: a050dfb80d ("LU-19229 mgc: client to handle IR with unknown nets")
