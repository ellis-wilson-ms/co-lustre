--- Commit Message ---
LU-3538 dne: eliminate dependency in striped dir creation

DNE needs to eliminate dependency between 'mkdir a/b' and any
operation on 'a' because 'b' may be a striped directory, which is
able to recover from update log, if 'a' can't be recovered, it
will cause problem.

To achieve this,  mkdir needs to save child lock, so that
subsequent mkdir under it can know the conflict.

Signed-off-by: Lai Siyao <lai.siyao@intel.com>
