Have you done a performance comparison with this patch enabled and disabled to see how much the OST object creation costs?  In theory, there should be no visible performance hit because there are more OSTs than MDTs, and because of precreate, but it would be good to verify this. 

If there is a significant impact from OST precreate then it would be worthwhile to file a ticket to fix it, since this can hurt real world performance, and there is no benefit if the MDT is fast but the OSTs are slow.
