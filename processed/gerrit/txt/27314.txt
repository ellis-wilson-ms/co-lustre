--- Commit Message ---
LU-9184 mdc: move layout data to the lr_lvb_data

Layout data is stored in lock lvb on client side and it
makes impossible to use this LVB for Data-on-MDT IO lock
at the same time. Patch moves this data to the resource lvb
which is not used on client, leaving lock lvb data for DoM
purposes. This is for testing purposes for now.
