--- Commit Message ---
LU-3374 tests: Only import zpool if not present

Only do zpool import if pool isn't already present, otherwise it will
fail needlessly.  This fixes tests that do mount_facet and expect it
to fail then run mount_facet again.

Only export pool if there are no datasets mounted from it.
