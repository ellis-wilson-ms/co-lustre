LU-6202 utils: allow device_list for non-root users

Allow /dev/obd to be opened by a non-root user.  If a read-write
open fails, try opening the device read-only.

Allow ioctl(OBD_IOC_GETDEVICE) to be run by a non-root user.

The actual /dev/obd device is currently only accessible by root,
but if the permission is changed to allow non-root users, they
could use "lctl dl" to list the OBD devices without root access.
