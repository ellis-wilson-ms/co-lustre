--- Commit Message ---
LU-18175 lustre: create shrinker symlinks

Lustre's use of shrinkers end up creating debugfs interfaces
for newer kernels. The debugfs naming is not consistent every
time since the shrinkers id is using to create a unique name
that varies over time. Have Lustre create symlinks into its
debugfs tree with a constant name so user can make static
tools to manage memory usage.
