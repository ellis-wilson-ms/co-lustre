LU-5420 mgc: MGC should retry for invalid import

Since MGC import is not fully replayable, (i.e. the import might
be evicted during recovery) mgc_process_log() should retry calls
to mgc_enqueue() (for retrieving config log) until the import is
fully recovered or closed.
