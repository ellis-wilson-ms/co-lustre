--- Commit Message ---
LU-18985 tcu: handle the removal of a striped directory

When a user is deleting a striped directory with Trash Can
enabled, MDT needs to create a striped trash directory on Trash
Can where the shards' FID are FIDs of the corresponding stub shard
objects.

When deleting a striped directory and create a striped trash dir,
it still needs to rename and move the stub shard objects in Trash
Can into the REMOTE_PARENT_DIR on MDT for remote agent objects.

Add sanityn/test_117r.
