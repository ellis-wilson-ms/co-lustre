LU-18997 lnet: set rc when dev_get_by_index_rcu() fails

The function lnet_sock_create() returns an ERR_PTR so rc needs to
be set to a real error code. This was not happening for the case
when dev_get_by_index_rcu() fails. Set rc to ENODEV.
