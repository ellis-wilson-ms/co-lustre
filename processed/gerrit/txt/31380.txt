LU-9091 tests: use upstream units for proc settings

The work for LU-7734 was written long before the linux kernel
introduced string helpers. This patch uses the units approved
of upstream instead of the ones currently supported, M,K,G etc.
This will show that the current work by LU-7734 is written such
that it will not see a difference between the upstream units and
what is currently supported.
