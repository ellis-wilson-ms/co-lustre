LU-12400 ldiskfs: Case insensitive dirhash ext4fs_dirhash

Ext4 added case-insensitive (per directory) support.
This change affects the directory hashing scheme so that
on case-insensitive flagged directories the mixed case
names hash to the same value.

This patch updates the API call but does not support the
case-insensitive flagged directory scheme.
