LU-1627 build: use HTTP for ZFS/SPL cloning

Use the https://github.com/zfsonlinux URL for doing clones of the
ZFS and SPL repositories, instead of the SSH URL.  This will avoid
the need to have an SSH key at github for cloning and make the
deployment of new Lustre build nodes easier.
