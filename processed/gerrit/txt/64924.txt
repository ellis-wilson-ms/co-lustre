LU-18548 tests: ensure sanity/160f GC has started

When waiting for the "idle" changelog user to be removed, wait a
few seconds for the process to start.  Otherwise it is possible
that the GC thread hasn't yet started when the test is checking
for it to have completed, which causes intermittent test failures.
