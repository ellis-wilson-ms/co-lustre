--- Commit Message ---
LU-19074 ec: Add EC library sanity test to test suite

Add erasure coding library testing to the Lustre sanity test suite
to validate Reed-Solomon functionality and enable automated testing
of EC algorithms in CI/CD pipelines.

The test validates multiple RAID configurations and demonstrates
core EC operations including matrix generation, encoding, and
recovery algorithms.

Includes unified build system allowing the same EC source files
to work in both kernel and userspace environments, eliminating
code duplication and maintenance overhead.

Test-Parameters: trivial
