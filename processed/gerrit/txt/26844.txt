LU-9406 test: conf-sanity 82b start all clients

conf-sanity test 82b stops all clients and servers,
sets new indicies on the OSTs and then brings up the
file system with only one client. In the case of the
MGS and the MDS on separate nodes, for OST pools to
work, you need a client on the MGS. The single client
that mounts the file system in this test may or may
not be the client on the MGS.

For conf-sanity test 82b, all clients must mount the file
system
