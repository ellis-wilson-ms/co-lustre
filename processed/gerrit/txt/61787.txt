--- Commit Message ---
LU-19463 llite: fiemap extent copy individually

Avoid usercopy failure with CONFIG_HARDENED_USERCOPY:

  usercopy: Kernel memory exposure attempt detected from \
    SLUB object 'kmalloc-96' (offset 32, size 112)!

Bug is triggered due to detecting an attempt to copy beyond
the end of the kmalloc'd memory. The memory area is undersized
and the allocation should have been for and additional extent.
