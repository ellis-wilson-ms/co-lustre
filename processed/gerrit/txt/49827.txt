LU-16516 tests: remove dup convert_facet2label()

The test-framework.sh::convert_facet2label() function is largely
redundant with facet_svc() and has a longer name, so it will be
removed and callers replaced with facet_svc().

Add an explicit client_svc so that it doesn't fail when empty.

There are a few callers that pass "ost" as the facet which is
replaced by "ost1" internally, but they can also pass ost1 directly.
There is a similar substitution from "mgs" to "mds1" for combined
MGS/MDS servers, but this doesn't appear to be used anywhere.
