LU-0000 lfs: add --immediate option to lfs mirror create

Add support for the --immediate flag when creating mirrored files
using the lfs mirror create command. This allows users to create
mirrors with immediate synchronization behavior.

The implementation adds:
- New --immediate option to lfs mirror create command
- Validation to ensure --immediate is only used with multiple mirrors
- Proper application of LCM_FL_IMMEDIATE_MIRROR flag at layout level
- Integration with existing mirror creation infrastructure
- Updated help text to document the new option

The immediate flag is applied to the final merged layout after
mirror count validation, ensuring it's set as a layout-level
flag rather than a component-level flag.

Examples:
  lfs mirror create -N2 --immediate /mnt/lustre/file
  lfs mirror create -N3 --immediate -c 2 /mnt/lustre/striped_file

This provides the foundation for creating immediate mirrors that
can then be found using lfs find --mirror-flags and tested with
the comprehensive test suite.
