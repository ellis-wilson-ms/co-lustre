--- Commit Message ---
LU-13011 wbc: print PCC file path for data on PCC

In this patch, it prints the PCC file path and cache mode for
regular files that already assimilated file data into PCC.
The output for the command "lfs wbc state $file" will show the
following information:

"/mnt/lustre/d32.sanity-wbc/f32.sanity-wbc, state: (0x00000037)
protected sync complete reserved, data: PCC(/d32.sanity-wbc/0004/
0000/0401/0000/0002/0000/0x200000401:0x4:0x0), dirty: (0x00000000)
uptodate, flush_mode: aging_keep, cache_mode: dop"
