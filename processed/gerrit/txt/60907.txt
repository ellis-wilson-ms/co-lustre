LU-19769 compat: cleanup linux-prim.c file

Most of the code in linux-prim.c was written to support RHEL7
which no longer is supported. We can remove all the code in
linux-prim.c that are wrappers that don't require a modification
to the core LNet or Lustre kernel code.
