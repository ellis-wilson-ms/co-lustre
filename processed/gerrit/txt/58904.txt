LU-18939 llite: allow readahead params to be set as percentage

Allow the following readahead parameters to be set as a
percentage of total RAM, similar to max_cached_mb:
- llite.*.max_read_ahead_mb
- llite.*.max_read_ahead_per_file_mb
- llite.*.max_read_ahead_whole_mb

This allows administrators to set these parameters as a
percentage of the total RAM on the system, which is more
convenient than calculating the absolute values, especially
on systems with different amounts of RAM.

Added test to verify the percentage-based settings work
correctly.
