--- Commit Message ---
LU-0000 mdt: fix concurrent write lease compatibility

Fix compatibility issue with concurrent write leases by
ensuring that LL_LEASE_CWLCK is properly handled in the
lease compatibility checking logic.

The concurrent write lease type should be compatible with
other concurrent write leases on the same file, allowing
multiple writers to operate simultaneously on mirrored
files while maintaining proper FLR state management.

This change updates the lease compatibility matrix to
correctly handle the new LL_LEASE_CWLCK lease type.
