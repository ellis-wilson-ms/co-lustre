--- Commit Message ---
LU-18676 osd: check all commit callbacks done

DMU's txg_wait_synced() may return before all corresponding commit
callbacks have been called. this break Lustre's quota logic which
builds internal accounting on top of commit callbacks.

add extra "wait" to ensure all commit callbacks registered before
osd_sync() have been called.
