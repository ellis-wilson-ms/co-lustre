--- Commit Message ---
LU-20178 utils: lfs setstripe compression support

Add compression options to the lfs setstripe command for
creating files with client-side compression:

- lfs.c: add --compress-type/-Z and --compress-chunk
  options to lfs setstripe, with validation that
  compression requires a composite layout (-E flag).
  Supported types: lz4, lz4hc, lzo, none.
  Supported chunk sizes: 64k, 128k, 256k, 512k,
  1m, 2m, 4m.
- liblustreapi.c: pass compression parameters through
  the layout API to the kernel
- liblustreapi_layout.c: implement getter/setter
  functions for compression type and chunk size in
  llapi_layout component entries
- lustreapi.h: declare new layout compression API
  functions

Usage example:
  lfs setstripe -E eof -Z lz4 --compress-chunk 256k file
