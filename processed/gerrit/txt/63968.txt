--- Commit Message ---
LU-0000 tests: add EC recovery EOF boundary tests

Add test_51a and test_51b to sanity-ec.sh to exercise
EC recovery with EOF at four specific positions within
RAID set / stripe set geometry for -c 6 --ec 3+1:

  - RS0/SS0: EOF within first stripe set
  - RS0/SS1: EOF within second stripe set
  - RS1/SS0: EOF in second RAID set, first SS
  - RS1/SS1: EOF in second RAID set, second SS

test_51a uses a single-component layout, test_51b uses
PFL with two components ([0,4M) + [4M,EOF)).

These tests found three bugs in the parity read path
that are fixed in a companion commit.
