--- Commit Message ---
LU-13643 tests: add sanity-flr tests for immediate mirror flag

Add comprehensive tests for the immediate mirror flag functionality
in sanity-flr.sh. The tests cover:

test_212a: Tests mirror creation with --immediate flag
- Verifies immediate flag is set correctly on created files
- Verifies immediate flag is not set on regular mirror files
- Tests that --immediate flag is ignored in mirror extend operations
- Validates readonly flag behavior

test_212b: Tests lfs find --mirror-flags functionality
- Tests finding files with immediate mirror flag
- Tests negation with ! operator
- Tests finding files with readonly flag
- Tests combination with other mirror search options
- Tests ^flag syntax for negation

These tests ensure the immediate mirror flag and lfs find
--mirror-flags functionality work correctly together.
