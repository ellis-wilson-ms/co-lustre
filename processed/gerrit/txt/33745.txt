Note that *optional* test sessions like review-ldiskfs-arm above and review-dne-zfs-part-[34] do not need to pass in order for a patch to land (they are optional because they are newly added and fail too often, but we want to run them a lot to ensure we catch intermittent failures that aren't seen by just a few runs).

On the other hand, *required* test sessions like review-dne-part-4 DO need to pass in order for a patch to land, so those are the ones that people should pay attention to.
