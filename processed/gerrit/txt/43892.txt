LU-13055 mdd: make current changelog mask writable

Current changelog mask is the composition of per-server mask
and all user masks. It is recalculated when either of them
is changed. Meanwhile it is also useful to be able to change
the effective mask temporary without touching per-server mask
or adding a new user with needed mask. One use case is disabling
logging temporary. Such changes will be reverted when either
per-server mask is set or user with mask is registered or
deregistered.

Patch makes mdd.*.changelog_current_mask writable and adds
new special mask name 'off' to turn changelog off temporary,
the same is applicable for debug mask
