--- Commit Message ---
LU-16469 mdt: add parent FID to CL_LAYOUT changelog records

The parent FID on CL_LAYOUT changelog records seems to be incorrect
when performing OST migration, MDT migration, setting striping
parameters on a directory, creating a mirror and when doing HSM
restore on a file. It is also missing completely when doing a mirror
split, a mirror extend and when doing an HSM release.

Use the link EA to fix this, as this field can be useful
in enabling certain HSM-related indexing tools to operate more
efficiently, as per LU-13151.
