LU-12345 llite: add client-side IT_WRITE_LEASE support

This patch implements client-side support for write lease
intents in the Lustre file system. Write leases provide
exclusive write access to files, preventing concurrent
modifications and ensuring data consistency.

The implementation adds:
- IT_WRITE_LEASE intent constant definition
- Client-side intent processing in write operations
- Debug string mapping for write lease intents
- Proper error handling in MDC layer

The write path now uses IT_WRITE_LEASE intents when
performing write operations, allowing the server to
grant exclusive write access and coordinate between
multiple clients.

This is the first part of write lease implementation,
providing the client-side infrastructure. Server-side
lease management will be added in subsequent patches.
