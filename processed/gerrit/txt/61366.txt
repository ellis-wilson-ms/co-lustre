LU-19386 lnet: support multiple interfaces in YAML import

Add support for importing and deleting multiple network interfaces
in a single YAML configuration listing all interface names under
a single "interfaces:" sequence. This format was supported in 2.15,
but stopped working with the move to netlink.

Key changes:
- Added lnet_del_ni_by_interface() helper function to encapsulate
  common deletion pattern of finding and deleting NI by interface
- Modified lnet_genl_parse_local_ni() to handle interface arrays
  by counting interfaces, allocating configuration arrays, and
  processing each interface separately for both add and delete ops
- Updated error handling to provide clear messages for multi-interface
  operations and proper memory cleanup
- Added test case 450 in sanity-lnet.sh to verify multi-interface
  YAML import and deletion functionality
