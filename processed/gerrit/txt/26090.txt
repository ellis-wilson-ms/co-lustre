LU-2435 osd-zfs: for test only

If ZFS userobj_accounting feature exists, use it for osd-zfs object
accounting; otherwise estimate object use by block space.

Disable quota check in conf-sanity:32b if the underlying zfs version
is below 0.7.0. For ZFS 0.7.0 and higher, upgrade the zpool image to
enable dnode quota accounting. That ensures we also test the upgrade
procedure.
