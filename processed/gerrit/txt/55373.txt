--- Commit Message ---
LU-17916 llite: enable folio writeback support

Linux kernel commit: v6.10-rc7-2-g9aac777aaf945
  filemap: Convert generic_perform_write() to support large folios

Allows traditional file systems that make use of
generic_perform_write() for writing can now opt-in to folio
backed allocations for buffered i/o using __filemap_get_folio()
with fgf_set_order() flag.

This patch switches from page cache allocation to folio
allocation to enable multi-order folio allocation.

Enable large folio support via mount and get/set param:
   lctl get_param llite.*.large_folios
   lctl set_param llite.*.large_folios=[0|1]
