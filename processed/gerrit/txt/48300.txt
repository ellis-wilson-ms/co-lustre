LU-13642 lnet: Allow dynamic IP specification

This change is for the lnetctl app for dynamic configuration.

It allows selecting an interface by specifying an IP address.
These variations of interface and IP address are considered.

1 interface and no IP: Select main IP address
2 no interface and IP specified: Select first interface
         that has the IP address
3 interface and IP specified: Verify that interface and IP match

For further reference please read

IP specification in LNet
https://wiki.whamcloud.com/display/LNet/IP+specification+in+LNet
