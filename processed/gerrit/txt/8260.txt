--- Commit Message ---
LU-4248 tests: parametrize racer progs and file_create.sh

In racer/racer.sh allow RACER_PROGS to be overridden from the
environment. In racer/file_create.sh allow the block size, max block
count, and max stripe count to be overridden from the environment. Set
the default block size and max block count to the previously used
vaules. Set the default the max stripe count to the number of
OSTs. Use $((RANDOM % (max_stripe_count + 1))) for the stripe count
passed to lfs setstripe, ensuring that fully striped files will be
occasionally created.
