--- Commit Message ---
DDN-77 osd: add OI file duplication functionality

2.1 OI file may have consumed lots of space because it didn't release
free blocks, to shrink OI file, add OI file duplication functionality.
The duplicated OI file only contains valid records, so the size is
much smaller than old one.

During OI file duplication, if new files created, the newly added OI
records may get missing, so it's suggested to umount lustre clients
during OI file duplication.

Add a sanity test 223 for this.
