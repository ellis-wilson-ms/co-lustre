--- Commit Message ---
LU-5077 ldlm: remove some shrinker trace

Remove DLMTRACE debug statements from ldlm_pools_count() and
ldlm_pools_shrink(). Under low memory conditions these functions get
called repeatedly which in turn causes the debug log to consist only
of the output of these statements.
