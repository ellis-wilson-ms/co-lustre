LU-2808 lod: create striping on truncate

Truncating a stripeless file to a non zero size will now create
striping, alleviating the need to truncates from OSP layer.  Delete
test sanity 27q (append to truncated file with all OSTs full should
fail) and rewrite sanity 34a-d (checking if various situations create
striping on a file).
