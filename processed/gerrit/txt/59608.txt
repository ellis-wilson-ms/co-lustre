LU-13643 mdt: add active writers counter

Add a counter to track the number of granted ACTIVE_WRITER
locks on each mdt_object. This provides visibility into
how many active writer locks are currently held on a given
file.

Implementation details:
- Increment counter when ACTIVE_WRITER lock is granted in
  mdt_intent_active_writer()
- Decrement counter when ACTIVE_WRITER lock is cancelled using
  custom blocking AST

The counter follows the same pattern as existing mot_lease_count
and mot_open_count counters. It uses atomic operations for
thread safety and includes debug logging for troubleshooting.

A custom blocking AST (mdt_active_writer_blocking_ast) is used
specifically for ACTIVE_WRITER locks to handle the counter
decrement when locks are cancelled. This ensures accurate
tracking of active writer lock state.

This counter will enable the required layout transitions by
letting the MDT know when they are required.
