--- Commit Message ---
LU-5984 obd: fix lastid file name in compat code

The lastid_compat_check() tries to find last_id file from
pre-2.4 Lustre with wrong name, so lastid counter in the new
LASTID file is initialized with wrong value. This causes FID
re-use and inability to create new files in given sequence.

Patch contains also fix for possibly damaged filesystem, it
reads old lastid file always to check if it contains bigger
lastid value.
