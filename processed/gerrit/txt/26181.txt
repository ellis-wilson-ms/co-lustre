LU-9254 llite: Add ladvise dropcaches

For benchmarking purposes, it is extremely useful to have
the ability to drop the caches on the server from the
client.

This patch allows elevated users on the client to tell the
OSSes hosting the OSTs for a particular file drop their
caches.
(/proc/sys/vm/drop_caches)
