--- Commit Message ---
LU-18876 build: de-autotool the LNet Makefiles

We can use plain Makefiles for the LNet kernel build.
Rather than including Rules in each Makefile.in, we can
set the cflags globally in the top-level Makefile. This
allows us to remove the explicit INCLUDE_RULES in each
Makefile.in. Also - define proper CONFIG options for
use in these Makefiles.
