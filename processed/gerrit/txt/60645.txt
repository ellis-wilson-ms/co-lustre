--- Commit Message ---
LU-12187 lfs: add EC setstripe support with X+Y notation

Add support for Erasure Coding (EC) layouts in lfs
setstripe using simple "X+Y" notation where X is data
stripe count and Y is parity stripe count.

Usage examples:
  lfs setstripe --ec 4+2 -E EOF /mnt/lustre/ec_file
  lfs setstripe --ec 6+2 -S 1M -E EOF /mnt/lustre/ec_1m
  lfs setstripe --ec-stripe-count 4+2 -E EOF /mnt/lustre/ec_long
