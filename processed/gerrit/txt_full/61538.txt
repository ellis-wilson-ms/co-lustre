LU-19296 changelog: retrive changelog user info from MDT

This patch uses RPC MDS_GET_INFO with new key KEY_CHANGELOG_USER
to retrive changelog user information on server side:
- Add new handler mdt_changelog_get_user_info() to extract RPC
  request and forward data to MDD layer by mdd_iocontrol().
- Use mdd_iocontrol() with new ioctl OBD_IOC_CHANGELOG_FILTER to
  call new function mdd_changelog_user_lookup() to implement
  changelog user lookup in MDD layer.
