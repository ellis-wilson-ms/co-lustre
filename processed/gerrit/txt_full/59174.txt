LU-18996: Fix race condition in obd_device_list_seq_show

When calling obd_device_list_seq_show, if a
class_unregister_device is happening at same
time, the race condition can cause unnecessary
LBUG when asserting obd->obd_minor == num.

In this case no need to LBUG, just return NULL
and behave like the obd device does not exist.
