--- Message (Andreas Dilger, 2012-11-02 18:43:32.000000000) ---
Patch Set 1: I would prefer that you didn't submit this

(1 inline comment)

Instead of all the inline #ifdefs, it would be better to have a compat wrapper for this code, like:

#ifdef HAVE_DENTRY_D_ALIAS_HLIST
#define ll_d_hlist_node hlist_node
#define ll_d_hlist_empty(list) hlist_empty(list)
#define ll_d_hlist_entry(ptr, type, name) hlist_entry(ptr, type, name)
#define ll_d_hlist_for_each(tmp, i_dentry) hlist_for_each(tmp, i_dentry)
#define ll_d_hlist_for_each_entry(denty, p, i_dentry, alias) \
        hlist_for_each_entry(dentry, p, i_dentry, alias)
#else
#define ll_d_hlist_node list_head 
#define ll_d_hlist_empty(list) list_empty(list)
#define ll_d_hlist_entry(ptr, type, name) list_entry(ptr, type, name)
#define ll_d_hlist_for_each(tmp, i_dentry) list_for_each(tmp, i_dentry)
#define ll_d_hlist_for_each_entry(denty, p, i_dentry, alias) \
        p = NULL; list_for_each_entry(dentry, i_dentry, alias)
#endif

The "p = NULL" is an attempt to quiet the "unused variable" warning, not sure if it will work.
