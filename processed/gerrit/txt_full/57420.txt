LU-18550 osd-ldiskfs: oied_name is not null-terminated

osd_it_ea_dirent->oied_name is not null-terminated,
we need to be careful when using it.

kstrtox expects the string must be null-terminated, using
oied_name directly could give us a wrong result, contruct
a null-terminated string for it.
