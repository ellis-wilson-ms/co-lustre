LU-19230 obdclass: mirgate from mount to fs_context API

Reports of memory pressures from excessive inodes for Lustre
exist. The solution is to use the kernels super block shrinker
but that is only available when using the fs_context API.
Do the most basic port to fs_context. This gives us a default
inode shrinker for free.
