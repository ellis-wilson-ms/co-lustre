--- Commit Message ---
LU-19996 utils: Pass negative errno to idd_err

It's confusing that writing data with a specified errno (idd_err)
into identity_info interface always returns the specified errno.
It should return -1 and set errno to the specified errno instead.

Also add comments to remind how to use sid_err, scd_err and idd_err.
