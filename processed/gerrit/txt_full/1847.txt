It isn't clear why this patch was resubmitted, rather than just restarting the testing? This is especially important for the OFD patches and others landing from orion, since they are making large changes to the OST stack and have the chance to convert large parts of this code to use tabs without the need for useless tab-only patches.

It also makes sense to begin using tabs for indentation in orion, but this can be done on a patch-by-patch basis so that it doesn't disrupt all of the ongoing work.
