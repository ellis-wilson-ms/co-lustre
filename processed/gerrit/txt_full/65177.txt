LU-20100 obdclass: max_cached_mb module parameter

Add module parameter to lustre to allow default setting of
max_cached_mb

Use max_cached_mb to set the initial for
  lctl set_param llite.*.max_cached_mb=<MB>

Ex: /etc/modprobe.d/lustre.conf contains:
options lustre max_cached_mb=6173
