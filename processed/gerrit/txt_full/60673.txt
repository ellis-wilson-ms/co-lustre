--- Commit Message ---
LU-18357 tests: fix interop for multiple fileset syntax

Multiple filesets use a new syntax that is incompatible with the
previous single fileset representation. Depending on the client and
server versions, this can cause tests to fail as the can't validate a
fileset was set correctly.

This patch adds better version checks and transparently handles the
new and old syntax in "wait_update_facet_fileset()",
"wait_nm_sync_fileset()", and "do_facet_check_fileset()".

Fixes: 40cd140b85 ("LU-18357 ptlrpc: Support add and del alternate filesets")
