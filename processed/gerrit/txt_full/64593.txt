LU-12187 llite: default FLR EC module params to off

EC connect flag module parameters should default to
disabled so the feature is off in production until
explicitly enabled. The test framework already sets
them to 1 via MODOPTS_MDT and MODOPTS_LLITE defaults
in load_modules_local.

Set a tombstone to raise an error if the FLR EC
connect flag is still off after 2.17.90.
