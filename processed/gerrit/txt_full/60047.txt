LU-17231 doc: Create script to generate man pages

Created generate-param.sh script that can be used to generate
the bare bones of a man pages with the appropriate sections included.

In particular this script will be used to create pages for the new
lustre parameter man pages, which there are several hundred.

This script can be used as follows:
  generate-manpage.sh NAME SECTION [OUTPUT_FILE]
or
  generate-manpage.sh [DIR/]NAME.(1-8)

for example:
  generate-manpage.sh lctl-list_param 8
will generate a single man page for the command 'lctl list_param'
under the folder Documentation/man8.

or

  generate-manpage.sh lctl-list_param 8 lustre/doc/lctl-list_param.8
will generate the man page in lustre/doc/lctl-list_param.8 instead
of the default directory.

or

  generate-manpage.sh lctl-list_param.8
will generate the man page in lctl-list_param.8 in the current
directory.
