LU-18687 debug: restore option to disable cdebug

Currently, --disable-libcfs-cdebug does not compile (both because of code
drift and because of an explicit #warning). Reimplement --disable-libcfs-cdebug
to disable the Lustre debug buffer _only_ and implement the remaining features
using printk() debugging. Remove the explicit warning as well.
