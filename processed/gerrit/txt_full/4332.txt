--- Commit Message ---
LU-1187 osd: Do not add remote fid to idmap cache.

OI cache suppose only includes <local fid, local ino>, if we add
<remote_fid, agent ino> to the cache, it might confuse osd in some
cases, i.e. get an object through oi_lookup by a remote fid.
