LD-19066 obdfilter: add failure_domain field

Do not merge. Has been replaced by a different patch based on master
so it can go in concurrently with the ec series.
Just here so that we can base the later stripe allocator changes
ontop of the stripe allocator changes that are part of the ec
series and which we depend on.

Add a failure_domain field to the device.

This can be set on a per-device basis using:
  lctl set_param obdfilter.lustrewt-OST0000.failure_domain=0x55
and will be used to allow LFR stripe allocation
to become failure_domain aware.
