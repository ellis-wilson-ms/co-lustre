I think it makes more sense to leave parse_size() to parse arbitrary passed strings, and then check in the caller when a negative value is not permitted.

static int lfs_setstripe(int argc, char **argv)
{

        /* get the stripe size */
        if (stripe_size_arg != NULL) {
                result = parse_size(stripe_size_arg, &st_size, &size_units, 0);
                if (result || st_size < 0) {
                        fprintf(stderr, "error: %s: bad stripe_size '%s'\n",
                                argv[0], stripe_size_arg);
                        return result;
                }
        }

The error message is also improved a bit to report that this is the stripe size.
