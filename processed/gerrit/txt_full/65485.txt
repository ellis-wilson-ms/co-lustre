--- Commit Message ---
LU-20178 tests: add compression test suite

Add sanity-compr.sh test suite for CSDC with 19 test cases
covering:

- Basic write/read with each compression algorithm
  (lz4, lz4hc, lzo) and all supported chunk sizes
- Compression ratio validation ensuring compressed data
  is smaller than uncompressed for compressible content
- Incompressible data handling (random data)
- Mixed I/O patterns: sequential, strided, append
- Edge cases: partial chunks, exact chunk boundaries,
  files smaller than one chunk
- Multi-component layouts with compression on selected
  components
- Integration with existing Lustre features: OST pools,
  striping configurations
- Stress tests: concurrent compression from multiple
  clients
