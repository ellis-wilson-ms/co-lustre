LU-12188 uapi: add LOV_PATTERN_PARITY

Both parity and I/O data use struct lov_mds_md but how that
information is handled is very different. To make the code
aware of how to treat the struct lov_mds_md data we create
the LOV_PATTERN_PARITY to pattern. LOV_PATTERN_PARITY will
always be used with LOV_PATTERN_RAID0 so much of the lov
code can be used for both parity and normal I/O data.
