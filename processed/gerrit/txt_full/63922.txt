LU-19867 ofd: quota enforcement for offset local root

Root users can bypass quota restrictions if the root UID is not
squashed and the CAP_SYS_RESOURCE capability is set on the client.
However, when nodemap offsets are used and the local_admin rbac is not
set on the nodemap, quota restrictions should still be enforced for
the client root user. If the local admin rbac is set, quota
restrictions can be bypassed.

Currently, the local_admin rbac is not considered on the ofd-side and
the mdt-side (for DOM files) during writes so that the
OBD_BRW_SYS_RESOURCE flag is dropped for each page when the
local_admin rbac is not set.

This patch adds this check in ofd and mdt and adds sanity-sec
test_64i.

Fixes: f7495bc57f ("LU-18694 sec: nodemap local root user capabilities")
