--- Commit Message ---
LU-432 Fixes error: ‘struct sock’ has no member named ‘sk_sleep’

Starting from kernel 2.6.35 sock struct don't include the sk_sleep
field. Instead there is the sk_sleep() function to provide this
information. This checks whether the sock struct provide directly
support for sk_sleep and if not uses the sk_sleep() function.
