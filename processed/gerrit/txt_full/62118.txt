LU-19469 osc: add tunable for size threshold to cache hole

Add a tunable for the minimum page count threshold to determine
whether cache a hole extent on the rbtree of an OSC object on a
client.
	lctl set_param osc.*.hole_extent_kb=128

The above command will set it with 32 pages (128KiB in size).
The default value is 64KiB.
