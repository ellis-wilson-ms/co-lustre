LU-19724 build: Compatibility updates for kernel v6.18

v6.17-rc1-37-gf99b3917789d
  fs: rename generic_delete_inode() and generic_drop_inode()

Use inode_just_drop() and inode_generic_drop() and provide:
  inode_just_drop() wrapper for generic_delete_inode() and
  inode_generic_drop() wrapper for generic_drop_inode()
for older kernels.

v6.17-rc4-147-g53fbef56e07d
  mm: introduce memdesc_flags_t
struct page.flags is now a struct, use struct page.flags.f

v6.17-rc4-6-g180a9cc3fd6a
  make it easier to catch those who try to modify ->d_name
d_name is const, propagate const as needed, use __d_name for
changing d_name contents.

v6.17-rc4-90-g2f7d98f10b8f
    Have cc(1) catch attempts to modify ->f_path
f_path is const, propagate const as needed, use __f_path for
changing f_path contents.

CONFIG_FS_VERITY only build verity.o if enabled.

v6.2-rc5-72-g5e6a51787fef
  uuid: Decouple guid_t and uuid_le types and respective macross
Kernel vs util: uuid_le vs guid_t breaks.
