LU-15692 tests: clean up sanity test_316/test_319

These tests use the deprecated "lfs mv" insted of "lfs migrate".
Also, test_316 is not specifying which MDT to create 'd' on, so
MDT space balancing may result in 'd' created on MDT0001 and this
defeats the purpose of the test trying to migrate it to MDT0001.

Also fix some minor script style issues.
