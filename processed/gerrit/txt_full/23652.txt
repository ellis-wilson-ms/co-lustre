LU-8812 test: add failback test with errors=panic

This is test for MRP-1658 which create activity and
emulate failback operations. Option errors=panic
is using similar to production servers. Test requires
ENABLE_QUOTA=yes option.

Intel_bug_id: SEA-107
Seagate_bug_id: MRP-1658

It is unnecessary to wait IO process in test_92. It
may spend a long time to finish on a big OST.

Test-Parameters: trivial  envdefinitions=ENABLE_QUOTA=yes,ONLY=92  clientcount=4 osscount=2 mdscount=2 mdtcount=1  austeroptions=-R failover=true iscsi=1  testlist=replay-single
