Patch Set 3: Looks good to me, approved

The only question about the patch testing is whether the patch allows mounting a filesystem of just below 24TB without any extra mount options?  It looks like it should be OK.

The other question is whether this should allow filesystems exactly equal to 24TiB in size?  We did test that worked OK, though for real "3TB" disks they will always be slightly smaller than 3TiB, so an 8+2 RAID-6 LUN will also be smaller than a full 24TiB in size and it probably doesn't matter. 

Approving this as-is, so that we can land it sooner. None of these issues are blockers.

Since this patch has already been merged I don't think any additional change is needed.  In the past we wanted to restrict filesystems to be strictly less than 2^32 blocks.  For the future it is probably better to use "> {tested size}" instead of ">=" since it is likely that LVM LVs and other LUNs will have exact power-of-two sizes.
