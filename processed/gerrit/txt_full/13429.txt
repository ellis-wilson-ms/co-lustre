--- Commit Message ---
LU-5628 build: undefine macro to avoid conflicts with kernel

When building Lustre client on a kernel which has already merged
Lustre client codes, there will be definition conflicts between
kernel and external Lustre codes. This patch undefines existing
definiton of kernel in config.h to avoid this problem.
