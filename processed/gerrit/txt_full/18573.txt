LU-7801 libcfs: add function level debug message filter

Here add a function level debug message filter support, idea comes
from ftrace.
A new proc interface named "func_filter" is added, which is used to
control the function filter, it supports regular expression (only
support just simple ones), for example:
^ll_	---> front pattern, functions start with "ll_"
_end$	---> end pattern, functions end with "_end"
read	---> full pattern, functions contains "read"
and you can add/delete pattern with symbol +/- prepend, for example:
+^ll_	---> add regular "^ll_" pattern
-_end$	---> remove regular "_end$" pattern

This version is still a WIP one, any comments are welcome.
