--- Commit Message ---
LU-15234 lnet: add mechanism for dumping lnd peer debug info

Add ability to dump peer debug info on lnd level:
	lnetctl debug peer --prim_nid=<nid>

The debug info is dumped to the log as D_CONSOLE and can be
retrieved with "lctl dk".
This mechanism hasbeen added for socklnd and o2iblnd peers.
