--- Commit Message ---
LU-16335 header: implement native strchrnul()

LC_IOC_REMOVE_ENTRY in configure fails because strchrnul() is
_GNU_SOURCE only, and any program including lustre_user.h
(even indirectly) need to define _GNU_SOURCE, otherwise
compile will fail. Replace strchrnul() with a native implementation
ll_strchrnul().
