--- Commit Message ---
LU-18977 efalnd: set hstatus on error before kefalnd_tx_done()

Currently, kefalnd will not return a proper LNet error
if the connection establishment fails. The remote NI
will not be put into recovery, which could lead to LNet
attempt to connect to it indefinitely.

Set hstatus in other cases before kefalnd_tx_done(), as
needed.
