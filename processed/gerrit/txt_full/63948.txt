LU-12669 tests: Add EC recovery tests for varied failures

Add test_43 series for testing EC recovery with different
numbers of failed OSTs:
- test_43a: Single OST failure (minimal case)
- test_43b: Maximum recoverable failures (2 OSTs)
- test_43c: Exceeding max failures (3+ OSTs, expect EIO)
- test_43d: Progressive failures (sequential)

Add test_44 series for testing EC error handling:
- test_44a: Too many OST failures, verify EIO and recovery
- test_44b: Specific parity vs data OST failures
- test_44c: Read errors during recovery (fail_loc injection)
- test_44d: Invalid/corrupted EC configurations

All tests use 4+2 EC configuration with 1M stripe size
and 8MB files. Tests verify correct error handling and
data integrity during various failure scenarios.
