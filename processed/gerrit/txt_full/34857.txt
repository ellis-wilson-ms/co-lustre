--- Commit Message ---
LU-9667 lnet: Create new structure for sysfs peer kobjects

This patch creates a new structure lnet_sysfs_peer for the
peer_ni kobjects. This is needed so that the cleanup of
kobjects and the peer_ni structure can be done separately.
The peer_ni structure is deleted within the spinlock
lnet_net_lock but kobjects cannot be cleared within this lock
because in the kobject_put() api, mustex is used which when done
inside a spin_lock gives error.

To avoid this locking issue, peer_ni kobjects are added to a
sysfs_peer_zombie list which is then traversed and cleanedup
outside the lnet_net_lock.
