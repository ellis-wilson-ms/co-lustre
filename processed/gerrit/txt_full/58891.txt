LU-18932 tcu: implement user command lfs trash unrm

Implement a user command "lfs trash unrm $path" to recover files
or directories from Trash Can.

The path name recovering from Trash Can should be in the form:
- "$lustre_path/.Trash/basename": Recover the top level sub file
  specified by @basename under the stub dir ".Trash";
- "$lustre_path": If only specified a Lustre file path (a Lustre
  directory), then it will recover all files under ".Trash" in
  Trash Can.

Add tese case sanityn/test_117k to verify it works as expected.
