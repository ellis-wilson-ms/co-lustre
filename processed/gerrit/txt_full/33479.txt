--- Commit Message ---
LU-11573 osd: enable large bios

currently BIO_MAX_PAGES limits number of pages in a single bio to
256 limiting IOs submitted by osd-ldiskfs by 1M. bump this limit
to 1024 pages (4MB IOs).

brw_stats in KVM with /dev/vdb mapped to /dev/ram:

                           read      |     write
disk I/O size          ios   % cum % |  ios         % cum %
32K:		         0   0   0   |    1   1   1
64K:		         0   0   0   |    0   0   1
128K:		         0   0   0   |    0   0   1
256K:		         0   0   0   |    0   0   1
512K:		         0   0   0   |    1   1   2
1M:		         0   0   0   |    0   0   2
2M:		         0   0   0   |    1   1   4
4M:		         0   0   0   |   64  95 100
