--- Commit Message ---
LU-4285 kernel: fix unintialized variables in kernel patch

One of the SLES base kernel patches leaves local variable un-inited
in __dquot_initialze().  This can lead to kernel OOPS during common
operations like mkfs.lustre and mount.
This fix alters the kernel patch to init variables properly.
The same patch is used for both SP2 and SP3 kernels.
