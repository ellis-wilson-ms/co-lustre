LU-15234 lnet: add debug info for lpni refcounts

In order to narrow down the issue with some peer refcounts
going too high on router nodes, count every instance when
lpni refcount is changed separately.

Add ability to report this detailed account via lnetctl:
	lnetctl show peer -v 5
