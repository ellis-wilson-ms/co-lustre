--- Commit Message ---
LU-5060 zfs: execution fails due to fix lib. name libzfs.so.1

Executing mkfs.lustre with --backfstype=zfs seeks for
libzfs.so.1 via handle_libzfs = dlopen("libzfs.so.1", RTLD_LAZY);
The current version of ZFS 0.6.2-274_g2c33b91 compiles version
libzfs.so.2.0.0 and create links libzfs.so -> libzfs.so.2.0.0,
libzfs.so.2 -> libzfs.so.2.0.0 which will not be found by
the fix library name provided to dlopen.
