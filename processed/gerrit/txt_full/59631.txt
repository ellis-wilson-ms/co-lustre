--- Commit Message ---
LU-13643 mdt: add FLR transition for immediate mirrors

Add automatic FLR state transition from WRITE_PENDING to
RDONLY when the last active writer lock is released on
immediate mirror files.  Un-stale all stale mirrors.

Once error handling is implemented, we will only unstale
mirrors with no errors.

Key features:
- Only operates on immediate mirror files
  (LCM_FL_IMMEDIATE_MIRROR)
- Transitions from WRITE_PENDING to RDONLY when active
  writer count drops to zero
- Uses existing layout update infrastructure
- Uses workqueue rather than callback context
- Adds a new layout intent used only on the server side
