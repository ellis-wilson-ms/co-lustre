LU-14633 lnet: have lnet layer use struct iov_iter

The native Linux client LNet layer was updated to use
struct iov_iter but due to RHEL7 support was never backported.
Now that RHEL7 support is dropped we can forward port Al Viro's
work. Due to delay to land that work much of the original
code has evolved over time. The LNet code is brought up to
sync with the current native Linux Lustre client
