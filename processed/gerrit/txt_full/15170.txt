LU-6531 lnet: DLC interface for o2iblnd Channel Bonding

This patch migrates the Channel bonding interface to the DLC API.
o2ibs can be added, showed and checked via lnetctl, which uses
liblnetconfig.

The original lctl and portals interface has been removed.  All future
LNet/LND configuration should only be available via DLC.

Signed-off-by: Amir Shehata <amir.shehata@intel.com>
