--- Commit Message ---
LU-19998 tcu: add support for notrash tag on symlink

This patch adds support for notrash tag on symlink files.
Add a new option "--dereference | -L" to lfs chattr command to
dereference the symlink file (follow links).

It also adds a LL_IOC_FSFLAGS IOCTL to support setting/getting
file flags on a some special file such as symlink which can not
operate via fgetflags/fsetflags (i.e. lsattr/chattr commands).

Add a test case sanityn/test_118p to verify the notrash tag on
symlink file.
