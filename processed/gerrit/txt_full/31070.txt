LU-10530 utils: Scanning tool for HSM POSIX storage

This tool is design to scan the directory tree of the HSM
storage and cleanup the directory tree.
Also, since PCC(LU-10092, LU-10499) also use the HSM directory
structure, this tool could also be used for PCC to:
1) scan the whole PCC storage to detach all the cached files
when it is not suitable any more to cache the files on PCC,
e.g. when a job stops and another job is starting to use PCC.
2) apply some kind of policy (e.g. based on UID/GID/ProjID, etc)
for PCC cache management.
These use cases needs full scanning of the HSM storage directory.
