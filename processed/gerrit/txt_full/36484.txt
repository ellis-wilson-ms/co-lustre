LU-12827 ldiskfs: fix ldiskfs_xattr_inode_iget error checking

In ldiskfs_xattr_inode_iget, ldiskfs_iget could return an
error value and calling is_bad_inode on it will crash the server.

The problem has already been addressed in upstream by
bab79b04999c ext4: change ext4_xattr_inode_iget() signature,
merge the commit to our ext4-large-eas.patch.
