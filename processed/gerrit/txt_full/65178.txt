LU-20100 obdclass: module parameter osc_max_dirty_mb_default

Add module parameter to obdclass to allow default setting of
  osc_max_dirty_mb_default to override the fix value
  OSC_MAX_DIRTY_MB_DEFAULT=2000

Use osc_max_dirty_mb_default to set the initial for
  lctl set_param [osc|mdc].*.max_dirty_mb=<MB>

Ex: /etc/modprobe.d/obdclass.conf contains:
options obdclass osc_max_dirty_mb_default=1500
