LU-14021 llite: add asserts when trying to access ll_file_data

Normally, ll_file_data is held together with struct file,
we hit a case where file looks file, but @ll_file_data
has been freed and assigned as NULL pointer.

This patch try to add some assertions to figure out
if some race happen here.
