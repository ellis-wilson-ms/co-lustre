LU-5851 scripts: build osd-related mount dso in DKMS

Since LU-4606 changes, an osd-specific dso for mount command is
shipped as part of lustre-osd RPMs. This breaks Lustre Server
DKMS packaging which builds osd module but not this library.

This patch allows this library to be also built and installed by
Lustre Server DKMS RPM.
