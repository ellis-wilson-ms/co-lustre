LU-5030 utils: create llapi_{get,set}_param function

Abstract out {get,set}_param pathname generation so that it is
common between the different parts of liblustreapi by creating
new functions called llapi_{get,set}_param. This reduces code
duplication as well as simplifies changes to the /proc
parameter location in the future.
