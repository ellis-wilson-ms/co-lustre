LU-1027 build: more flexibility for building OFED

Introduce new variable to target file
INFINIBAND={none,inkernel,ofed}
none: do not build IB
inkernel: build using kernel ib driver
ofed: build ofed using OFED_VERSION

We modify lbuild to support these scenarios

1. if we don't use any --with-ib, --ofed-version
lbuild will use target file
2. if we use only --with-ib={none,inkernel,ofed}
lbuild will build using info provided. In case of
ofed, it will use OFED_VERSION in target file
3. if we use --ofed-version=x.x.x, lbuild will
build x.x.x and ignore target's file
