LU-2456 lnet: Dynamic LNet Configuration (DLC) Feature

This is the first patch of a set of patches that enables DLC.

This patch adds the following features to LNET.  Currently these
features are not driven by user space.
- Enabling Routing on Demand
- Adjusting router buffer pools
- Adding/deleting routes dynamically
- Adding/Deleting Networks dynamically

Comments on handling the ping info:

A suggested solution is to create a new ping info that contains the new information and replace the pointer of the md from the old pointer to the new pointer.  This will require some changes in the md infrastructure.

so basically we'd have:
lnet_replace_ni_ping_info(ni)
  . create a new lnet_ping_info_t
  . which will have the new information after the addition or removal of the ni.
  . convert the lnet_ping_info md handle to an md and replace the pointer to the start of the ping info, to the new one, and delete the old one.

This will require some changes to the md infrastructure, to be able to make this modification.
