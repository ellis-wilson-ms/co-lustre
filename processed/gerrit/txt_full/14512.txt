LU-6245 lustre: include kthread.h

Soon the libcfs wrappers for linux primitives will be removed
and any code needed to use kthread will need to include the
linux header kthread.h. While testing DNE2 along side the
libcfs cleanup kthread.h was missing so this patch adds the
needed header to avoid a potential build issues when both
works will land.
