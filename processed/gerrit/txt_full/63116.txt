--- Commit Message ---
LU-19733 ptlrpc: prefer NIDs on local networks

Sort NID list provided by IR moving NIDs on local network
ahead. This allows to mitigate NID filtering problem in
a setup when client configured networks are subset of
server networks, e.g. multi-tenancy

Patch does the following:
- create/export LNetHasLocalNet() supplimentary function
- MGC re-arranges incoming NIDs from IR moving NIDs on
  local networks ahead, so they are always tried first
  for a new connections
