--- Message (Hongchao Zhang, 2011-04-08 03:42:19.000000000) ---
Patch Set 1: I would prefer that you didn't submit this

the existing patch in b1_8 contains several problems,
1, in mount_facet, "do_facet" will look for the host of the active
facet, but the $(facet_host ${facet}) used by the patch won't
take it into account.

2, remote_node should only take one host as its argument, and
furthermore, the do_rpc_nods already deal with the case of 
remote nodes

how about the following one,

set_default_debug_facet() {
    local HOST=`facet_active_host $1`
    [ -z $HOST ] && echo No host defined for facet $1 && exit 1
    do_rpc_nodes $HOST set_default_debug
}

set_default_debug_nodes() {
    local nodes=$1
    do_rpc_nodes $nodes set_default_debug
}
