--- Commit Message ---
LU-17501 libcfs: allow CPT exclude list for cores

Allow a CPU core exclude list to be specified when constructing the
CPT distribution instead of having to specify all of the CPU cores
except the ones that should be avoided. The exclude list is set by
specifying 'C' for the CPT number in /etc/modprobe.d/lustre.conf:

    options libcfs cpu_pattern="N C[0-3]"

This would exclude cores 0-3 on each NUMA node from use by Lustre.
Additionally to exclude specific cores, specify 'X' for the CPT
number. Note that this option is equivalent to 'C' if 'N' is not
specified.

It isn't possible to specify both include and exclude lists at the
same time, but it doesn't really make sense to do this anyway.
