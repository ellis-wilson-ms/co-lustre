--- Commit Message ---
LU-149 tests: enable async journal commit

Currently, the async journal commit feature (bug 19128) and
the cancel lock before replay feature (bug 16774) are disabled
by default on b1_8 branch.

This patch is to enable them by default so as to trigger
Jenkins build and autotest regression test run.

Test-Parameters: fortestonly \
envdefinitions=SLOW=yes,ENABLE_QUOTA=yes \
clientdistro=el6 serverdistro=el5 clientarch=x86_64 \
serverarch=x86_64 testgroup=full

Signed-off-by: Jian Yu <jian.yu@intel.com>
