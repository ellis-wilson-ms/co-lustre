LU-12345 llite: implement IT_WRITE_LEASE intent support

Add support for IT_WRITE_LEASE intent to enable write lease
functionality in Lustre. This implementation provides the
foundation for write lease operations by adding the necessary
client-side and server-side infrastructure.

Client-side changes:
- Add IT_WRITE_LEASE intent handling in ll_file_write_iter()
- Implement write lease acquisition before write operations
- Use ll_intent_lock() to send IT_WRITE_LEASE to server

Server-side changes:
- Add IT_WRITE_LEASE case in mdt_intent_opc() routing
- Implement mdt_intent_write_lease() handler function
- Use MDS_INODELOCK_WRITE_LEASE lock bits for write leases
- Fix client-side intent packing to use mdc_intent_getattr_pack()

The implementation acquires write leases during write operations
and properly routes them through the existing intent lock
infrastructure. This provides the foundation for future write
lease features and conflict detection.
