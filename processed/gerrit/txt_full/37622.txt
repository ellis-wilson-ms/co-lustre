LU-9091 sysfs: add memparse format support for defunit

Currently sysfs_memparse() acts as a wrapper to take a pure
numeric string or a string with memparse() units i.e B,K,M,G etc
and map it to what string helpers handles. In the case of a pure
numeric string a defunit unit is merged to the string. For defunit
the current code only accepts string helpers formats but its been
request to expland this to cover the units passed to memparse()
as well. To do this we expand string_to_size() to handle memparse
units as well. This lessens the need to modify the input string
like what sysfs_memparse() does today.
