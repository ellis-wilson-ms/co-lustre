--- Commit Message ---
LU-10602 llite: add file heat support

File heat is a relative attribute fo files/objects which reflects
the access frequency of the files/objects.
File heat mainly desinged for cache management. Cache like PCC
could use file heat to determine which files to removed from
the cache or which files to fetch into cache.
This patch adds file heat support on llite level.
