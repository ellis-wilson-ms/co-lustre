LU-13643 lfs: add --mirror-flags option to lfs find

Add support for finding files based on mirror flags using
the new --mirror-flags option in lfs find. This allows
users to search for files with specific mirror flags such
as immediate mirroring.

The implementation adds:
- New --mirror-flags option to lfs find command
- Support for "im" (immediate) and "ro" (readonly) flags
- Negation support with ! or ^ prefix
- Integration with existing mirror search functionality
- Comprehensive test coverage in sanity.sh

Examples:
  lfs find --mirror-flags=im /mnt/lustre
  lfs find ! --mirror-flags=ro /mnt/lustre

This complements the existing --mirror-state and
--mirror-count options for comprehensive mirror file
searching capabilities.
