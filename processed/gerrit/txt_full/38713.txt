LU-13601 llite: OBD_ALLOC_LARGE for ll_sb_info

struct ll_sb_info needs an order 4 page allocation,
which fails if memory is fragmented.
Changing OBD_ALLOC_PTR to OBD_ALLOC_LARGE allows to
fail back to vmalloc() if kmalloc() fails.
