--- Commit Message ---
LUS-12215 utils: gcc 12 warnings in utils

gcc 12 uncovers several warnings in utils.

A few cases where snprintf() could truncate.

Rework these to check the result rather than length check the
inputs to make gcc happy.

A few cases where a fixed size buffer is treated as a nullable
pointer.
Make sure the buffer can be an empty string and check for an
empty string instead.

One case of use-after-free in a list interation, update to
use list_for_each_entry_safe().
