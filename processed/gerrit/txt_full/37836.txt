LU-13346 osd-zfs: set attr stores LUSTRE_LMA_FL_MASKS

Fix a race condition exposed by sanityn test 110.
This race between link and rename triggers a file system corruption.

The root cause is the LUSTRE_ORPHAN_FL flag not being propagated
correctly in the zfs-osd.
