LU-0000 llite: complete IT_WRITE_LEASE implementation

Complete the IT_WRITE_LEASE intent implementation by adding
missing support in core Lustre infrastructure functions.

This patch adds IT_WRITE_LEASE support in it_to_lock_mode()
returning LCK_EX, IT_WRITE_LEASE string representation in
ldlm_it2str(), and client-side write lease acquisition in
ll_file_io_generic().

The IT_WRITE_LEASE intent now works end-to-end, allowing
clients to acquire exclusive write lease locks during write
operations. This provides the foundation for concurrent write
functionality in mirrored files.

Testing shows the implementation works without kernel crashes
and properly acquires/releases write lease locks during file
write operations.
