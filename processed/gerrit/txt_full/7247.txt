(5 inline comments)

Note that the file handle syscall support is really a function of the kernel and not of the glibc.  It is possible to call syscalls that are not defined in glibc, like:

 #ifdef HAVE_PRCTL
        if (prctl(PR_GET_DUMPABLE, 0, 0, 0, 0) == 0)
                return NULL;
 #elif (defined(linux) && defined(SYS_prctl))
        if (syscall(SYS_prctl, PR_GET_DUMPABLE, 0, 0, 0, 0) == 0)
                return NULL;
 #endif /* HAVE_PRCTL */

This would provide earlier test coverage on kernels that support these syscalls before we actually get to newer distributions.
