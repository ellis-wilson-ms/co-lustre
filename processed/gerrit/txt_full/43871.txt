--- Commit Message ---
LU-12262 llite: improve ll_sbi_flags_seq_show()

Fix the LL_SBI_* flags printing so that the addition of new flag
values doesn't get out of sync with the names for those flags.
Otherwise, ll_sbi_flags_seq_show() only printed an error message,
and not even the known flag names.

Add enum ll_sbi_flags_bits as a automatic enum to track the feature
bits instead of setting explicit values, since these are internal
use only.  Use the bits to index into LL_SBI_FLAGS_NAMES and add
build-time check to verify that all bits have proper names so that
they can't get out of sync, and there is no need for placeholders.

Fixes: 4af3ab1945fd ("LU-2017 mdc: add layout swap between 2 objects")
