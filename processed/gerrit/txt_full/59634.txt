--- Commit Message ---
LU-0000 mdt: add automatic RDONLY to WRITE_PENDING transition

Implement automatic FLR state transition from RDONLY to
WRITE_PENDING when the first active writer is acquired
on immediate mirror files. This completes the server-side
automatic state management for concurrent write operations.

When mdt_active_writer_inc() increments the counter from 0
to 1, it triggers mdt_flr_state_transition_to_write_pending()
which transitions immediate mirror files from RDONLY to
WRITE_PENDING state and stales all mirrors by writing to
the entire file extent.

This ensures immediate mirror files automatically enter
the correct state for concurrent write operations without
requiring explicit client commands.
