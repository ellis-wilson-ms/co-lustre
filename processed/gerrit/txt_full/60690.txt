LU-12345 docs: add '%k' format to lfs-find.1 man page

Document the '%k' format for the 'lfs find --printf' option, which
prints the number of 1K blocks allocated for the file.

Also fix a number of style issues and typos in the man page.

Fix the --printf formatting options to be ordered alphabetically
to make the options easier to find.
