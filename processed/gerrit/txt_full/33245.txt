LU-10968 hsm: Create an external HSM coordinator

Create an interface for an external HSM coordinator
Create an example external coordinator lhsmtool_cdt
Create llapi calls for external cdt to communicate
with Lustre

sanity-hsm can be set to use an external coordinator
by specifying COORDINATOR=external on the command line

sanity-hsm starts and stops lhsmtool_cdt along with the
coordinator

You must run the external coordinator on the MDS

Handles all current HSM activities, no changes to any
other program is needed.

Internal coordinator is used by default.
