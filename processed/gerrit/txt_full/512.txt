--- Commit Message ---
LU-287 handle the glimpse callback race with lock cancel

  If lock cancel and glimpse callback happened at the same time on
  client, glimpse callback could read the shrinked kms(which is less
  than actual size).
  When such race happen, we should let the glmipse callback return
  -EINVAL to inform server to update LVB from disk.
