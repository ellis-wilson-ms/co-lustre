LU-8336 osd-ldiskfs: add sanity check for ldiskfs journal size

Lustre need some metadata updates for a single transaction,
which might declare more than 512 blocks credits, this need
at least we have more than 4096 blocks journal size for RHEL7

This patch try to at least for 8192 blocks requirements for
journal size.

RHEL6 don't have problems because it limit max credits as 1/4
of journal size, but RHEL7 codes limit it as 1/8 of journal size.
