--- Commit Message ---
LU-8712 tests: refresh mgs config database before add mds

conf-sanity test 72 failed to start mds because mds failed
with -EADDRINUSE. Registering mdt to MGS while mounting mds
index of mdt was already present in MGS configuration
database so mds mount failed with -EADDRINUSE

Fix:
stop and format mgs when mgs and mds is separate. This will
make sure index of mdt will not be present in MGS config
database.
