--- Commit Message ---
LU-2335 build: add script to copy client kernel code

By running
./build/copy-buildin <client-code-dir> <kernel-tree>
one can copy client code generated by
build/build-client-source.sh into a kernel source tree.

The same script shall also take care of generating proper Kconfig
and Kbuild files for the in-tree Lustre client.

NOTE: the kernel source tree should be the same one that Lustre
is configured against (e.g., via --with-linux= option).
