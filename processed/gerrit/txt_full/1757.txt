--- Commit Message ---
LU-80 striping: large stripe count support

Currently a file can be stripped across OSTs up to a limit of
160 stripes. This patch expands that limit to 2000 and it is
possible to go even to larger stripe counts. The patch applies
against both RHEL5 and RHEL6.
This patch differs from Yu Jian`s patch in:
1) Fixed endians in the ext4 patch
2) Removed changes not related to wide stripe
3) Maximum OST number increased up to 2K
4) Reply buffer size shrinking removed,
because no client side support for this code
5) Fixed MDS buffer size
