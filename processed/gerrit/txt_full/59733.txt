LU-13643 mdt: Eviction detection for AW locks

We need to know when an active writer lock cancellation
is due to eviction.  Add code to detect this.

Eventually, we will give this special handling, where it
marks certain mirrors inconsistent.  Tentatively, we'll
have it mark all mirrors except the primary one
inconsistent.

There's an issue here, though - what if there are other
errors from other AW locks?  It's not quite clear how
to handle that, so this is more by way of a note of
what's possible than a serious patch.

Note: It also removes the active writer lock cancellation
and ageing policies to make testing this easier.
(Ageing policy patch is just excluded from this history)

This is confirmed to work by looking at debug logs.
