--- Commit Message ---
LU-15425 utils: 'lfs migrate' can accept FIDs

Enhance 'lfs migrate' command to accept FIDs as argument.

If --lustre-dir=LUSTRE_MOUNT_POINT and --fid are specified at
the same time, then FIDs are specified on command line, or from stdin,
or by --files-from=LIST_FILE argument instead of file paths.

If there is only one mounted Lustre file system,
option --lustre-dir=LUSTRE_MOUNT_POINT is not necessary.
