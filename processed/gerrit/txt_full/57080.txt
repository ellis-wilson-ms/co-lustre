--- Commit Message ---
LU-18456 mdd: create recycle bin directory during MDT setup

The recycle bin is a feature in FS that acts as a virtual trash
can, allowing users to store deleted files temporarily before
permanently deleting them. It provides a way to restore or
retrieve deleted files if needed.

In this patch, MDT creates a "ROOT/RECYCLE" directory as a recycle
bin to store the "undeleted" files during the setup.
And when the recycle bin feature is enabled, the MDT moves the
deleting file into recycle bin just like the way to handle the
orphan file.
