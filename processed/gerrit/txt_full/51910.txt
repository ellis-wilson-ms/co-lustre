LU-10329 obdclass: tidy up linkEA code

Tidy up linkEA interfaces and related code:
* simplify linkEA interfaces, and add more checks.
* allocate linkEA buffer in advance, whose size is MAX_LINKEA_SIZE,
  then reading/writing linkEA doesn't need to handle -ERANGE error,
  and adding new entry will set overflow time if there is not enough
  space.
* tidy up MDT/MDD/OSD/LFSCK/LLITE linkEA code.
