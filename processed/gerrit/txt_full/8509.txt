This could be a little stronger if we added const in a few more places. In fact if we just had: 

 void (*hs_keycpy)(struct hlist_node *hnode, const void *key);

and

 static inline void cfs_hash_keycpy(cfs_hash_t *hs, struct hlist_node *hnode, const void *key);

Then the compiler would have rejected cfs_hash_keycpy(hs, new_key, hnode). Also the comment above the definition of hs_keycmp is backwards.
