--- Commit Message ---
LU-19326 ssk: automatic SSK key loading for client mounts

To ease SSK deployments, implement automatic SSK key loading for
Lustre client mounts. When mounting a client filesystem, the mount
process will automatically check for a .lgss file in the mount point
directory and load it as an SSK key if present.

This eliminates the need to specify "-o skpath=" for every client
mount when the SSK key has been pre-staged in the mount directory.

Also adds test_102 to sanity-sec.sh which validate the automatic key
loading functionality.
