LU-3399 ldlm: report last_committed value using export/obd

Both export and obd value is used for report last_committed value.
Patch takes into account obd_last_transno number when reporting last
commit transaction number to the client. This helps for client with
open/close activity to get real commit number and free requests from
replay queue.
