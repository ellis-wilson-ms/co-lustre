--- Commit Message ---
LU-1147 llite: Reduce CPU overhead on larger striped files

The current read implementation in 1.8 results in an excessive
number of calls to ll_readahead->obd_merge_lvb which will
then call lov_stripe_size for every stripe in a file.  As stripe
count increases this overhead begins to reduce read performance
on uncached files.
