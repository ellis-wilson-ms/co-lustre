LU-17716 utils: add "tunefs.lustre --replace" option

If a new target is not formatted with "mkfs.lustre --replace"
for some reason, then it would be useful to have a
"tunefs.lustre --replace" option to reset the OST config,
label, etc. so that it can be re-added to the filesystem
without trying to register with the MGS again.

This patch makes the "--replace" option work for both
mkfs.lustre and tunefs.lustre. With this option specified,
the LDD_F_VIRGIN (never registered) flag will be cleared,
then the newly-formatted OST will be able to use the same
index as the previously used target and skip MGS registration.
