It isn't really clear to me why it is useful to add a configure check that is checking for a #define that can be checked directly in the code?

If you prefer to have the #ifdef checks using HAVE_ZFS_PROJECT_QUOTA, then it would be possible to add a single check in osd_internal.h:

 #ifdef ZFS_PROJINHERIT          /* since ZFS 0.8.0 */
 #define HAVE_ZFS_PROJECT_QUOTA
 #endif

or similar?
