LU-2335 build: add script to generate client code

build/build-client-source.sh copies client only
code out and clean it up according to config.h.

build/upstream-kbuild provides makefiles for kbuild.

With this commit, one can generate client kernel source
code and build via:
1. ./configure
2. sh build/build-client-source.sh
3. cd lustre-client
4. make
