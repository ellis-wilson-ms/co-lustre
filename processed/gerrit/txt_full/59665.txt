LU-0000 llite: implement working DIO mirror iteration

This patch implements a complete mirror iteration system for
direct IO writes to immediate mirror files, successfully
writing to all mirrors sequentially.

Key achievements:
- Add mirror detection in ll_file_io_generic() to identify
  immediate mirror files and pass mirror information to cl_io
- Implement complete mirror iteration loop in cl_io_loop()
  that writes to each mirror sequentially
- Add proper position reset for both cl_io and kiocb when
  moving to next mirror
- Handle async IO completion codes (-529) correctly for
  mirror iteration logic
- Add comprehensive debug logging for mirror iteration flow

The implementation detects immediate mirror files using
ll_get_mirror_info() and stores mirror count and immediate
flag in cl_io structure. For DIO writes to immediate mirror
files, cl_io_loop() iterates through all mirrors, resetting
IO position and state between iterations.

Testing shows successful mirror iteration:
- All mirrors receive complete data (correct size)
- No system crashes or assertion failures
- Debug output confirms proper iteration flow
- Both mirrors marked as 'init' indicating successful writes

Next step: Implement data content synchronization to ensure
identical data across all mirrors.
