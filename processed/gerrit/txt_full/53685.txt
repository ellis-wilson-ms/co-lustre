LU-15798 build: fix gcc-11 -Wstringop-overread warnings

Gcc-11 detects a -Wstringop-overread warning on using 'strlen' here.
Because string access exceeds the 'struct changelog_rec cr' var size.
This is a false positive here, count the string length manually to
avoid using 'strlen'.
