--- Commit Message ---
LU-12058 tests: skip sanity 51d if stripe offset is unsuited

sanity.sh:test_51d needs stripe offset to be -1 so that round robin
algorithm was used to select an OST for a first stripe. If stripe
offset is not -1 the test fails.

Check that root and working directories both have stripe offset set to
-1 and skip the test if they are not.
