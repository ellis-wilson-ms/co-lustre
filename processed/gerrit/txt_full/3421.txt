--- Commit Message ---
LU-1199 build: Overhaul spec file and rpm build process

- Make it possible to override Requires and BuildRequires at
  rpmbuild time.
- Use rpmbuild --with/--without parameters.
- Group configuration settings for each distro into
  clear sections at the top of the spec file, rather than
  strewn throughout the file.
- Eliminate all configure_args parsing.  Instead explicitly
  pass arguments to rpmbuild as needed.
- Improve release string handling.  No more stray underscores
  in source rpm name!
- Don't include kernel string in version of source rpm.
- "make rpms" now employs "make src" to first build the
  source rpm.  This will help enforce disipline in keeping
  the source rpm rebuildable.
