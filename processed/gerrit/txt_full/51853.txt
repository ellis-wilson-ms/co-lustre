> We might be able to write a script to semi-automate the cleanup and generation of cleanup patches (one per file or so). That could help minimize some of the effort required. Arshad, do you have any tools you use for these clean ups?

I use vim-macros to get most of the job done. This is not as perfect as a script. However, it lessens the burden a lot. I would still sound manual, however, it is super easy. I did not start it this way, it was mostly manual, eventually wanted to better and ease the changes and have arrived what I am pasting below.

" pathogen
call pathogen#infect()
filetype plugin indent on
set ts=8
set nu
set ai
colorscheme elflord
set hlsearch
set laststatus=2
set lcs=trail:@,tab:>>
highlight SpecialKey ctermfg=red guifg=red
set list
set colorcolumn=80
nnoremap <silent> <Leader>b :TlistToggle<CR>
" nnoremap <silent> <Leader>c :CtrlP<CR>
nnoremap <silent> <Leader>S :set spell spelllang=en_us<CR>
nnoremap <silent> <Leader>s :set nospell<CR>
nnoremap <silent> <Leader>L :set colorcolumn=80<CR>
nnoremap <silent> <Leader>l :set colorcolumn=0<CR>

" move spaces to tabs
let @a='v<^[i>>>^[0j' 
" Take care of == NULL
let @z='xxxxxxxhx%li!
take care of != NULL
let @x='xxxxxxxhx'
" Handle bad space between assignment
let @s='elldwj0'
let @d='eelldwj0'
" Handle opening /*
let @w='eli^M^[0'
" Handle closing */ style:0
let @e='$bbli^M^[Bxx^[0j'
" Handle closing */ style:1
let @q='$bhi^M^[Bxx^[0j'
" Handle ENTRY style 1
let @r='0i^M^[jdd'     
" Handle ENTRY style 2
let @t='jdd'
