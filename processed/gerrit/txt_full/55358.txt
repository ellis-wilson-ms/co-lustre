LU-17852 gss: improve kernel key usage

Once kernel key has been instantiated, set proper flags so that
in-kernel processes can dispose of the key without restrictions in
case of error:
- KEY_FLAG_ROOT_CAN_CLEAR: set if key can be cleared by root without
  permission
- KEY_FLAG_ROOT_CAN_INVAL: set if key can be invalidated by root
  without permission
Also explictly mark the key as expired from userspace in case of
error.
