--- Message (Christopher J. Morrone, 2013-04-11 18:06:26.000000000) ---
Patch Set 1:

I like the idea of more comments, but I'd suggest changing them in two ways:

1) use a few more words
2) don't document the default behavior here

For instance, instead of:

# --enable-ldiskfs (default if server)

I would say:

# Files are made present by configure option --enable-ldiskfs

I advise against listing the defaults mainly because defaults can change, and there is no way the lustre folks will update these comments correctly in the future.  Looking up the default also is (or should be) fairly easy.
