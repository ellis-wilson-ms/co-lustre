--- Message (James Simmons, 2016-04-11 16:48:22.000000000) ---
Patch Set 2:

the module param change is 90 percent of the patch so breaking it out would still be a pretty big patch unless we do a dozen smaller patches. I thought about creating lots of little patches but instead went for the big change since module params are rarely changed so this lowers the chance of a collision. The rest of kp30.h I used for linux-misc.h since that is only used for backwards compatibility support for older distros. I'm working to make all the files in libcfs/linux/*.h to be handers for distros that will not be present in the upstream client. Also I removed kp30.h to be in sync with the upstream client.
