--- Commit Message ---
LU-6100 mdd: Not set xattr if object is DEAD

If the object is already being unlinked, do not set xattr,
otherwise the race might happen between the close and reint
operation (like set xattr), because close will unlink the
orphan, but not enqueue ldlm locks. Especially if the unlinked
object is striped directory.
