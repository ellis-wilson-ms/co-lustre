--- Commit Message ---
LU-5814 lov: add lov_object_xattr_get()

Add cl_object_xattr_get() to the CL object API. Implement
lov_object_xattr_get() to return LOV xattrs for LOV objects. Add
lov_lsm_pack() to generate LOV xattrs from a LSM.

In ll_getxattr() factor out the LOV xattr specific handling into a new
function ll_getxattr_lov() which calls cl_object_xattr_get(). In
ll_listxattr() call ll_getxattr_lov() to determine if a lustre.lov
xattr should be emitted.
