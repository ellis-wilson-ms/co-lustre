--- Commit Message ---
LU-10092 pcc: Use lease lock with intent to attach a file

Current PCC attach progress is not so elegant. It queries whether
the archive is finished by loop with a sleep, which is not good
for performance.
We improve the attach progress in this patch. It takes a lease
lock on the file being attched into PCC, and then copies the data
into PCC directly, final set the archived and released HSM state
flags for the file by executing the lease close with an intent.
