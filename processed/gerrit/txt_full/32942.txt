--- Commit Message ---
LU-11218 mdd: do not cache atributes on mdd_parent_fid

mdd_is_parent() brings link xattrs of not locked objects into osp
cache.
If that cache become stale a striped directory rename brings an
outdated link xattr into object xattr table. That makes a directory to
have more than one link.

Invalidate osp cache built during mdd_is_subdir().

Test illustrating the issue is added.
