LU-16035 kfilnd: Debugfs for transaction stats

While tools such as LNet selftest can be used to infer kfilnd
transaction duration, it does not provide any insight into where in
the transaction time is spent. For each kfilnd LNet network
interface, four sets of transaction stats are recorded: average time,
in nsecs, an initiator and target transaction is spent in each
transaction state with various LNet message lengths, and the average
time, in nsecs, an initiator and target transaction takes to
complete with various LNet message lengths. In addition, a reset file
is provide to reset all the stats.
