LU-16622 utils: 'lfs find --ost' supports index range

'lfs find --ost' command can accept an index range. For example
    lfs find --ost 0,1-3
It equals to:
    lfs find --ost 0,1,2,3
'lfs find --mdt' can accept index range too.
