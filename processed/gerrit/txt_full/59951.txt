LU-17429 tests: sanity-quota/48 file needs proj=10000

The sanity-quota test_48 file needs to explicitly set project ID
to 10000 as required by the test case, to avoid depending on a
previous test having used this QID.

Allow a delay for the QID to be added/removed from visibility.
Improve code style to allow test file to be created on any OST.
Replace direct /proc file access with "lctl get_param".
