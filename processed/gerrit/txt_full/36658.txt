LU-12931 lnet: fix usage of dl_msg_send in net_fault.

The 'dl_msg_send' in net_fault.c is described as a value
in 'jiffies', but sometimes used as a number of seconds.
It is declared as "unsigned long" but sometimes compared
against -1.

The value is (or should be) an absolute jiffies number:  it is
used to set a timer (mod_timer).  The "-1" special value
is used to indiciate that the timer hasn't been started.

So change all usage to assume jiffies, change the special-case
value to zero, and always test for zero after setting it.

Fixes: b11be372c21d "LU-9019 lnet: move ping and delay injection to time64_t"
