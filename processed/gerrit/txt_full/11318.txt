--- Commit Message ---
LU-5448 cfs: Fix compilation without libreadline

The two void declarations added by LU-5418
(http://review.whamcloud.com/11231):
static void noop_int_fn(int unused) { }
static void noop_void_fn(void) { }

Are not used if HAVE_LIBREADLINE is not defined, which
results in a compilation error.

Fix is easy, just wrap those declarations in
ifdef HAVE_LIBREADLINE.
