--- Commit Message ---
LU-12885 osd: rename xattr operation flags consistently

Use the existing enum lu_xattr_flags type where the Lustre xattr
operations flags are used, and name the variables similarly.

Use "int xattr_flags" for the kernel xattr operations flags.
These are defined in the kernel uapi so cannot be changed to enum.

Record use of XATTR_CREATE/XATTR_REPLACE in the RPC protocol.

Test-Parameters: trivial
