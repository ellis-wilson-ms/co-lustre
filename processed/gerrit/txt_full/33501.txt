Andreas, Thanks to you!, James and others for review. Without the reviews this would not be possible.

You did mention to check open files that i do before taking up the file. Eg file:<file> is:open branch:master

I have a rudimentary script to check the 'checkpatch' errors. Once this script report the error I go and manually inspect if change are required or not. This script does not catch everything, but it gives hint that this file needs to be inspected. 

I only run this script on a single folder. Presently i am targeting obdclass folder and all files within in. 

This is the script and it reports plenty of checkpatch warnings/errors. The script is ...

#!/bin/bash

for folder in fld ldlm lmv mdc mgc nodist ofd osd-zfs ptlrpc target lfsck lod mdd mgs obdclass osc osp quota fid llite lov mdt  obdecho osd-ldiskfs ost scripts  utils
do
	for file in $(ls lustre/$folder/*.c)
	do
		./contrib/scripts/checkpatch.pl -f --no-tree $file | grep tabs 2>&1 >/dev/null
		if [ $? -eq 0 ]; then
			echo "Found tabs erron on file [$file]" >> err.file
		fi
	done
done
