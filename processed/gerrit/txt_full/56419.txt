LU-18231 obd: change obd_replayable into OBDF_REPLAYABLE

Change the use of the obd_replayable bitfield into a bit in
the obd_flags bitmap and use atomic bit set/clear/test
operations on it.
No logical changes to the code and no changes to locking.

Add obd_replayable and previously-converted bitflags to
spelling.txt to help developers catch changes in old patches.
