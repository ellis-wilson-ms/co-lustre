LU-2456 lnet: DLC lustre config API Python/C SWIG binding

This is the seventh patch of a set of patches that enables DLC.

This patch adds Python/C SWIG binding to the Config API.
It allows the API to be called from python by first importing
the python module created from the library.  This can be used
by IML to call the configuration API.
