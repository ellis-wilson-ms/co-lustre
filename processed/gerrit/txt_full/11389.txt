--- Commit Message ---
LU-5470 lu: cleanup some lu_object functions

Convert lu_object_{exists,remote}() from macros to static inline
functions. Change the return type of
{dt,mdd,mdt}_object_{exists,remote}() from int to bool. Do the same
for several MDD object predicates.

Rename LU_OBJECT_HEARD_BANSHEE to the less cryptic LU_OBJECT_IS_DYING.
In lu_object_is_dying() take a pointer to struct lu_object rather than
to struct lu_object_header. Add lu_object_kill() to set
LU_OBJECT_IS_DYING and use it. Add dt_object_is_dying() and
dt_object_kill(). Remove lfsck_is_dead_obj() and use
dt_object_is_dying() instead.

Replace expressions of the form lu_object_exists(o) != 0 and
lu_object_exists(o) == 0 with lu_object_exists(o) and
!lu_object_exists(o).
