LU-0000 tests: add layout verification to EC tests

Add helper functions and verification code to EC tests in
llapi_layout_test to ensure that files created with EC
layouts have the correct layout parameters.

Three reusable helper functions were added:
- __verify_ec_comp(): Verifies EC parameters (PARITY flag,
  cstripe_count, dstripe_count)
- __verify_comp_extent(): Verifies component extents
- __verify_stripe_count(): For future stripe count
  verification

Tests 50, 51, and 52 now verify the created file layouts
by reading them back and checking component extents and EC
parameters.
