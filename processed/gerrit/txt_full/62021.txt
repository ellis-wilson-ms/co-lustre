LU-19019 tcu: use local topmost trash object for remote dir

When moving a remote directory into Trash Can, it should use the
topmost trash object located on the local MDT, not the one on the
remote MDT.
Otherwise, it will cuase the orphaned dir in Trash Can wrongly.

Add sanityn/test_117pb.
