--- Commit Message ---
LU-19179 misc: remove RHEL7 shrinker handling

To support RHEL7 kernels we needed special shrinker handling.
We can now remove support for the shrink function pointer.
With this removal we no longer need ll_shrinker_ops. The
function register_shrinker always returns an error code so
remove the special handling.
