LLAPI_LMV_GET_UUIDS(3)                    Lustre Library Functions                    LLAPI_LMV_GET_UUIDS(3)

NAME
       llapi_lmv_get_uuids, llapi_lov_get_uuids - Get device uuids

SYNOPSIS
       #include <lustre/lustreapi.h>

       int llapi_lmv_get_uuids(int fd, struct obd_uuid *uuidp, int *mdt_count);
       int llapi_lov_get_uuids(int fd, struct obd_uuid *uuidp, int *ost_count);

DESCRIPTION
       The functions llapi_lmv_get_uuids() llapi_lov_get_uuids(), gets the uuids for respective device type.

       fd file decriptor for mount point, uuidp is preallocated array of obd_uuid to hold uuids

       Count of devices.

RETURN VALUES
       Return the device uuids on success or a negative errno value on failure.

AVAILABILITY
       llapi_lmv_get_uuids()  llapi_lov_get_uuids()  is part of the lustre(7) user application interface li‐
       brary since before release 2.17.0

SEE ALSO
       llapi_file_fget_lov_uuid(3),         llapi_file_get_lmv_uuid(3),          llapi_file_get_lov_uuid(3),
       llapi_get_obd_count(3), llapi_search_ost(3), lustreapi(7)
