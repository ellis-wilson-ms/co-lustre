--- Commit Message ---
LU-13783 libcfs: provide and use sizeof_field().

Since v5.5-rc5~15^2 Commit 1f07dcc459d5 ("kernel.h: Remove unused
FIELD_SIZEOF()"), Linux no longer provides or uses FILED_SIZEOF().

Instead, sizeof_field() is used.  So provide that when it isn't
already provided, and use it in libcfs/crypto.
