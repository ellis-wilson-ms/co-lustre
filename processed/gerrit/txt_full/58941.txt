LU-0000 nodemap: nodemap as a mount option

applies only to local clients due to security reasons,
can be useful for testing purposes:

$ lctl list_nids
192.168.126.78@tcp
$ mount -t lustre ... /mnt/lustre
$ lctl get_param nodemap.default.exports
nodemap.default.exports=
[
 { nid: 0@lo, uuid: adb1d585-d975-48c6-87e3-faacf64fdedf },
 { nid: 0@lo, uuid: lustre-MDT0000-mdtlov_UUID },
 { nid: 0@lo, uuid: lustre-MDT0000-lwp-MDT0000_UUID },
 { nid: 0@lo, uuid: lustre-MDT0000-lwp-OST0000_UUID }
]
$ lctl nodemap_add testnm
$ mount -t lustre -o nodemap=testnm ... /mnt/lustre2
$ lctl get_param nodemap.testnm.exports
nodemap.testnm.exports=
[
 { nid: 0@lo, uuid: 20acba0b-972d-444a-af36-614de4cd298d },
 { nid: 0@lo, uuid: 20acba0b-972d-444a-af36-614de4cd298d }
