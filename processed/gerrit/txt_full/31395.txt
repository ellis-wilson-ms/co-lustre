--- Commit Message ---
LDEV-644 utils: always build liblustreapi.la

Commit 7b659a1e68174212c443761be1880774b440a901
("LU-5541 build: move libcfs and liblustreapi over to libtool")
has introduced a regression where liblustreapi.la will not be
built if configured with "--disable-utils" but also that some test
binaries linking to this lib are unconditionally built even when
"--disable-tests". This causes build to fail when both
"--disable-utils --disable-tests" are specified, like presently
during on-target configure step of lustre-dkms RPM.

This patch allows liblustreapi.la build even if "--disable-utils".
