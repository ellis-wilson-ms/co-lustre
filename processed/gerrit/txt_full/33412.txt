LU-11554 som: Make stat() work with LSOM

Currently, stat() does not work with LSOM yet, so using "ls -l" or
"mdtest" for testing could not exercise the LSOM.
This patch adds a lctl interface:
	lctl set_param llite.*.lsom=1
to make stat() behavior selectable to use LSOM data as returned
file's size and blocks for stat() system call.
