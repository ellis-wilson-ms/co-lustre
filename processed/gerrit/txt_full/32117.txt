LU-10938 wbc: Introduce LLIF_VIRTUAL flag for WBC

Directories with LLIF_VIRTUAL flag set create children without
talking to server. Set inode/file operations accordingly.
Override readdir to be populated from dcache too.

mkdir/open/create/rmdir/rm/stat/readdir operational.
File io does not work.
