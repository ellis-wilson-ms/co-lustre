LU-15909 lnet: Use lnet_send_data for NI selection

Make better use of the stack by leveraging lnet_send_data struct
for local NI selection.

lnet_get_best_ni() now assigns its return value to
lnet_send_data::sd_best_ni. lnet_get_best_ni() is renamed to
lnet_select_best_ni() to better reflect its usage.

Some variables in lnet_select_best_ni() are renamed to be more
consistent with other path selection functions.

The functions that previously called lnet_get_best_ni() all returned
an lnet_ni that was assigned to lnet_send_data::sd_best_ni. These
functions are now all adjusted to be void functions.
These functions have also been renamed to better reflect their usage:
  lnet_find_best_ni_on_spec_net -> lnet_select_best_ni_on_best_lpn
  lnet_find_best_ni_on_local_net -> lnet_select_best_ni_on_local_net

lnet_select_preferred_best_ni() also assigns an lnet_ni to
lnet_send_data::sd_best_ni. For consistency, this function has also
been renamed to lnet_select_preferred_best_ni().
