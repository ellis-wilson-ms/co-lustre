LU-14459 mdt: trigger dir split in create/open

After new file creation, if dirent count under parent directory
exceeds threshold, trigger directory split.

The dirent count is cached in struct mdt_object, and updated in
create/unlink. It may not be accurate, because it's not updated
if parent object is on remote MDT.

Updated sanity 230q.
