--- Commit Message ---
LU-6202 libcfs: fixed buffer overflow in IOC_LIBCFS_MARK_DEBUG

If this ioctl is called with data->ioc_inllen1 == 0, we
dereference ioc_inlbuf1, which hasn't been validated,
and index *before* the pointer, then create a debug message
with all the bytes *after* the pointer until the first nul.
This could create a debug message from an arbitrarily large
amount of garbage.
