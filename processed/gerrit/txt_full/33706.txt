LU-11688 tests: Enable ea_inode by default

It's been a few years, it should be safe to turn on
ea_inode by default in sanity, and it should be the
standard in deployed ldiskfs systems.

The requirement to set a 4 GiB journal size is not needed
and prevents it from running in VM type settings with a
small MDS.
