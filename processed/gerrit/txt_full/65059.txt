LU-20076 tests: speed up sanity/77f,103e,276

Speed up sanity test_77f by limiting the failed RPC resend count.
The test waits for an RPC error for each checksum type, but RPC
retry sleeps increasingly long for 10x retries before it gives up.

Speed up sanity test_103e by creating ACLs in batches, instead of
signly.  This adds ACLs at hundreds per second instead of minutes.

Speed up sanity test_276 by reducing the test duration unless
SLOW=yes is enabled.  This is a non-deterministic race test that
fails with a crash, and the aggregate test time across sessions
is sufficient to catch a regression in this code.

Clean up test script style issues in affected subtests.
