LU-16035 kfilnd: Handle transaction bad states

When a transaction is transitioned to an unknown state or a invalid
transaction event occurs, the transaction is in an unrecoverable
state. This should only occur if kfilnd has a bug. Thus, treat
transactions in an unrecoverable state as an LBUG().
