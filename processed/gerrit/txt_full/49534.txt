--- Commit Message ---
LU-16160 llite: handle filemap_fault() returns SIGBUS error

When ll_fault()->vvp_io_fault_start()->vvp_io_kernel_fault() try to
read a vmpage and memory pressure or truncate cuts in and invalidates
the page, so that filemap_fault() would return an VM_FAULT_SIGBUS
error.

When this happens, this patch try to find the page and if it's not
over the file size and its uptodate bit was cleared, it would
returns VM_FAULT_RETRY so that ll_fault() would try to read the
page again.
