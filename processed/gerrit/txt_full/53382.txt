--- Commit Message ---
LU-16915 osd-ldiskfs: use __vfs_getxattr in __osd_xattr_get

__osd_xattr_get() fails to get security attibutes via vfs_getxattr:
vfs_getxattr
  xattr_getsecurity
    security_inode_getsecurity
      cap_inode_getsecurity
        dentry = d_find_any_alias(inode)
          if (!dentry)
            return -EINVAL;

because ldiskfs’ inode has empty alias list.

Resort to __vfs_getxattr() in __osd_xattr_get().
