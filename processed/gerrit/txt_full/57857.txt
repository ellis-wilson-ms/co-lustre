--- Commit Message ---
LU-10026 csdc: fix compr header for recent kernels

On recent kernels, struct scomp_alg was modified to introduce
COMP_ALG_COMMON. So scompress.h header we copied from kernel sources
needs to be adjusted accordingly.
Avoid duplication of this header for all compression algs we build
internally, by moving it to libcfs.
