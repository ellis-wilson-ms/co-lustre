LU-1033 build: CentOS kernel 2.6.32-220 new config options.

New kernel options were introduced with the 2.6.32-220 kernel.
This patch sets these new options to their default values in
the config file.

This new config file means the Lustre can be built manually (without
lbuild) and does not require user intevention during the
'make oldconfig || make menuconfig' phase.
