LU-0000 utils: create lfs ec read and ec write commands

Add lfs ec read and lfs ec write commands for managing
EC (Erasure Coding) files. These commands provide
subcommands under 'lfs ec' for reading from and writing
to specific mirrors of EC files.

The commands are implemented as userspace-only for now,
using placeholder calls to the existing mirror read/write
functions. When EC-specific ioctls are implemented in
the kernel, these can be replaced with EC-specific
functions.

Commands added:
- lfs ec read: Read content from a specified mirror of
  an EC file, with options for mirror ID and output file
- lfs ec write: Write content to a specified mirror of
  an EC file, with options for mirror ID and input file

Both commands support stdin/stdout and file I/O, use
Direct I/O for performance, and include proper error
handling.
