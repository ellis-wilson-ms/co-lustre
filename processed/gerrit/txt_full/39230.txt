LU-13737 lnet: Have lnetctl display discovered routes

When discovery finds that a router is multi-rail it will consolidate
any routes that are equivalent. This hides useful information about
individual route status. Modify the lnetctl route show command so
that it prints information about all routes through MR gateways.

This change also moves the route state into the default output
rather than verbose.
