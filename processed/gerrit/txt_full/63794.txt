--- Commit Message ---
LU-0000 tests: EC recovery tests for different file sizes

Implement test_42a-42d in sanity-ec.sh to test EC recovery
with files that don't fill complete raid sets.

test_42a: Files smaller than one stripe (500K with 1M
stripe) - tests if recovery works when file doesn't span
all data OSTs.

test_42b: Files with partial stripes (1.5M = 1 full + 0.5
partial) - tests if recovery handles partial stripe
reconstruction.

test_42c: Files that don't complete a raid set (3M with
4+2 EC) - tests if recovery works with incomplete raid
sets (< 4MB).

test_42d: Files at PFL component boundaries - tests if
recovery works across component transitions in PFL
layouts.

These tests verify that EC recovery (LU-12669) works
correctly for edge cases where files are smaller than or
don't align with raid set boundaries.
