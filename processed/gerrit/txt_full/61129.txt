LU-19327 obdclass: fix obd_uuid2str()

- fix obd_uuid2str() check for UUID NULL-termination by
  using strnlen() instead of last byte check
- fix wrong sizeof(*uuid->uuid) which returns 1 to
  sizeof(uuid->uuid) which is real UUID buffer size
- obd_uuid_equals() to use strncmp() limited by UUID size
- obd_uuid2str() returns const char* so update some of it
  callers accordingly
