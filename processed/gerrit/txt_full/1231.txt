LU-591 MRP-139 1.8->2.x lustre migration tool

Convert on-disk data structures on MDS of Lustre FS 1.8 to Lustre FS
2.x ones in-place:
- assign true FIDs, stored in LMA EA;
- assign parent FIDs, stored in Link EA;
- assign children FIDs, stored in directory entries;
- create OI so that FIDs point to the proper inode numbers;
