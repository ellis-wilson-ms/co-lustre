--- Commit Message ---
LU-19428 nodemap: fix fileset flag modification on dyn nodemap

When the MGS and MDS are on the same node, it is not possible to set a
fileset on the dynamic nodemap to read-only/read-write because it
looked for a non-existing fileset header on the nodemap IAM.

This patch fixes this issue by recognizing dynamic nodemaps correctly
on nodemap_storage.

Fixes: 1e94980e4e ("LU-18357 ptlrpc: Add in-place modification of filesets")
