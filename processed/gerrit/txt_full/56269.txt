LU-18070 tests: fix sanity test_103a

On recent distributions like Ubuntu 24.04, the default behavior for
default ACLs is different than on other distros.
So to make sure we do not inherit unwanted default ACLs, make sure
they are cleaned up before adding any other ACLs.
