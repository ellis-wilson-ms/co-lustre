--- Commit Message ---
LU-19980 build: improve lock_manager_operations tests

Configure tests that use "struct lock_manager_operations" will
fail after:

Linux commit v6.2-rc3-9-g5970e15dbcfe
  filelock: move file locking definitions to separate header file

Run the configure test LC_HAVE_LOCKS_LOCK_FILE_WAIT_IN_FILELOCK
early so the resulting HAVE_LINUX_FILELOCK_HEADER is available
for "struct lock_manager_operations" tests to conditionally

  #include <linux/filelock.h>
