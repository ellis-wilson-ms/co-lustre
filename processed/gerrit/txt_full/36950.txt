LU-13037 nrs: dump stats of TBF clients

A new entries is added for each service. The TBF information of
all client classifications will be dumped from each entry when
reading the entry. Following is an example of the dumped
information:

- key:             _10.0.1.253@tcp_10_0_0
  cpt:             0
  queue_type:      reg
  refs:            0
  rule:            default
  rpc_rate:        10000
  ntoken:          2
  token_depth:     3
- key:             dd.0_10.0.1.253@tcp_10_0_0
  cpt:             0
  queue_type:      reg
  refs:            0
  rule:            default
  rpc_rate:        10000
  ntoken:          2
  token_depth:     3
