--- Commit Message ---
LU-4017 pools: generate pool ID for OST pools

This patch try to add a unique pool id for every pool, pool id
is unique identifier for every pool.

Pool id is added for project quota, we use ost pool to manage
inode project quota, project quota design need an id for every
inode, and quota usages are grouped by this id. after OST pool
id is added, we can enforce project quota by using existed ost
pool attributes to file.

Pool ID is saved in MGS llog too, for old system which contains
some OST pools saved in MGS llog, it will return -EINVAL.
Administrators need destroy old ost pool and regenerate them
with new format.

Aslo add quota aware support when creating
and destroying OST pool.
