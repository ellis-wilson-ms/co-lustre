--- Message (Andreas Dilger, 2012-10-28 17:58:26.000000000) ---
Patch Set 1: I would prefer that you didn't submit this

I agree with Alex that the testing already takes too long. The filesystem was just reformatted at the end of the previous test (which is itself probably unnecessary), and there is no reason to believe that last_rcvd is not present on the MDT.

Also, while the test verifies that the MDT can be mounted, it does not actually check that the last_rcvd file was recreated.  It should actually mount the filesystem again and check that the last_rcvd file exists.

I would hope that this test could also be made to work for ZFS, just by using $(facet_fstype $SINGLEMDS) instead of hard coding ldiskfs in the mount line.
