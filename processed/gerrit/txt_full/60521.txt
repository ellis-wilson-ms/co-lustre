LU-19233 build: add system-sb-certs as a buildreq

The Jenkins build has failed with el8.10(inkernel) because the
"system-sb-certs" package dependency specification is missing.

So this patch adds the "system-sb-certs" package to the RPM
spec file as a BuildRequires dependency.
