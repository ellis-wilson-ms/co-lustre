LU-12668 ec: raidset improvements

This patch adds the following improvements to the liblustreapi_layout:
- Break out computation for a raidset into a separate function instead
  of duplicating the same code for both k0 and k1.
- If a raidset is fully covered by a hole we can skip
  computing/verifying the parities for it.
