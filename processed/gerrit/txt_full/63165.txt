LU-19598 utils: add a new API llapi_find_with_cb()

This patch adds a new API llapi_find_with_cb() to extend llapi_find()
with custom callbacks cb_init() and cb_fini() in type llapi_find_cb_t
to enable specialized traversal beyond standard find.
It also adds a new field void *fp_cb_data to struct find_param to
pass callback data.
