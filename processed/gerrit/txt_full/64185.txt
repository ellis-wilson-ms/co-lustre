LU-12668 ec: additional ec tests

This patch adds an additional 22 tests which cover:

EC layout operations (migration, split, truncate,
fallocate), error rejection for invalid EC component
changes, lfs find with parity flags, multi-stripe data
integrity, PFL lazy parity instantiation, and failure
injection scenarios (OST stop/restart recovery, write
failures during parity resync, exceeding parity
tolerance).
