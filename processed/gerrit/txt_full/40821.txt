--- Commit Message ---
LU-8837 build: make OpenSFS code buildable against Lustre linux client

If you try building the OpenSFS tree against a kernel with a
native Linux Lustre client you will have build conflicts due to
both the kernel and the OpenSFS tree having the same internal
Lustre kernel headers. This patch resolves most of those build
issues by making a cleaner separation of the server and client
in the internal headers.

We also fix a debug message in mdc_changelog.c that prints out
a FID that is available for client builds.
