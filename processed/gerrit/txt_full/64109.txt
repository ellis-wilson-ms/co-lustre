LU-19749 ldiskfs: remove ubuntu18 patch series

The Ubuntu18.04 and earlier kernels are no longer supported.

Remove the ldiskfs-4.15.0-*-ubuntu18.series file and all patches
that are no longer used by this (or other) patch series. The actual
patches under ldiskfs/kernel_patches/patches/rhel7.x may still be
referenced by newer kernel series, so cannot be removed wholesale.

The ./contrib/scripts/clearpatches.sh ldiskfs/kernel_patches command
will report which patches are no longer used by any series file.
