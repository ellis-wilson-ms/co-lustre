LU-6142 build: update checkpatch to v7.0-rc5 kernel

Update checkpatch.pl to upstream v7.0-rc5-51-gbbeb83d318 kernel.

This avoid unnecessary 'strcpy()' warnings and adds is_userspace()
that can be used to select userspace spelling rules.  Add files in
lustre/{tests,utils}, lnet/utils, contrib/ to userspace list.

Rename 'userspace_spelling.txt' to 'spelling-userspace.txt' to better
match the existing 'spelling.txt' and 'spelling-man.txt' filenames.

Remove stale contrib/scripts/lustre-checkpatch.patch file. It can be
regenerated by comparing against the specified kernel checkpatch.pl.

Reduce some of the delta from the upstream kernel checkpatch:
- add $gerrit_ok option instead of deleting Change-id: check block
- reduce code duplication when handling user vs. kernel spelling
- move explicit LASSERT, GPLv3, CDEBUG_LIMIT checks into spelling.txt
