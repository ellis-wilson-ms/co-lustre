--- Commit Message ---
LU-20037 mdc: allow error on write to mdt on CL_FSYNC_DISCARD

mdc_io_fsync_start() should not return error running for
CL_FSYNC_DISCARD so that cl_io_start() did not break its loop over
slices. Otherwise, ll_truncate_inode_pages_final() finds undiscarded
pages and fails with: (osc_page.c:182:osc_page_delete()) Trying to
teardown failed: -16.

Test to illustrate the issue is added.

Fixes: 050c2fb23b ("LU-16043 osc: allow error for write on CL_FSYNC_DISCARD")
