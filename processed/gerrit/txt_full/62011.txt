LU-14554 lfs: Add "--stale" option to lfs mirror verify

This patch adds "--stale" option to lfs mirror verify command to
check the mirror of a file and mark them stale based on intelligent
heuristics

- For 2 mirrors: mark non-prefwr mirror as stale
- For multiple mirrors: mark mirrors that don't match the majority
- Use prefwr/prefrd as tie-breakers
