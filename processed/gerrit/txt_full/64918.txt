LU-19712 lfs: Add 'lfs pool pin/unpin/pin_info' commands

Introduce lfs pool commands to manage pool pinning via lustre.pin
xattr.  Three new commands added: "lfs pool pin" to pin
files/directories or FIDs to an OST pool, "lfs pool unpin" to unpin
files/directories or FIDs from an OST pool, and "lfs pool pin_info" to
report whether files/directories or FIDs are pinned to an OST pool.
All the new commands get pool name as argument. If pool name is not
provided, it is inferred from the primary mirror layout of each file
or directory. They also support operating on FIDs via --fid option.
