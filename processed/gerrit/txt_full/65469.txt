--- Commit Message ---
LU-20177 mdt: Directory migration fails with not-empty error

Migration used to fail during the shrink phase because
it found a directory with objects still on the migrating
MDT. This happened because the namespace for some objects
was not updated, and -EALREADY was returned for those
objects. A test test_230B has been added to verify
the fix.
