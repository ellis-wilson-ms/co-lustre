--- Commit Message ---
LU-9184 lov: component layout entry can be not just RAID0

Current implementation of component layout works entries as
with RAID0 type always, meanwhile component entry may be
not RAID0, e.g. Data-on-MDT type or any other.
Patch modifies component layout to work with any entry type
by defining per-entry set of operations by its type.
