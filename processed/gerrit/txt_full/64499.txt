--- Commit Message ---
LU-19963 nodemap: add projid_set rbac role

The purpose of the new projid_set rbac role is to control setting or
clearing of project ID on files and directories. When not set, clients
cannot change project ID ('lfs project -p/-C') but can still see the
project ID value.

sanity-sec test_64j is introduced to exercise the new projid_set rbac
role.
