LU-12668 tests: add EC failure orchestrator

--WIP--

This is an early version of an automated stresstest / failure
injection / benchmarking tool for EC. It can currently only be run
by the user, but will be adapted to allow integration to sanity-ec.

At its current state the user can disabled specific OSTs, see
which OSTs contain the parity information, run a basic read on a
created file and see the results. 8 OSTs are required to run for
now.

Added different layout options and added action logging for
bug recreation.

Going to rebase ontop 64938 to see how things go.
