LU-11371 lutf: set socklnd modparams using DLC

This test script tests setting the socklnd
modparam - peer_credits using DLC.

This script configures lnet tcp with a
non-default peer_credits value using DLC.
It then checks if the value is set by
running "lnetctl net show -v" and cleans
up the configuration before returning.
