LU-17322 yalt: Yet Another Lustre Tester

New testing script written in Python designed to
enable new features to the Lustre test framework
without breaking compatibility with current testing
tools.

This patch contains a first revision of this tool.
It intends to replicate the functionality of the
current Lustre testing tools, while setting up
some infrastructure for later patches.

This tool allows for custom test providers to
be defined. The job of a provider is to setup
the environment and infrastructure needed
to run Lustre sanity tests. The providers are
defined a separate scripts with a common API.
This mean that they can be written in any
language and make use of any IaC tools.
