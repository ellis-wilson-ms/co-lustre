LU-19969 utils: fix some issues in ltrash_purge

This patch fixes the following issues in ltrash_purge:
 - improve the main loop process and trigger/exit condition
 - improve space-based policy check
 - in daemon mode, if target usage becomes below watermark,
   ltrash_purge should stop the current scan, then go to sleep and
   wait for the next scan, but not exit
 - check target usage after purging
 - fix the trash file check and print more debug information in
   sanityn.sh/test_119d
 - correct stats calculation and dump stats for SIGINT/SIGTERM too
 - output fuller configuration information
 - do some code cleanup
