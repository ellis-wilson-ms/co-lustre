(1 comment)

Kit, you are right - the code under the top-level "if" condition is identical except for the one error message. It would be better to put it in a small helper function, maybe mdt_pack_acl2body(info, repbody, o) to match mdt_pack_attr2body()?  Please avoid the deep nested "if" blocks and just have it return early when possible. The function should also be under CONFIG_FS_POSIX_ACL is this code is so it doesn't generate a warning if that is not set. 

That would reduce code duplication and would avoid similar problems in the future.
