--- Commit Message ---
LU-18479 nodemap: ncf_obj remains NULL in error path

The nodemap_save_all_caches() function leaves ncf_obj as NULL
if nodemap_save_config_cache() fails. This can lead to a system
panic when ncf_obj is accessed later without proper NULL checks.
