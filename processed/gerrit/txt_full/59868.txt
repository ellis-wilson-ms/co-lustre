--- Commit Message ---
LU-19095 mgc: notify MGS about LNet changes

Patch adds functionality to MGC to send notifications
to MGS about network changes on target, i.e. network/NIDs
mangement dynamically. So adding new network on online
target will cause changes in MGS nidtable for that target
and IR update for clients.

If network is deleted then network identifier is sent
as single NID in list identified by special token
