--- Commit Message ---
LU-6602 llog: increase llog chunk size

Increase update llog chunk size from 8192 to 32768, so
one cross-MDT operation can includes more update records,
and it can create striped directory with more stripe
count.

Adding a LLOG_F_BIG_CHUNK flag in the llog header flag,
if the llog file has such flag, then it means it will
use the bigger chunk size (32KB).

And also remove a few padding in object_udpate_param,
so we can pack more updates and parameters in one
llog record (the maximum size is 32KB).
