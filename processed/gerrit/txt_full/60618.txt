--- Commit Message ---
LU-19131 utils: mlock lfs DIO buffer pages

Use mlock to force memory buffer pages to be locked in memory,
avoiding them being swapped out during memory pressure, which can
result in the thread becoming stuck trying to access the swapped page.
This is important when the "lfs mirror extend" or "lfs migrate"
commands are used to copy files with DIO, when the file size is
larger than RAM using cached IO.

Fix some places where error handling did not free allocated pages.
