--- Commit Message ---
LU-18857 mgs: add timeout mechanism for allow_register

Add automatic timeout functionality to the allow_register
parameter to improve security by limiting the window during
which new filesystems or targets can register.

allow_register=0: Registration disabled
allow_register=unlimited: Registration enabled indefinitely
allow_register=N (N > 0): Registration enabled

When set via 'lctl set_param allow_register=<value>, a timer
is started that automaticaly disables registration when
expired.

Supported time suffixes:
 s = seconds
 m = minutes
 h = hours (default)

Add test_157c to verify the functionality
