LU-18222 quota: project Pool Quotas POC

Proof of concept how project PQ could be
implemented. The main idea is to handle project
ID as an OST id to find whether this project
belongs to some project pool. As for project PQ
OST index is the same as project ID I had to change
some things in lov and lod to be able to non presented
in a system OSTs. This is not going to be commited and
is required just to show the idea. To understand that
it is a project PQ I used prefix "prjPQ_". Later if
we add a separate API to create project PQs without
ost pools, we could avoid using the prefix. If we
project PQ would be implemented, we need some faster
way to search project IDs in project pools instead of
tgt_tgt_check_index.

It has a memleak and a lot of printk I used for debugging.
