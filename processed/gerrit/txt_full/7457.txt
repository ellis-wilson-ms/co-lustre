LU-3467 mdt: clean up structure packing/alignment

Clean up the packing and alignment of struct mdt_thread_info,
along with related structures, shrinking it by 72 bytes.

Replace typedef mdl_type_t with enum md_handle_type.
Add enum md_lock_mode to existing typedef mdl_mode_t.

Optimize mdt_thread_info_init() to use memset() instead of
explicit field-by-field zero initialization.

Fix misc coding style issues.
