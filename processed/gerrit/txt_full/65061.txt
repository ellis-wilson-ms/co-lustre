LU-19921 ptlrpc: add GSSIAM upcall operations and helper

This patch adds the upcall operations and tunable attributes
for the GSSIAM security flavor, along with the userspace helper
script (l_gssiam_upcall) which retrieves the user's enterprise
GSSIAM token and passes it back to the kernel via a sysfs downcall.
