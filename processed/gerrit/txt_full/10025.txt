LU-4869 mdt: Do not retrieve dir layout for unlinked dir

1. Do not retrieve MD layout if directory has been
unlinked, then the client will be notified the directory
is unlinked when it revalidate the inode, so to avoid
other client, who still cache the layout, will create
files/directories under slave stripes.

2. Add a few tests to check the process.
