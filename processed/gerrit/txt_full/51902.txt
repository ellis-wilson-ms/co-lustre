LU-17022 obdclass: convert still more flags to bitops

	obd_stopping:1,		/* started cleanup */
	obd_starting:1,		/* started setup */
	obd_force:1,		/* cleanup with > 0 obd refcount */

are now

	OBDF_STOPPING,		/* started cleanup */
	OBDF_STARTING,		/* started setup */
	OBDF_FORCE,		/* cleanup with > 0 obd refcount */

Move some simple initialisation from class_setup to class_newdev to
allow a spinlocked region to be completely unlocked.
