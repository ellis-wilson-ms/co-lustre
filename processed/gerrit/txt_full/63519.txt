LU-12187 tests: clean up sanity-lfsck/20[ab] style

Clean up the code style of sanity-lfsck test_20a and test_20b.
- use ((...)) for numeric comparisons
- use [[...]] for string comparisons
- use bash built-in for-loop instead of external "seq"
- use "stat -c %s" to get file size instead of grep output
- use "grep -c" instead of also calling "wc -l"
- precompute expected value for consistency with error message
- quiet several shellcheck warnings
