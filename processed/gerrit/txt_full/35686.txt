--- Commit Message ---
LU-12628 lnet: do not assert on resend of msg_rx_committed

We should only resend messages that have msg_tx_committed, but we
needn't assert on this condition in lnet_health_check. If
msg_tx_committed is not set we can simply return -1 to finalize the
message.
