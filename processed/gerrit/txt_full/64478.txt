LU-19929 build: Disable strict errors for RISC-V initially

Disable strict compiler error flags for RISC-V during initial
porting phase to allow identification of architecture-specific
issues without blocking compilation.

Once the RISC-V port is stable and tested, this can be reverted
to enable strict error checking like other architectures.
