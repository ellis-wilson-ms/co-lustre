LU-930 doc: align lfs and lctl man page structure

Bring the lfs.1 and lctl.8 man page structure into alignment.
List the general usage description first, then common commands,
and top-level command-line options, before the command list.

Add a bash-completion section so that users can learn about
this useful functionality.
