--- Commit Message ---
LU-16897 llite: option to enable/disable SFO

By default Sparse File Optimization is enabled on client side.
"sfo"/"nosfo" options have been added to control that when mounting
Lustre FS.

A test was added to check there is no issue when a client does not
have SFO enabled (or by analogy does not have the required patch for
that).
