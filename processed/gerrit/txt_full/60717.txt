--- Commit Message ---
LU-19066 lod: take failure domain into account in lod_ost_alloc_qos

In multi-component qos stripe allocations, when we flag an OST to be avoided
because it has already been allocated from, if this OST has failure
domain configured then also flag for avoidance all other OSTs that are
in the same failure domain.

When collecting the set of OSTs that are potential candidates for
allocation, if and OST has failure domain configured then only
consider it a candidate for allocation iff it has not been flagged
for avoidance.
