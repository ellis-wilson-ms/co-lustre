--- Commit Message ---
LU-18357 ptlrpc: Support clearing all filesets from a nodemap

Deleting filesets requires the name of the fileset to delete. While
all filesets can be removed at once, this is only done implicitly when
deleting a nodemap.

To improve usability, this patch adds the "--all" argument for the
existing "lctl nodemap_fileset_del" command to delete all filesets
from a nodemap.

Sanity-sec 27c was modified to test this functionality.
