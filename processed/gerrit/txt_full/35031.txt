--- Commit Message ---
LU-12325 dom: use LCK_PR with 'trylock' mode

DOM lock at open has three possible values - never, always
and trylock. 'Always' mode takes DOM bit depending on open
mode - PW or PR, but 'trylock' mode could use less strict
PR mode always to get DOM bit in more cases.
