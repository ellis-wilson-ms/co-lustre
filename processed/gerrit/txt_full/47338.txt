LU-13012 pcc: abstract rule parsing and matching func

In this patch, we abstract the rule parsing and matching functions
in PCC and move the abstract common functions into the src file
"lustre/obdclass/obd_rule.c".
By this way, WBC can also use the shared abstract rule functions
for rule based auto caching.
