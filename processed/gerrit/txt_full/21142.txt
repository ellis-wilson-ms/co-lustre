(2 comments)

Please see my comments in LU-2377 about why this patch is just hiding problems rather than fixing them.

If you sun it this to the upstream ext4 maintainers and it is accepted then it will appear in ldiskfs eventually also. However, I think it would be better to actually fix the mballoc allocator to prefer empty groups near the start of the disk instead of strict round-robin as it is now, though that might also only make sense if the disk is rotational. 

Other minor issue is that this needs to use sys instead of proc.
