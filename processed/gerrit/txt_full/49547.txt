--- Commit Message ---
LU-16429 ahead: basic framework for ahead operations

This patch implements the basic framework for ahead operations
including:
- create-ahead
- open-ahead
- stat-ahead
- read-ahead for DoM-only files

It provides two RPC modes:
- One RPC pre operation
- Batch RPC

This patch implemented create-ahead via one RPC per operation RPC
mode.
