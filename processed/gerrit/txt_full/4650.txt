LU-2145 server: move ptlrpc services to the unified server

Create service threads as part of unified server. Now threads are
started always upon server start and all targets (i.e. MDT/MGS/OSS)
set just set of handlers for particular RPCs and register with
unified server.

This change allows more than one MDT to work on the same node and
provide the next step - unified request handler.
