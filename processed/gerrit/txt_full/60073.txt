--- Commit Message ---
LU-10026 osc: improve can_merge_pages()

let can_merge_pages() ignore OBD_BRW_ASYNC and OBD_BRW_SOFT_SYNC
flags which can be set on any page in RPC with the same expected
result.
this way, OSC can better find full RPCs even when different
hits are set on pages.
