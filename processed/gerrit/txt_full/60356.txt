LU-19209 utils: ofd_access_log_reader usage mention log_size

ofd access logs are not enabled by default; they are enabled by
setting the size of the circular buffer they require.

State this in the usage message of the utility used to read them, and
in the comment of the source file implementing them, so users need not
spelunk through code to learn how.
