--- Commit Message ---
LU-18857 mgs: add allow_register tunable parameter

Introduce a new tunable parameter 'allow_register' that
enables administrators to control whether new Lustre
filesystems can be registered with the MGS. This provides
a mechanism to temporarily prevent new filesystem
registration while maintaining existing filesystem
operations.

Current Implementation accepts:
 0 : Disable new filesystem registration
 1 : Enable new filesystem registration
