--- Commit Message ---
LU-19929 llite: include generic prefetch()

On some kernel the arch specific prefetch() is not available.
Including the `linux/prefetch.h` as fallback.

Impact:
- Fixes: 'implicit declaration of function prefetchw'
  compilation error on  some RISC-V kernel.
- No functional impact (prefetchw is a performance hint)
- Maintains optimization where available
  while enabling RISC-V builds with fallback where needed.

Tested on:
- Ubuntu 24.04 RISC-V with Linux kernel 6.8.0-35-generic
- Fedora 42 RISC-V with Linux kernel 6.16.4-200

Test-Parameters: trivial
