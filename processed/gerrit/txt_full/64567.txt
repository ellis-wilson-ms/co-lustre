--- Commit Message ---
LU-10026 utils: print all lmm_pattern flags

If an unknown pattern mask is present in lmm_pattern, the known
patterns should be printed, rather than skipping all of them.
The llapi_lov_pattern_string() function will already handle
printing the unknown bits as hex, so that doesn't need to be
handled (poorly) in lov_dump_user_lmm_header().
