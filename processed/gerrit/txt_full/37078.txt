LU-13095 mdc: cancel locks in ELC list when errors occurs

In MDC layer, when packing SELinux policy info failed, it forgot
to cancel the locks in the ELC list.
This bug was introduced in:
commit 0a773f04b28860c3748f9f1460818b8461c96ad1
https://review.whamcloud.com/24424

This patch solves this bug by calling ldlm_lock_list_put() to
release the locks in the cancel list when failed to packing
SELinux policy.
