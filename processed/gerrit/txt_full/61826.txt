LU-0000 tests: EC lease operations test framework

Add comprehensive C-based test framework for testing lease
operations on Erasure Coded files. The test validates basic
lease operations (read/write leases) and resync lease
operations (LL_LEASE_RESYNC, LL_LEASE_RESYNC_DONE) on EC
files.

Test Structure:
- test_ec_basic_lease: Basic read/write lease operations
  on simple EC file
- test_ec_mirror_basic_lease: Basic leases on EC file
  with mirror (currently skipped - not implemented)
- test_ec_resync_lease: Resync lease operations with
  RESYNC and RESYNC_DONE flags
- test_ec_mirror_resync_lease: Resync on EC mirror file
  (currently skipped - not implemented)

The test framework uses proper output handling: success
messages go to stdout, errors and skips go to stderr.

Test Results:
- EC file creation via llapi works correctly
- Read/write leases work on EC files
- Resync leases (LL_LEASE_RESYNC/RESYNC_DONE) work
- FLR state transitions correctly (RDONLY -> SYNC_PENDING)
- EC mirror files not yet supported (gracefully skipped)

The test is integrated into sanity-ec.sh as test 6a and
provides a framework for Phase 3 userspace parity
regeneration implementation.

Test-Parameters: trivial
