--- Commit Message ---
LU-9747 tests: compare Lustre/ZFS to local ext4 POSIX

The Lustre test script that checks POSIX compliance runs
testing twice; once on a local file system and then with
Lustre.  These results are compared to check Lustre’s POSIX
compliance.

Test if local ext4 POSIX results are the same as Lustre/ZFS.
If ext4 and Lustre/ZFS are not similarly POSIX compliant,
then we need to try to install ZFS on the client and run the
first pass of POSIX testing on the local ZFS file system.

Test-Parameters: trivial fstype=zfs testlist=posix
Test-Parameters: fstype=zfs mdscount=2 mdtcount=4 testlist=posix
