LU-19069 utils: automatically generate SSK client key

On client side, when mounting with the skpath option, automatically
generate the prime to be used by SSK if the provided key does not have
it yet.
Also allow 'lgss_sk -l' to generate the missing prime in the SSK if
the '-t client' option is provided.

This patch also adds test_100 to sanity-sec.sh which validates the
automatic prime generation functionality.

Test-Parameters: testlist=sanity-sec env=ONLY=100,ONLY_REPEAT=10,SHARED_KEY=true
