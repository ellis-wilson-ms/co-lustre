--- Commit Message ---
LU-00000 lustre: split non-Linux functions from linux-wait.h

linux-wait.h in libcfs contains some wait macros that duplicate
functionality in (recent) Linux, and also some macros that add new
functionality.

Split the new macros out into a new lustre_wait.h (they are only using
in lustre, not lnet).  This makes it easier to strip compatability
code when transforming lustre for upstream submission.
