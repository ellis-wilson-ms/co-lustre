--- Commit Message ---
LU-11819 tests: combine similar functionality in lib

There are several Lustre test scripts that run tests from an
external program. These executables are themselves composed of
several test routines. We should pull out all the common routines,
structures, etc. from each of these test (.c) programs and create
a library that they can link to.

This work will cut down on code duplication.

To start this effort, pull out common functionality and structures
from llapi_layout_tests.c and mmap_sanity.c and put them into
libtestutils.c .
