LU-12668 tests: add EC layout pattern tests

Add tests 54a-57d exercising EC recovery across
different layout patterns:

- test_54a-c: page cache and concurrency -- verify
  recovery with warm cache and concurrent readers
- test_55a-d: file size boundaries -- 1-byte, partial
  stripe, exact stripe, and multi-stripe sizes
- test_56a-d: PFL component interaction -- recovery
  at component boundaries and transitions
- test_57a-d: stripe set geometry -- different stripe
  counts and sizes (2+1, 4+2, varied stripe_size)
