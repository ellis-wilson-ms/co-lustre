--- Commit Message ---
LU-110 Old-style quota type handling fix

Version 1.8 uses a tunable named "mdt.quota_type" in order to store
quota parameters in config, while version 2.x has a tunable named
"mdd.quota_type" for the same purpose. This introduces incompatibility
when trying to run a 2.x Lustre instance on top of a filesystem with a
1.8 disk format.
