--- Commit Message ---
LU-11377 lod: set [x]attr to slave stripes if needed

When getting ATTR/XATTR from some directory object, Lustre client
will only retrieve the ATTR/XATTR from the master stripe, so it's
not needed to set the ATTR/XATTR to other stripes, which improves
the DNE performance.
