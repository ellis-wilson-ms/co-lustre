LU-6098 tests: skip client mount in test-framework

Skip the client mount in test-framework.sh:setupall() if
environnment variable SERVERONLY is set.  Complements existing
behavior if CLIENTONLY variable is set.

This makes examining the server startup easier, as debug logs
etc. are not cluttered with client mount activity.
