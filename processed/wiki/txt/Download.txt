Official production releases and pre-release versions of Lustre software are available for download. Official releases offer new features and enhancements, and have undergone thorough test cycles. They are available at the Lustre download (http://lustre.org/download) site. Pre-release versions of Lustre are still being coded or are undergoing release testing. They are available for checkout from the Lustre® source repository. 

As an open-source product, we encourage contributions to develop and test Lustre® by trying out pre-release versions of the software. To obtain Lustre® code from the source repository, you must have the Git version control system installed.
