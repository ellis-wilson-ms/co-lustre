Access resources such as information about current and upcoming Lustre features, a roadmap, publications about Lustre and training opportunities. 

Interoperability, Features and Roadmap

These resources detail Lustre releases and plans. 

-  <ins>Lustre Release Information</ins> lists current Lustre releases and supported kernels and networks for these releases. 

-  <ins>Lustre 1.8</ins> describes features and benefits offered by upgrading to Lustre 1.8. 

-  <ins>Lustre 2.0</ins> describes features being developed for the next-generation release of Lustre. 

Publications

Use these resources to learn about Lustre.

-  <ins>Lustre Presentations</ins> by Lustre engineers at Lustre User Group meetings, conferences, Lustre all-hands meetings, and other Lustre-related events. 

-  <ins>Lustre Publications</ins> from a variety of sources, including links to videos and podcasts, white papers and blueprints.

-  <ins>Lustre Operations Manual (http://wiki.lustre.org/manual/LustreManual20_HTML/index.html)</ins>, the primary source for Lustre user documentation.

Lustre Projects and Future Features

To learn about active Lustre projects that span releases or features that are not in Lustre 1.8 or Lustre 2.0, see <ins>Lustre Projects</ins>. 

Lustre Internals

-  See the <ins>Lustre Internals Documentation (http://wiki.lustre.org/lid/subsystem-map/subsystem-map.html)</ins> for detailed descriptions of the Lustre codebase in an easily accessible format.

-  The LID includes <ins>Doxygen API information (http://wiki.lustre.org/lid/doxygen.api/modules.html)</ins> for various Lustre modules.
