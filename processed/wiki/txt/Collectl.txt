Introduction
Collectl is a tool that facilitates the monitoring of system resources. Peter Piela has created custom Lustre data collection plugins for Collectl.

-  Collectl Project Site (http://collectl.sourceforge.net/)
-  Collectl Documentation (http://collectl.sourceforge.net/Documentation.html)
-  Collectl Github Mirror (https://github.com/sharkcz/collectl)
-  Collectl Lustre Data Collection Plugins Site (https://github.com/pcpiela/collectl-lustre)
