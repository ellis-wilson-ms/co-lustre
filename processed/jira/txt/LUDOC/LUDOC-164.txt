--- Description ---
Suggestion to standardize as much as possible.

For example, seen in the literature are LNET and LNet - we should ensure we're consistently using the one we want to use.  My vote, FWIW, is for LNet.

A more difficult one is file system (filesystem).  Although often used interchangably, there may be a distinction between them - meaning, a time to use one and a time to use another.  For example, I try to follow:

Use file system for a high level description.

Use filesystem when talking about the specifics of a particular file system.

Open suggestion for any other consistencies as well.
