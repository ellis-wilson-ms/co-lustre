--- Description ---
The original outline commit in response to LUDOC-266 lists the project goals as they appear in the contract. While that is comprehensive, it is not necessarily an optimum outline for the presentation of the material. New content that actually addresses project goals will be added in response to this ticket, and it will be good to have that new content fit will within the outline. 

The current outline is all one file. As we progress it will likely quickly become desirable to have the content available in a more modular form. The patches introduced for this will break the file up into topical pieces. 

The update will also include a new 'basement' directory where content can be placed that does not need aggressive review and is not specifically targeted to be part of the project deliverables.
