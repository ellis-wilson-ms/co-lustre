--- Description ---
There was a chapter describing Kerberos configuration in the original Lustre user manual (https://docs.oracle.com/cd/E19527-01/821-2076-10/821-2076-10.pdf#G18.5529) that was deleted from the user manual because this feature was never supported.  Now that work has been done to resurrect the Kerberos functionality this chapter should also be restored to the manual.

Unfortunately, the content of this chapter is somewhat out of date and would need to be updated before it is usable.
