--- Description ---
Hello, we're wondering about the DoM component size limitations for a future DoM enabled system. The Lustre manual says the size must be aligned by 64KB:

20.2.3.1. LFS limits for DoM component size

lfs setstripe allows for setting the component size for MDT layouts up to 1GB, however, the size must be also aligned by 64KB. 

 

Why is that? Another example in the documentation shows the use of 32KB, is that okay?

Thanks much.

 Stephane
