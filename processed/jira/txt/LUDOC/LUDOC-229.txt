--- Description ---
It would be helpful to draw attention to the libcfs module parameter which disables Lustre console message rate limiting.  This module parameter is called libcfs_console_ratelimit.  Here is its documentation from "modinfo libcfs"

parm:           libcfs_console_ratelimit:Lustre kernel debug console ratelimit (0 to disable) (uint)

A short section could be added under section 29.2 "Lustre Debugging Procedures".
