CentOS 7.3, Lustre 2.10.3

--- Description ---
I have discovered an error in the current lustre documentation describing tunefs.lustre. the example given in chapter 44.17.4 is incorrect, after "--erase-param" should follow the parameter to erase, in this case "mgsnode"

the given example in the documentation did not work on a lustre 2.10.3 installation, while the following line did:

 

tunefs.lustre --erase-param mgsnode --mgsnode=new_nid --writeconf /dev/sda

 

also, in 44.17.3, the --erase-param option is missing, I'd recommend to add the wording from the man page.
