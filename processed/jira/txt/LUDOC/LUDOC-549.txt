--- Description ---
LU-18357 offers a new approach to set filesets across the cluster permanently. Previously, lctl set_param -P had to be used. Starting with Lustre 2.17, this is no longer required such that lctl nodemap_set_fileset can be used as the primary command.

Since lctl set_param -P can have unexpected side effects, this should be documented in the Lustre Operations Manual.
