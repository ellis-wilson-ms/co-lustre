--- Description ---
Update the /etc/modprobe.conf references in the manual to use /etc/modprobe.d/lustre.conf.  The use of /etc/modprobe.conf has been deprecated for a long time already, and while many current distributions can use arbitrarily-named module parameter files, like in /etc/modprobe.d/Lustre, newer modutils need filenames to end in ".conf", like /etc/modprobe.d/lustre.conf.
