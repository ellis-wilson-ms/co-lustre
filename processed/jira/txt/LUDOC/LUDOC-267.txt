--- Description ---
There was a slide deck done by our SEs showing what tuning parameter changes make Truescale run optimally with Lustre.  This is marked Intel Confidential and none of the details have shown up in community documentation.

This ticket is to get this information into community documentation like the user manual to help direct sysadmins configure for best performance when dealing with Truescale.
