--- Description ---
llapi_path2fid is described in the Lustre manual, but ‘lfs path2fid’ is not mentioned except in one example. 

It would be nice to have a full description of this LFS option. Just repeating what is in the lfs man page in the manual would even be helpful; 

       path2fid [--parents]  ...
              Print out the FIDs for the specified path(s).  If multiple path‐
              names are given, then they will be printed one per line with the
              path as prefix.  The --parents switch makes it output the parent
              FID  and  name(s) of the given entries. If an entry has multiple
              links, these are displayed on a single line, tab-separated.
