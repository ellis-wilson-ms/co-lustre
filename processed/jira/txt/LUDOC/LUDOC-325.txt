--- Description ---
It appears that the Lustre manual content about ACLs is somewhat out of date.  The URL http://www.suse.de/~agruen/acl/linux-acls/online/ no longer works (404 error) so a new source of Linux ACL documentation should be found, for example https://access.redhat.com/documentation/en-US/Red_Hat_Enterprise_Linux/7/html/Storage_Administration_Guide/ch-acls.html should be around for 10 years or so.

Secondly, the discussion of the client acl option is outdated.  Since Lustre 2.7.0 the client acl mount option is no longer available at all and will return an error instead of just printing a warning message that it is deprecated (since it has been deprecated since Lustre 1.8), so mention of mounting the client with noacl should be removed.

Instead, an example of mounting the MDS with noacl should be added to allow disabling ACLs on a system-wide basis.
