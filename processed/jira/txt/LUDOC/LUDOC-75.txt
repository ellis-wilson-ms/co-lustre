In ticket LU-521, a new lst utility environment variable, LST_FEATURES, was added as a way to indicate what features to enable for the lst session.  This ticket has been created to update the documentation regarding this variable.  

Basically, a value of "1" for LST_FEATURES turns on Variable Page Size for testing.
