--- Description ---
In the Lustre manual, the headings that read "Introduced in Lustre 2.x" provide a very effective method to track multiple releases in one document.  The earliest of these headings appears to be a single 2.3, followed by multiple 2.4's, and onward.  With the five (5) 2.10.x releases, the use of 2.10 without the .0-.4 appended appears to be used.  With the release of 2.11.0 now available, and if multiple 2.11.x releases become available, this very effective method to track multiple releases in one document could diminish when not using the full version.  Thus, this suggestion is to use the full release number going forward, and also to update any existing 2.10 release indicators to the full release version 2.10.0, 2.10.1, etc, if warranted.  Lastly, it appears that there may be a typo ("Intro to 2.12") just ahead of section 16.4.  Thanks for considering.
