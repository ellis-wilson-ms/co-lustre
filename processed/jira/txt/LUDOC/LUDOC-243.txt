--- Description ---
The version tags in the Lustre manual obstruct the text of the material which they are tagging.  This is particularly ugly in the LFSCK section (28.4).

This happens because the versionlabel class has the following properties:

  position: relative;
  top: -10px;
  left: 10px;

Since the version label is moved right 10 pixels, it covers up the beginning of the text.  This should be cleaned up.
