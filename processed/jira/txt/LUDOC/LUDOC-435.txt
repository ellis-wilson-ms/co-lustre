--- Description ---
The manual mentions that project quotas are available when using the ldiskfs backend, but it does not mention that the servers must be running a Linux kernel with Lustre patches.  Attempting to run the patchless server Lustre packages will results in the MDT failing to mount if project quotas are enabled.
