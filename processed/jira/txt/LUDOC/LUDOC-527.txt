--- Description ---
Lustre manual section 14.9.3: "Removing an OST from the File System" makes reference to using lctl llog_cancel, but the syntax it uses is wrong for at least lustre 2.12.

It says:

lctl --device MGS llog_cancel testfs-client -i 138

But it should say:

lctl --device MGS llog_cancel --catalog testfs-client --log_idx 138

 

The documented syntax appears to not exist in 2.12 nor 2.15.
