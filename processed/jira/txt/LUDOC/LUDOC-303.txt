--- Description ---
Current documentation shows a file system limit of 1 EB (ZFS) yet shows a maximum file size of 8 EB (ZFS).  How can a file be larger than the underlying filesystem?If the 1 EB file system limit is correct, the maximum file size should also be 1 EB.

--- Comment (adilger, Tue, 23 Feb 2016 17:56:36 +0000) ---
There can be holes in a file, so the maximum file size can be larger than the maximum filesystem size if there are holes in the file.
