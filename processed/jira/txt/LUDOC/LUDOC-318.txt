--- Description ---
Fop is used to build the manual and does not hypenate text automatically. Output from fop includes:

SEVERE: Couldn't find hyphenation pattern for lang="en".

--- Comment (rhenwood, Wed, 16 Dec 2015 17:41:18 +0000) ---
The automatic builder does have hyphenation working correctly. It seems this was an issue local to my own environment.

I fixed it locally by downloading hyphenation pattern files from http://offo.sourceforge.net/hyphenation/index.html

making the jar file available locally and then using CLASSPATH=path-to-hypen-jar fop infile.fo outfile.pdf
