--- Description ---
For analysis of some lfsck behavior and for debugging problems where you want to investigate files where MDT fid is known but the fs is down, it would be helpful to have a tool to parse MDT oi files to extract inode # given a fid.

This is a request for a tool to be able to dump mdt oi files and show the inode to fid mapping.
