--- Description ---
The dkms install/build just runs with mostly an auto detection of features that can be built.

It would be better if the user is provided some additional control over the configuration, 

currently the /etc/sysconfig/lustre file is checked but only a very limited set of options are available.

This should be expanded to a useful set of features as well as the ability to pass custom configure parameters through the dkms installation phase.
