--- Description ---
The Lustre pacemaker script lacks a mount option parameter.  I'm going to submit a patch to support mount options which is useful for SSK (eg: "-o skpath=/path/to/key").  Also, the patch will add logging of the output from mount and umount commands and the return code from them on error.
