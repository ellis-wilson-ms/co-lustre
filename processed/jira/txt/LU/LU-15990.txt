--- Description ---
It does not currently appear possible to run "lfs find" to locate directories in the filesystem that are only partially migrated (i.e. directories with the "migrating" flag set).  This is useful for two reasons:

	to be able to scan the filesystem to locate directories that have not completed MDT migration (LU-14719)

	to be able to restart/finish the MDT migration for those directories (LU-14211, LU-15001), assuming the reason the migration was interrupted has been resolved
