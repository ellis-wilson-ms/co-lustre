--- Description ---
Previously, a use-after-free bug in the Common Block IO controller cgroup

interface code could result in a kernel panic on systems using the blkio

subsystem. An upstream patch that fixes this bug has been backported to the

kernel, and the kernel no longer crashes when working with blkio cgroups.

(BZ#1158313)
