--- Description ---
As an extension to LU-17516 it would be useful if "lfs df --mdt/--ost" took an optional argument "[INDEX]" to specify a single MDT/OST index for which to print the usage.  

That would avoid a lot of cases where "lfs df | grep ..." is used in test scripts to find space usage for a particular target, and is likely equally useful for end users.
