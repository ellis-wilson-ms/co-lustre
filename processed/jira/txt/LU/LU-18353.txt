--- Description ---
It is impossible to set inode limits for Pool Quotas until MDT Pools would be implemented. 

lctl pool_list dltawork
Pools from dltawork:
dltawork.ddn_hdd
dltawork.ddn_ssd
dltawork.ddn_sas_ssd

lfs setquota -p 101 --pool ddn_ssd -b 10G -B 11G -i 0 -I 0 /lustre/exafs/client/testfs/
lfs setquota: Cannot find pool 'ddn_ssd'
lfs setquota: quotactl failed: No such file or directory
setquota failed: No such file or directory 

We need to add a check in utils:set_quota to return an error and print a warning if someone tries to set inode limit for quota pool to avoid confusing.
