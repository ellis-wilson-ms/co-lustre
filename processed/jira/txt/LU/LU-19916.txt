--- Description ---
Start MGS replicas on MDTs/OSTs which are not located on the MGS0

primary and failover hosts.  And once the target which the MGS

replica is running on is stopped, the MGS replica will be stopped

as well, and it will notify the ll_cfg_requeue thread to start

another MGS replica on other target if possible.

Register MGS replica as observer of MGC, so that it can be notified

when the config is changed.
