--- Description ---
When creating remote directory, the file project quota for the agent inode has been

transferred to the project ID 0 if the project ID of the parent directory isn't 0 and the flag

LUSTRE_PROJINHERIT_FL is set, but the project ID of the agent inode is reset, then

the file project quota is decreased during destroying the agent inode when the remote

directory is removed.
