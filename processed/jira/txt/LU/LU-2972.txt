--- Description ---
Due to initialization failure of the infiniband interface for external OFED builds (ofa) a change (http://review.whamcloud.com/#change,5630) has been created, to add the (boot) initialization script provided by the OFED software to the kernel-ib RPM again. The problem is documented in LU-2907.

The distribution package 'rdma-3.6-1.el6.noarch' provides an initialization script to configure the infiniband interface, too. This might lead to conflicts during the system boot/shutdown phase.

Please investigate whether both scripts can be installed and operate without problems.

--- Comment (heckes, Mon, 22 Apr 2013 16:14:42 +0000) ---
Problem is obsolete since the startup of mlx4 kernel modules is handled by the rdma init script now (see LU-2907) correctly again. The openibd init script is removed from the packaging list of the ofa (OFED) RPM at build time.
