Linux 4.7 commit ae05327a00fd47c34dfe25294b359a3f3fef96e8 replaces ext4_dir_operations iterate with iterate_shared. dir_relaxed_shared() was also added in that commit, so we can use that function to verify that the ext4_dir_operations is using iterate_shared.
