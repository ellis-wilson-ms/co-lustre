--- Description ---
http://doc.lustre.org/lustre_manual.xhtml#dbdoclet.lfsmkdir specifies that enable_remote_dir can be used to control whether remote sub-directories can be created off of any MDT.  However, the control appears to have been removed with LU-3537, but the interface wasn't removed.  When this code changes, an LUDOC update will be necessary.
