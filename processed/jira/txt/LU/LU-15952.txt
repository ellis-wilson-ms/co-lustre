--- Description ---
The man page of lfs project -C states,

SYNOPSIS
   ...
       lfs project -C [-r|-k] 

      Clear the project inherit flag and ID on the file(s) or directories
       -C     Clear inherit attribute, project ID will be reset to be 0 in default
       -k     keep the project ID unchanged.

Actually there are 3 options to specify the recursion behavior:

 

-d  process only the top directory
-r  process the top directory and all its desendants recursively
  if neither -d nor -r is specified, process the top directory and its immediate children

Need to make the man page clear to users.
