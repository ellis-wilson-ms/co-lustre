--- Description ---
Add options to "lfs find" to be able to find files by their stripe_count, stripe_index, and stripe_size.  For stripe_index (alias ost_index) it is possible to specify a comma-separated list of OST indexes.  Make the short options for these consistent between "lfs find", "lfs getstripe" and "lfs setstripe", keeping the old options for user interface compatibility.
