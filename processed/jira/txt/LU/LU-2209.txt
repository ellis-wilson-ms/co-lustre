SLES 11 SP2 client

--- Description ---
When doing package updates in SLES11 SP2 I see the following warnings, apparently due to installed lustre rc scripts:

insserv: warning: script 'lnet' missing LSB tags and overrides

insserv: Default-Start undefined, assuming default start runlevel(s) for script `lnet'

insserv: warning: script 'lustre' missing LSB tags and overrides

insserv: Default-Start undefined, assuming default start runlevel(s) for script `lustre'

While this doesn't look at all fatal, it suggests some required header info is missing from lustre supplied rc scripts.  This should probably be cleaned up by adding headers that SLES likes/requires.
