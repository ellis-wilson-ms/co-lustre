--- Description ---
It is not possible to use "lfs find ..." to locate directories that have specific/bad default directory layouts on them.  This would be useful for finding directories that have e.g. default striped directories, which have caused many problems in the past, something like:

# lfs find /mnt/testfs/* -type d --mdt-default --mdt-count +1 -print0 |
     xargs -0 lfs setdirstripe -d
