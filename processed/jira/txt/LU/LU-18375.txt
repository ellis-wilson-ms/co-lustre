--- Description ---
sanity/27cc (and likely few other tests) can't parse PFL layouts and generates a false failure.

legacy layout (27cc can recognize) looks this way:

	obdidx		 objid		 objid		 group
	     0	             3	          0x3	   0x240000400
	     0	             4	          0x4	   0x240000400

PFL layout:

      - 0: { l_ost_idx: 0, l_fid: [0x240000400:0x2:0x0] }
      - 1: { l_ost_idx: 0, l_fid: [0x240000400:0x3:0x0] }
