--- Description ---
The liblustre testing is currently failing.  Since this functionality is not being used by any (or at least not many?) users since it is no longer used for the Cray Catamount compute clients, the liblustre testing should be removed from automated test runs.

If a champion for this code can be found in the community, the fixes can be added to the tree and this test can be re-enabled.  As yet, the liblustre code will not be removed from the tree.
