--- Description ---
The PFL 3b phase requires MDS does not instantiate all OST objects at the file create time, and only instantiate necessary objects when the file's size grows. So that the stripe lovea on the disk could contain layout component with invalid ost_id if its lcme_flags does not contain LCME_FL_INIT flag. LFSCK needs to take care of this kind of composite layout.
