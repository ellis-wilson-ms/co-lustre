--- Description ---
It would be very very lovely to be able to pass lfs changelog a value to have it automatically translate FID's into path names.

So something like this:

4105 08RENME 18:23:41.197370958 2013.09.11 0x0 t=[0:0x0:0x0] p=[0x200000400:0x73c:0x0]
        oldfile s=[0x200000400:0x743:0x0] sp=[0x200000400:0x73c:0x0] newfile

Would display like:

4105 08RENME 18:23:41.197370958 2013.09.11 0x0 t="" p="/save" "oldfile" s="/save/a" sp="/save" "newfile"
