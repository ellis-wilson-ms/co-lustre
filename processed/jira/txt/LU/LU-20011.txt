--- Description ---
If a file with a partial PFL layout (i.e. no EOF component) has a mirror added with "lfs mirror extend" it is then impossible to remove the mirror, add a new EOF component, or "lfs mirror split" to return to the original state, receiving the error "lfs mirror: Invalid layout: Incomplete mirror - must go to EOF".

Whatever generates this error should be ignored, and the split or "setstripe --comp-add" command should be allowed to bring the file back into a consistent state.
