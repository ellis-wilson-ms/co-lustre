--- Description ---
There is an rw_semaphore

struct iam_container {
    ...
        /*
         * read-write lock protecting index consistency.
         */
        struct rw_semaphore     ic_sem;   /* Protect ic_idle_bh */
        struct mutex         ic_idle_mutex;
     ...
};

There is initialization

 2    234  lustre/osd-ldiskfs/osd_iam.c >
             init_rwsem(&c->ic_sem);

There are wrappers

 3    622  lustre/osd-ldiskfs/osd_iam.c >
             down_write(&ic->ic_sem);
   4    627  lustre/osd-ldiskfs/osd_iam.c >
             up_write(&ic->ic_sem);
   5    632  lustre/osd-ldiskfs/osd_iam.c >
             down_read(&ic->ic_sem);
   6    637  lustre/osd-ldiskfs/osd_iam.c >
             up_read(&ic->ic_sem);

But this wrappers are not used. And, based on the git history, never been used.

Let's delete them.
