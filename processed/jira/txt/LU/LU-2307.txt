--- Description ---
MDT-OST consistency needs to implement functionality for distributed verification and repair of the MDT inode to OST object layout. This will check the file layout to verify that the objects referenced in the file layout exist, and that each OST object has a back reference to the correct MDT inode. Incorrect or missing back pointers on the OST will be fixed.
