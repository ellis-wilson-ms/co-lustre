--- Description ---
It should be possible to add a NOTRASH attribute to a file or directory, so that when it is deleted the MDS will bypass the Trash Can completely and delete the file immediately (as it was without the Trash Can feature).

If the NOTRASH attribute is set on a directory, then new files in that directory will inherit the NOTRASH attribute.

This is intended to mark directories that contain very temporary files that should not consume space in the Trash Can.
