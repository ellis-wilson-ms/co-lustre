--- Description ---
Analyzing kernel logs manually can be challenging. When reading kernel logs, it can be hard to find out manually when an error appeared more frequently. However, with some shell commands, this task can be done quite easily.

Although these scripts make it easier to observe the frequency of a particular error, comparing it to other errors and events on different machines can still be cumbersome.

Having a tool to plot this information and provide a visual representation quickly readable would be valuable.
