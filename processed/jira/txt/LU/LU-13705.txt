--- Comment (adilger, Wed, 28 Apr 2021 02:39:00 +0000) ---
It is now possible to specify stats files using the same syntax as "lctl get_param", for example "llstat osc.*OST0000*.stats" or equivalent.

What it doesn't yet do is aggregate stats between multiple stats files that match a "*" like "llstat osc.*.stats" into a single output.  It currently picks the first matching file and reports only those stats.
