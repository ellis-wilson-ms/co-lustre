Hi there

You are correct in deducing that SLES11 SP2 is not supported by Lustre 2.2. The problem is that SLES11 SP2 again jumps ahead in the kernel version used. We need to add support for the Linux 3.0 kernel before we can add SLES11 SP2 support. This development is in progress and is being tracked under LU-812
