--- Description ---
When using the changelogs to monitor changes to a file system, deleting directories can result in it being difficult to figure out what was deleted, as using the fid2path utility does not work with fids that reference deleted directories. Recording the full path in the changelogs would allow this information to remain accessible.

If the parent directories have been deleted, then there should also be Changelog RMDIR records for those directories later in the logs that can be used to reconstruct the full pathname. 

For most use cases the pathname of the deleted files/dirs is not critical to preserve, but it should be possible with the information available.
