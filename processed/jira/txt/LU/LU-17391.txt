--- Description ---
Linux recently added experimental support for Rust (https://rust-for-linux.com/). It would be interesting to experiment with Rust bindings for Lustre and LNet. In the future, Rust could be an option for new LNDs or OSDs.

 

Ubuntu provides some out-of-the-box support for Rust in the kernel (https://discourse.ubuntu.com/t/ubuntu-kernel-is-getting-rusty-in-lunar/34977). This makes is simple to test simple Lustre kernel modules in Rust.
