--- Description ---
We are seeing frequent messages like this:

Nov 21 12:20:58 hype352 kernel: LustreError: 7689:0:(osc_request.c:1229:can_merge_pages()) is it ok to have flags 0x420 and 0x5a8 in the same brw?

on the console of 2.1 lustre clients.  That is a pretty terrible console message.  One way or another we need to eliminate it.

If it is a problem, we need to fix whatever is going one.  If there is no problem, the message needs to be removed from the console.

0x100 = OBD_BRW_NOQUOTA

0x080 = OBD_BRW_NOCACHE

0x008 = OBD_BRW_SYNC

There was a patch recently merged to master to address OBD_BRW_NOQUOTA issue from LU-651:

http://review.whamcloud.com/1328
