Hi Etienne, I was going to point someone at the Changelog man pages that you previously added in LU-12837 to help them understand "lfs changelog" output and what records are returned via "llapi_changelog_recv()", but I don't see a description of the different Changelog records (or the ASCII formatted versions printed by "lfs changelog") anywhere in these man pages or in the Lustre manual.

Would you be able to update the llapi_changelog_recv.3 man page to describe struct changelog_rec and the various fields/flags/constants therein?  It looks like the CHANGELOG_FLAG and CHANGELOG_EXTRA_FLAG are defined in llapi_changelog_start.3, but there would need to be a description of what these flags mean for decoding changelog_rec in llapi_changelog_recv.3.  

That also makes me wonder if we should add a new llapi "wrapper" function that hides the details of FLAGS vs. EXTRA_FLAGS and decoding of the changelog records?
