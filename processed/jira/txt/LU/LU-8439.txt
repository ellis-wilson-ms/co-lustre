--- Description ---
Currently running ssh all the time for every minute command seems to add quite a bunch of extra time in our tests.

There's a hope that using permanent ssh connections is going to help it.

Apparently openssh hs an option to reuse the connection too, but I am not sure if we can make it totally permanent or if we'll need to have one deliberately held background one to make that work,

but this needs to be tried and possibly made a standard part of the framework.
