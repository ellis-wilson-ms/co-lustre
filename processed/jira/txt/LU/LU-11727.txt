--- Description ---
lfs hsm_* commands have a --filelist argument.  From the code, it appears it might have to be last on the command line.  It's only used in sanity-hsm with a single argument, which is not really a list.  It's not documented in lfs-hsm.1

 

I think we can just remove it completely, since the list of arguments at the end of the command line are assumed to be the list of files to operate on.
