--- Description ---
Since separate MDS and MGS is a common and recommended Lustre set-up, full test group and patch testing should be run against this Lustre configuration. We can request a separate MDT and MGT (both on the same node/VM) for autotesting by using ‘Test-Parameters: combinedmdsmgs=false’ in a patch commit message and, thus, we are able to test separate targets (only separate targets and not separate nodes). 

The purpose of this ticket is to track the various efforts/tickets that are working to get the Lustre test suites running on separate MDS and MGS.
