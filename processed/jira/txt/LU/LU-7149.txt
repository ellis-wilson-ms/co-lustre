--- Description ---
Test 224c sanity.sh always passed when executed consecutively. Even when executed without fix and expected to be failed.

But test_224c fails as expected with timeout if run ONLY=224c sh sanity.sh on fresh created lustre.

The problem is that some tests set off writethrough, using "set_cache writethrough off" and don't restore it before 224c, that needs writethrough to be set on.
