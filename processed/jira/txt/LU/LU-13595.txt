--- Description ---
It would be convenient for lustre_rmmod to have an option like "--debug-kernel|-d" to dump the kernel debug logs before the libcfs module is unloaded.  This would facilitate log collection related to memory allocation (if +malloc debugging is enabled), or subsystem cleanup.
