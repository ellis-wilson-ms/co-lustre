--- Description ---
The LUTF writes its results to a file once all of the tests have completed. If the controller crashes, the script exits prematurely, etc. then there will be no record of what happened. This ticket is for the LUTF to write the results out as the tests are running so crashes, timeouts, etc. can be diagnosed
