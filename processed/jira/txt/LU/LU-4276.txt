--- Environment ---
sles11 sp2 and sp3 servers

--- Description ---
ext4 and hence ldiskfs in sles11 servers has config and module param options to turn on read/write access to the loaded filesystem.  Currently it is off by default.  For ldiskfs in our sles11 servers we really need it to be on by default.  It shouldn't need to be changed from default by module options in order to work correctly.

I plan to push a patch to add a build time config option setting it on by default.
