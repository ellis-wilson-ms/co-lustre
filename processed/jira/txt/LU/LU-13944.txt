--- Description ---
Patch adds the ability to set clients parameters via CLIENT_LCTL_SETPARAM_PARAM.

Example:

      CLIENT_LCTL_SETPARAM_PARAM="lbug_on_eviction=1 debug=-1 "

sets lbug_on_eviction=1 and debug=-1 on all clients.

The ability to set clients parameters makes failures investigation more effective.
