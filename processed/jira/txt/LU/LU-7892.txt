--- Description ---
The lustre/utils/create_iam.c utility was used at one point to format the OI file in userspace during MDT formatting.  However, this has not been needed for a long time since osd-ldiskfs will create the OI file now, and it can be deleted.  There are several places in the code that reference "create_iam" in the comments that should also be deleted.
