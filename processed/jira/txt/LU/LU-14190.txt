--- Description ---
The krb5 releases 1.18 and later completely remove support for all DES3 enctypes (des3-cbc-raw, des3-hmac-sha1, des3-cbc-sha1-kd).

--- Comment (sebastien, Tue, 18 Mar 2025 07:10:03 +0000) ---
Done as part of "LU-18256 gss: deprecate insecure enctypes" https://review.whamcloud.com/56512, merged from 2.16.0-RC1:

LU-18256 gss: deprecate insecure enctypes

    A number of encryption types declared in the GSS code are deprecated
    for security reasons, and should not be used. So remove support for
    them in the Lustre code:
    - des-cbc-crc
    - des-cbc-md4
    - des-cbc-md5
    - des-cbc-raw
    - des-hmac-sha1
    - des3-cbc-sha
    - des3-cbc-raw
    - des3-cbc-sha1
    - arcfour-hmac
    - arcfour-hmac-exp
