--- Description ---
Lustre will acquire a group lock while 'lfs migrate --block' running. But the CBPENDING flag will cause a new group lock be created against same file. This issue exists a long time. It was revealed by change http://review.whamcloud.com/16456/ and fixed by http://review.whamcloud.com/17838/. So we should add a test case for 'lfs migrate --block' to avoid such issue lurking again.
