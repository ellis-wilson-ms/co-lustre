When an OST is set to degraded mode, there is no way of verifying its state from the OSP side, the only way to check if the state has been transferred to OSP side is to run 'lfs df', whicn is not necessary the most convenient method.

This ticket will add a procfs entry on the OST side to show the OST state. The patch is written by Andreas.
