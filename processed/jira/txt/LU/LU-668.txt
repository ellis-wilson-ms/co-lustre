--- Description ---
Fully remove MDS_FL and replace them with LUSTEFL.  These values are explictly not defined using LDISKFS*_FL as was done in mdd_internal.h to avoid a dependency on ldiskfs.h.

"These flags are part of the wire protocol.  We already have portable definitions of these as "MDS_APPEND_FL", et. al. in lustre_idl.h.  Since MDS_FL is no longer used in the Lustre code (in preference to LUSTRE_FL), it makes sense to just rename these in lustre_idl.h, and their corresponding uses in wire*.c."

  – Andreas

This was originally filed as Lustre bugzilla 22408.
