--- Description ---
Test steps:

1) Mount the large stripe count (2000 OSTs in total) filesystem on Lustre 1.8.7-wc1, 2.1.1 and 2.2.0 clients.

2) Run "lfs setstripe" on CLIENT1 (version 2.2.0) to create a file with 2000 stripes.

3) Run "lfs getstripe" on CLIENT2 (version 2.1.1) and CLIENT3 (version 1.8.7-wc1) to get the striping information.

......

While running step 3), the striping information was gotten correctly. However, in the meantime, lfs hit the following issue on both 2.1.1 and 1.8.7-wc1 clients:
