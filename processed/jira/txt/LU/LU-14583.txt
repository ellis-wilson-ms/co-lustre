MPI-IO using llapi_file_get_stripe() returns error, when trying to get striping info for a soft-link. llapi_file_get_stripe() returns “get_stripe failed: 61 (No data available)”

llapi_file_get_stripe should follow softlink. 

The softlink point back to a file on lustre.
