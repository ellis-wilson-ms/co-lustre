--- Description ---
The handler for 

LL_IOC_{GET,SET}_LEASE

 may return 

FMODE_{READ,WRITE}

 according to the type of lease. 

FMODE_{READ,WRITE}

 are internal to the Linux kernel and should not be used in this way. The only user of these ioctls is multiop so it's a good time to fix them.
