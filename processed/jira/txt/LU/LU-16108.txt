--- Comment (adilger, Fri, 26 Aug 2022 13:23:17 +0000) ---
If you don't have any "lod.*" parameters it is likely because you are not running this on the MDS, or the filesystem is not mounted, or using a Lustre version that does not support DoM (no version information is given in the ticket). This is a parameter on the MDS and should be set there.  If setting it permanently, then use "lctl set_param -P lod.*MDT0000*.dom_stripesize=NNNN" on the MGS (if separate from the MDS).

mds# lctl get_param lod.*.dom_stripesize
lod.myth-MDT0000-mdtlov.dom_stripesize=1048576
