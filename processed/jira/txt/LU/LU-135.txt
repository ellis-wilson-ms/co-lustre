--- Description ---
Update e2fsprogs to be able to build against both b1_8 and master versions of Lustre.  Much of this work was done as part of https://bugzilla.lustre.org/show_bug.cgi?id=13698.  Cleanup is needed in order to build and run correctly with the upstream e2fsprogs "master" branch (for 1.42).

This will also resolve the "build against installed RPM" problem discussed in LU-45.
