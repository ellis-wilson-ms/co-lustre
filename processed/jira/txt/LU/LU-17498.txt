--- Description ---
It is useful to see the NIDs of the nodes involved in a test run so that error messages can be identified as relating to a specific client or server.

Currently the Maloo test session summary page shows the network type and hostnames, but not the addresses/NIDs themselves. Since the hostnames can only be resolved internally, it is harder to find this information by digging through the logs.
