--- Comment (niu, Mon, 5 May 2014 03:41:57 +0000) ---
MDT assumes the path of ROOT is '\0', see mdt_path():

        if (lu_fid_eq(&mdt->mdt_md_root_fid, mdt_object_fid(obj))) {
                path[0] = '\0';
                RETURN(0);
        }

However, lustre_rsync assumes the path of ROOT is '/', see lr_create():

        if (len == 1 && info->path[0] == '/')
                snprintf(info->dest, PATH_MAX, "%s", info->name);

I think all lustre components should consider the path of ROOT as '/'.
