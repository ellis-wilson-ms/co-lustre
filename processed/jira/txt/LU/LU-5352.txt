--- Description ---
In dt_index_read() we test to if the unsigned int rdpg->rp_count is 
        /* rp_count shouldn't be null and should be a multiple of the container           
         * size */
        if (rdpg->rp_count rp_count & (LU_PAGE_SIZE - 1)) != 0)
                RETURN(-EFAULT);

The '&&' should be '||'.

This was found using the clang analyzer.
