--- Description ---
Extend the existing File Level Replication (FLR) state machine to support erasure-coded layouts. EC files will follow the same state transition model as mirrored files, including transitions through RDONLY, WRITE_PENDING, SYNC_PENDING, and related intent handling.

This work is limited to enabling state transition and management logic for EC components. It does not include I/O support or any operations involving parity components.
