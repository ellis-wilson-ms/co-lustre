--- Description ---
The "project" feature should be enabled on newly formatted ldiskfs filesystems by default, since adding a projid to a directory tree is useful for a number of different reasons, not just limiting space usage by quota.

The projid is an easy way to get "du" usage for a subtree, for example, or for PCC.
