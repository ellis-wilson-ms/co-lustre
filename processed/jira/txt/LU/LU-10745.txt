--- Description ---
The SLE12 SP2 and SP3 kernels use the same underlying base version but have release numbers that are indistinguishable.

The SP3 kernel has clean_bdev_aliases while the SP2 kernel does not, so we can use its presence to detect which kernel is used.

It has thus far just worked because the baseline ext4 implementation is more or less the same between them.
