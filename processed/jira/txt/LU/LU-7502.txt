--- Description ---
There should be options for lfs find and lfs getstripe to use --mdt-count and --mdt-hash options for lfs find and lfs getstripe to return the striped directory stripe count and directory hash function respectively. There isn't a good short option for --mdt-count, since -M is already used for mtime.  While --mdt-hash could use -H, it isn't clear that this is an option that will be used enough to justify having a short option.
