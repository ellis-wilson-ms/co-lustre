--- Description ---
Currently 'lfs project' only prints the project id followed by the file name, it could be useful to include the project name along side the id, something like:

$ lfs project -p quota_usr /mnt/lustre/tfile 
$ lfs project /mnt/lustre/tfile 
60000:quota_usr - /mnt/lustre/tfile

--- Comment (adilger, Wed, 15 Jan 2025 18:58:34 +0000) ---
For commands like "ls" they will default to printing the name, if it can be looked up, and print the number of the lookup fails.  "ls -n" will only print the number, for cases where that is important, or if the name lookup is slow/broken (eg. using a network LDAP database that is overloaded or offline).

I think "lfs quota" is doing the same (prefer name, fall back to number).
