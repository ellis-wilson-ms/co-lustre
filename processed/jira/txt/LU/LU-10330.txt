--- Comment (adilger, Tue, 5 Dec 2017 08:11:59 +0000) ---
It looks like this is caused by the test checking for version 2.8.52 on the server before setting brw_size to be larger. With 2.7.x the 16MB RPC patch is available, but the default brw_size is 1MB.

The test needs to be updated to check for version 2.7.17 or later to set brw_size=16 and enable the 8MB and 16MB RPC tests. The suffix="M" change is NOT available in 2.7.x and should be left under the version > 2.9.52 check.

This should be fixed in b2_10 and master.
