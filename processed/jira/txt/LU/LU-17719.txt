--- Comment (simmonsja, Wed, 10 Apr 2024 00:15:14 +0000) ---
Your config file is invalid. The latest code supports

net:

    - net type: o2ib9

       local NI(s):

          - interfaces

So you can remove the "- nid:" line. The reason for this new restriction is due to LU-13642 which allows picking an IP to use. This was added because an interface can have multiple IP addresses.
