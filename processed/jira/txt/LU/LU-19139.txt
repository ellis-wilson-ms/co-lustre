--- Description ---
Add a new module parameter (e.g. acceptor_port_bulk). If specified, this port will be intended for use with bulk connections, while control connection shall be using the original acceptor_port. 

The intention is to separate bulk and control traffic so that network administrators are able to apply QoS priorities based on the connection type.
