--- Description ---
This ticket is to make the changes in t-f, for ost-pools suite, when the cluster setup has separate MGS and MDS configured on different nodes.

The Lustre client needs to be mounted on MGS node for ost-pools suite.

The lctl command MUST be run on the MGS. Another requirement for managing OST pools is to either have the MDT and MGS on the same node or have a Lustre client mounted on the MGS node, if it is separate from the MDS.
