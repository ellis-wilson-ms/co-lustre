--- Description ---
For new writting, Lustre reserve credits in worst cases, eg for 1M write, we need reserve more than 500 credits.

This means we need much more journal space than data for each write, which means we need large journal to reach good performance.

One of potential way to improve this could be we allow transaction restart for write commit

case.
