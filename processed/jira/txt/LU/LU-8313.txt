--- Description ---
There is no real definition of what constitutes a SLOW test, the general rule of thumb is, if a test runs for five minutes or more, it is eligible to be marked as SLOW. Attached is the list of tests which can be reintroduced to the normal review process as they are not running for five minutes or more and some new tests that can be deemed as SLOW test as they run for five minutes or more. The tests were run multiple times to have more accurate time values.

List of tests in respective test suites to be removed or added to SLOW list:

replay-oat-single.sh

Removed - test_8a, test_8b

sanity-quota.sh

Removed - test_9, test_12a from ZFS_SLOW

sanity.sh

Removed - test_24D , Add - test_24v, test_51e, test_230d

sanityn.sh

Add- test_33b, test_80b
