--- Description ---
Server side part of layout swap is implemented in MDD layer, which isn't optimized, because that'll expose the layout information in MDD layer (the layout information is supposed to be understood by LOD only). We'd better re-implement it in the proper layer: LOD.

We also looked into layout in MDT layer to check if file is released (see mdt_pack_attr2body()), that needs be fixed somehow.
