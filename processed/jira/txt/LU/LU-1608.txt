--- Description ---
Require that the filesystem name be specified at format time.  The current default is "lustre", but this can cause problems if multiple filesystems are created at the same site with this default name.  At a later time, if the filesystems need to be mounted on the same client, this will cause problems.

This is also confusing in many examples in the manual, where it isn't clear whether "lustre" needs to be specified literally, or it is just an example.
