--- Description ---
Enable building lustre against the Oracle Linux UEK kernels

Add detection for uek kernel signature and suppress warnings about expecting an Ubuntu, RHEL or SLE kernel.

Enable building if Oracle OFED extensions are not suppressed (kernel default is enabled).
