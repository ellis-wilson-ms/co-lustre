--- Description ---
Before Lustre 2.10.1 GA IML has been using Lustre Jenkins builds. These builds include zfs rpms that IML uses.

In the Lustre download site, we don't have these same rpms in tow, so installing with ZFS fails.

This is a request to add those same zfs rpms to the download site, or failing that, someone to sign off that they are ok with us shipping IML 4.0 against a lustre jenkins URL.
