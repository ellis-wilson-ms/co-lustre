--- Comment (utopiabound, Thu, 31 Aug 2017 14:39:17 +0000) ---
https://review.whamcloud.com/#/c/28464/ landed with a larger disk2_7-zfs image and re-enabled this test (it passed with ZFS 0.6.5.9).  Quota verification isn't enabled for ZFS prior to 0.7.0, and this test fails for 0.7.0.  This test may NEVER have successfully passed quota verification.  I think the 40K value may be correct and we can fix the image by adding a blimit file with 40960 as a value.  New images use 40K and have blimit, old images defaulted to 20K (which gets used if no blimit file is present).
