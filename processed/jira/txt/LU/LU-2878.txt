--- Description ---
A recent set of Livermore changes to the build system has broken "make install".  "make all" still works.  For some reason, ldiskfs is failing a dependency check:

 *** Missing ldiskfs symbols. Ensure you have built ldiskfs:
 *** - /home/build/lustre-release/ldiskfs/ldiskfs/Module.symvers

Either a- this needs to be fixed so developers who don't want to have to build RPM's every time can continue to use "make install", or b- "make install" should be removed.
