--- Description ---
The following doesn't find a directory that has a default file layout with stripe-offset set to 0:

lfs find -type d -i 0 /mnt/lustre

 

Reproduction:

# mkdir /mnt/lustre/dir0
# lfs setstripe -i 0 /mnt/lustre/dir0
# touch /mnt/lustre/dir0/stripe0

# lfs find -type d -i 0 /mnt/lustre

This should print /mnt/lustre/dir0, but returns nothing.
