--- Description ---
Traditionally the permanent parameter setting command 

lctl conf_param/set_param -P

 is run on MGS node, now that these parameters are maintained by Raft, such command should be run on Raft node.

To simplify the administration, such command can be run on any Raft voting nodes, if the node happens to be Raft leader node, the parameter is appended to Raft, which will be synchronized to other Raft nodes; otherwise the parameter is sent to the Raft leader node.
