--- Description ---
We found performance drop in a buffered write up to 15%-20% if compare with a baseline without PQ patches.

In both cases quota was turned on (enabled="ugp") and set only one limit - block hard limit 500T. Also there was at least one OST pool without any quota limits.

Note, this performance issue can't be seen if there is no OST pools in a system.
