--- Description ---
The LUSTRE_DIRECTIO_FL was added from the upstream kernel FS_DIRECTIO_FL in the hopes that it might be useful, but it has since been removed from the upstream kernel in commit v4.4-rc4-22-g68ce7bfcd995 and replaced by FS_VERITY_FL using the same value in commit v5.3-rc2-4-gfe9918d3b228.

Since we have never used this flag in Lustre, there is no risk to remove it.
