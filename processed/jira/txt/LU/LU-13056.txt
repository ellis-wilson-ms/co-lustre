--- Description ---
The "lctl llog_print", "lctl llog_catlist", and related commands should default to using the "MGS" device if it is not specified.  This is overwhelmingly the most common device to use (it isn't clear that the work for any other device), and having to add "--device MGS" for each use just adds usage difficulty without any benefit.
