--- Description ---
Adding an additional rule that calls an lnetctl script after module start. to enable network configuration (ip2net, ..) from user level during startup and bypassing kernel parsing. Basically moving configuration from lnet.conf which is parsed by the kernel to a user level script and avoiding kernel parsing crashes and making it more flexible.  

This is described in https://wiki.whamcloud.com/display/LNet/Dynamic+LNet+Configuration+Concepts+and+Implementation
