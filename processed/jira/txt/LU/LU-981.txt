--- Description ---
To increase performance using small files, as several lustre guides recommend, we created a "big" file to use it mounted as loopback. When you write some data the client stops and the whole computer hangs.

	
	
		It works using version 1.8.5!!!

	
	

Example:

dd if=/dev/zero of=/lustre21/my_fs_personal.iso bs=8192 count=9999

mkfs.ext4 /lustre21/my_fs_personal.iso

mount -t ext4 -o loop /lustre21/mi_fs_personal.iso /mnt

dd if=/dev/zero of=/mnt/example_file.dat bs=8192 count=8999
