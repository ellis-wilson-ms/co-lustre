--- Description ---
Volatile files are used as temporary file to copy data and then swap layout with some other  file, e.g. during HSM release and restore. Such files should have no DoM layout because it cannot be swapped without data loss. Considering that volatile files should drop DoM component or replace it with RAID0 stripe either when striping was provided during create/setxattr or was taken from default striping.
