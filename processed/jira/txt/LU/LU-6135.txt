--- Description ---
In LU-4665 "utils: lfs setripe to specify OSTs" support was added to lfs setstripe to allow specifying a list or range of OSTs on which to create a new file using the -o option.

It would be useful to fully support this functionality using other methods such as setting default directory layouts and using the new llapi_layout_* API.
