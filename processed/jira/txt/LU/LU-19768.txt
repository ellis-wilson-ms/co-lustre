--- Description ---
The Lustre stats collection via Netlink/YAML is incomplete.  It is not currently possible to access statistics from non-"stat" files, such as md_stats, jobstats, brw_stats.

These (and other) statistics need to be accessible via Netlink in order to provide a comprehensive monitoring interface for Lustre without having to scrape and parse text output via lctl.
