--- Description ---
If a yaml file configures multiple interfaces in one go, only the first interface is configured with the LND tunables. The other interfaces are not.

Also adding ntx tunable to be configurable from lnetctl. This will allow users not to rely on the ko2iblnd.conf to configure the recommended setting for increased OPA performance. It will also allow NIs to be configured with their own unique tunable values.
