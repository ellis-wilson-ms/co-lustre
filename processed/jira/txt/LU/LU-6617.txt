--- Description ---
Quota migration code was used to migrate old quota admin files into new format when user upgrade system from old 2.1 system.

These code can be removed now, if anyone want to upgrade from such an old release (and want to keep old quota limits settings), he should upgrade to an intermediate release to migrate the quota admin files first.
