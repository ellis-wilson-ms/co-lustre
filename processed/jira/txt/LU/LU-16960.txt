--- Description ---
The official package containing the kernel modules of the Lustre client for RHEL 9.2 is named kmod-lustre-client-2.15.3-1.el9.x86_64.rpm, so in case a rebuild is needed against a newer kernel version, the name would stay the same and the newer package wouldn't be an upgrade to the older version. 

Instead, I propose the release number to be appended with the kernel version like it is done with the openafs package:

kmod-openafs-1.8.10-1.5.14.0_284.18.1.el9_2.x86_64.rpm
