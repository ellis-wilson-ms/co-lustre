--- Description ---
When we want to change MGS node, we need change mgsnode param for all OST devices, and "tunefs.lustre --mgsnode [device]" will only append this MGS node nid to existing ones. I think it should replace the old ones, the multiple --mgsnode would be using append mode only, like "tunefs.lustre --mgsnode nid1 --mgsnode nid2 [device]" will replace all old mgsnode param on the device, and save these two new MGS nids to it.
