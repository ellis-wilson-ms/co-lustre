--- Description ---
The MDT will return st_blocks=1 for a HSM released file.

The LOV layer should also return st_blocks=1 for a HSM released file in the call ->coo_attr_get.

Otherwise this caused tools like tar then to consider the file as fully sparse and to archive it as is without attempting to acess/restore its content.
