--- Description ---
Some connection flags MDSs send to OSSs are, at least, unnecessary.  In certain cases, flag bits could be wasted if the corresponding features are not supported by MDSs, since future OSSs may not be able to use the flags bits to reliably determine if an MDS supports the features.  This issue was discovered in ORI-22.  A list of candidate flags to remove from the connection data can be found in this comment.
