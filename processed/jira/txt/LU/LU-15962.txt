--- Description ---
After building Lustre with in-kernel OFED, installing ko2iblnd module hit the following errors:

ko2iblnd: disagrees about version of symbol __ib_alloc_pd
ko2iblnd: Unknown symbol __ib_alloc_pd (err -22)
ko2iblnd: disagrees about version of symbol rdma_resolve_addr
ko2iblnd: Unknown symbol rdma_resolve_addr (err -22)

This is because the in-kernel Module.symvers (which contains the exported symbols) was not added to the symbol path while building Lustre.
