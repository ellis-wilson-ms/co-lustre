--- Description ---
An admin command like "setquota" should be part of "lctl" (which is for Lustre  administrator configuration of filesystem-wide settings), rather than "lfs" (which is for end users and mostly targets individual files).

I would propose to add an "lctl setquota" command that just calls "lfs_setquota" internally and then change the documentation over to use "lctl setquota" for quota administration.
