--- Description ---
when moving a file into a directory with project inheritance flag, it can ignore the quota

enforcement and cause the project quota to be over limit.

# lfs quota -p 1000 -h /mnt/lustre
Disk quotas for prj prjid1000 (pid 1000):
     Filesystem    used  bquota  blimit  bgrace   files  iquota  ilimit  igrace 
    /mnt/lustre    297M      0k    300M       -       5       0       0       - 
# lfs project /mnt/lustre/
 1000 P /mnt/lustre//tdir
# lfs project /mnt/lustre/tdir
 1000 - /mnt/lustre/tdir/tfile3
 1000 - /mnt/lustre/tdir/tfile2
 1000 - /mnt/lustre/tdir/tfile4
 1000 - /mnt/lustre/tdir/tfile1

# dd if=/dev/zero of=/mnt/lustre/tfile bs=1M count=100
100+0 records in
100+0 records out
104857600 bytes (105 MB, 100 MiB) copied, 0.333245 s, 315 MB/s

# mv /mnt/lustre/tfile /mnt/lustre/tdir/
# lfs project /mnt/lustre/tdir/
 1000 - /mnt/lustre/tdir//tfile3
 1000 - /mnt/lustre/tdir//tfile2
 1000 - /mnt/lustre/tdir//tfile
 1000 - /mnt/lustre/tdir//tfile4
 1000 - /mnt/lustre/tdir//tfile1
# lfs quota -p 1000 -h /mnt/lustre
Disk quotas for prj prjid1000 (pid 1000):
     Filesystem    used  bquota  blimit  bgrace   files  iquota  ilimit  igrace 
    /mnt/lustre   394M*      0k    300M       -      6*       0       0       -
