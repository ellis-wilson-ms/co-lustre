--- Description ---
This code allows "running" to be bumped once for each

thread that enters "lock()" before "bumped_running"

gets set to 1.  This has been simplified from code in obd.c:

static int bumped_running;

lock();

if (!bumped_running)

    running++;

unlock();

bumped_running = 1;
