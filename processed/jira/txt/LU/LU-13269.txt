--- Description ---
lnet-selftest.sh adds both LST tests: from client to server and from server to client, but sometimes we are interesting to execute the tests only from server or only from client.

Patch allows to set required "from" parameter via lst_FROM variable.

The following values are used:

lst_FROM=c : add the tests from clients to server only

lst_FROM=s : add the tests from server to clients only
