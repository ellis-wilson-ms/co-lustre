--- Description ---
Update Lustre e2fsprogs to 1.46.6. 

It turns out that the RHEL8 anaconda is unhappy with the use of 4-digit RPM version 1.46.5.wc1-0.el8, so instead we need to change the versioning to use 1.46.6-wc1.el8.

This change should be handled properly by RPM itself:

# rpmdev-vercmp 1.46.5.wc1-0.el8 1.46.6-wc1.el8 
1.46.5.wc1-0.el8
