Add options to 'lfs project' to lookup the file/directory project id and name in /etc/projid only, without performing any filesystem action.

This is useful for test scripts to confirm that "lfs project" itself is looking up the projid/projname correctly in the /etc/projid file, as well as end-user scripts for provisioning, etc.
