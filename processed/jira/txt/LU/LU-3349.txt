--- Comment (adilger, Fri, 26 Jul 2013 21:24:16 +0000) ---
Keith, the whole point of these test images is to allow testing that Lustre version X can mount and use filesystems that were formatted with Lustre version X-1, X-2, etc.  The only way to test such an upgrade is to actually format the filesystem with that earlier version of Lustre, and mount it with the current version of Lustre.  Doing that at runtime would be very complex, while creating static filesystem images with version X-1, X-2, (or version X for use in the future with X+1) is relatively easy.
