--- Description ---
There are a number of issues with running test scripts on a single-node system.

Several minor cleanups to the test scripts are also possible (coding style, more clear output, etc):

	sanity.sh test_51ba() should clean up subdirectories even if test_51b() failed

	sanity.sh test_61ic() shouldn't print all files to stdout

Implement acceptance-small.sh via auster, so that we don't have to maintain two separate test wrappers, and it would be possible for users running acc-sm to upload results to Maloo.
