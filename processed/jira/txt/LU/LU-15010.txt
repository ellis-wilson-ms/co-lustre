--- Description ---
Implement the grant shrink mechanism for MDC->MDT connections, in the same manner as OSC->OST connections. There is already an existing OBD_CONNECT_GRANT_SHRINK flag, so this could be added without affecting interoperability.

It would be desirable to include this into 2.15.0 if it is not too complex.
