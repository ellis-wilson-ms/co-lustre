--- Description ---
Similar to LU-11643, we need to create upgrade test images for conf-sanity test_32a/b/c/d so that we are sure upgrades from those previous versions are working properly.

They should contain files with features added in those releases (e.g. PFL/FLR files, LSOM, Project IDs/Project Quotas, overstriped files, OST pool quotas, ldiskfs large_dir, encrypted directory, CRUSH directory hash type, etc.).  They should also be created with multiple MDTs and OSTs to better match normal usage.
