--- Description ---
This is a POC test for BYD.

Our customer BYD reported an issue that repeated deletion of same name directory failed: "cannot remove 'dirname': File exists".

as shown below:

root@cn02:/aifs/trash_test# cd trash_test_1
root@cn02:/aifs/trash_test/trash_test_1# mkdir dir0 ; rmdir dir0
root@cn02:/aifs/trash_test/trash_test_1# mkdir dir0 ; rmdir dir0
rmdir: failed to remove 'dir0': File exists
