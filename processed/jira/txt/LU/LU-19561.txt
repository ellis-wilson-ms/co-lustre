--- Description ---
Provide a Rust interface for lustreapi and lnetconfig to enable using Rust to develop Lustre user space tools.  This is not adding support for writing kernel-based Lustre code in Rust.

An initial library has already been started outside of the Lustre tree, but for long term support, it should be move into the Lustre tree to ensure it stays current with changes to Lustre apis.
