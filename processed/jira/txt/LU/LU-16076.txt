--- Description ---
Add an optional argument for lfs check command to specify lustre file system to be checked.

 

As a reference, lfs osts shows all the OSTs for all mounted lustre file system, and lfs ost /mnt/lustre shows only the OSTs of /mnt/lustre.

So will enhance command lfs check servers /mnt/lustre to show only the servers of /mnt/lustre. lfs check servers shows all the servers of mounted lustre file system as before.
