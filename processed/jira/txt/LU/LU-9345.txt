--- Description ---
In test_29d there is:

lfs hsm_remove -a 0 $fid

But to use a FID (required when the file has been deleted from Lustre like in test_29d), one needs to use the --mntpath option.

Consequently the test fails with:

sanity-hsm test_29d: @@@@@@ FAIL: remove not broadcasted to all CTs
