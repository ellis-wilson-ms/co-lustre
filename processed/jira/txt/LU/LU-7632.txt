--- Description ---
This update fixes the following bug:

	Due to a regression, the NUMA node was not reported for PCI adapters, which

affected every IBM Power system deployed with Red Hat Enterprise Linux 7 and

caused significant decrease in the system performance. The underlying source

code has been patched, and the system performance no longer suffers in this

situation. (BZ#1283526)
