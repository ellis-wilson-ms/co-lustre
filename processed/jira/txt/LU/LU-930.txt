--- Description ---
This Jira issue can be used as the tracker for any trivial changes to a Lustre man page, or code documentation/comments, instead of having to file a separate ticket when making simple changes to the documentation that is not associated with another ticket. 

When pushing patches to Gerrit use LU-930 as the Jira ticket number in the summary line, and ensure that your commit comment fully describes the man page change.  In many cases the commit comment may be the new or revised text of the man page itself.

Note that this tracker is exclusively to be used for additions or changes to the man pages, messages printed to the console or from userspace tools, or comments in the code.  It is not to be used to track changes that affect code functionality.
