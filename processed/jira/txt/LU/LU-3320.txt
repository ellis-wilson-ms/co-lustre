--- Description ---
There recent RHEL 6.4 kernel update from LU-2846 broke builds on RHEL 6.3.  That's because there is only one series file for 2.6-rhel6 and one of the new patches shouldn't be applied to RHEL 6.3.

The ldiskfs kernel patches have two series files, ldiskfs-2.6-rhel6.series and ldiskfs-2.6-rhel6.4.series.  If that solution would be acceptable for the ldiskfs patch sets, then hopefully the same approach ought to be acceptable for the lustre kernel patches.
