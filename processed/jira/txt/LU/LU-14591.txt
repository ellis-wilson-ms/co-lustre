--- Description ---
The "llite.*.dump_page_cache" output should be excluded by default from "lctl get_param -R '*'" output, since it can be very verbose (millions of lines), and is almost always not useful unless looking for specific issues related to the page cache.  It should only be shown if explicitly requested by name.
