--- Description ---
The following mirror commands exist:

lfs mirror create
lfs mirror extend
lfs mirror delete
lfs mirror split
lfs mirror resync
lfs mirror verify
lfs mirror copy
lfs mirror read
lfs mirror write 

From this list, I believe only mirror extend will need special handling - we want to be able to add the immediate flag on a mirror extend invocation.  mirror create will get support for the flag as part of setstripe, and none of the others should need support.
