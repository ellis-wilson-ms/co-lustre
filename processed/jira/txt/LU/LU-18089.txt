--- Description ---
It would be useful to allow NRS TBF rules to be specified based on the nodemap name that a client is mapped into. This will simplify configuration of QOS for multiple tenants in a single filesystem. 

However, as a stopgap measure it is also possible to use UID, PROJID, or NID as a proxy to identify RPCs arriving from a specific tenant, if they are mapped uniquely for each tenant.
