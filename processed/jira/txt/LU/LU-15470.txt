--- Description ---
With the original mount looking like below

# mount | grep lustre
192.168.50.95@tcp:/lustre on /mnt/lustre type lustre (rw,checksum,flock,user_xattr,lruresize,lazystatfs,nouser_fid2path,verbose,noencrypt)

Umount and re-mount with no-checksum flag

 

# umount /mnt/lustre
# mount.lustre 192.168.50.95@tcp:/lustre /mnt/lustre/ -o nochecksum

The result is the changed "user_xattr" -> "nouser_xattr"

# mount | grep lustre
192.168.50.95@tcp:/lustre on /mnt/lustre type lustre (rw,nochecksum,flock,nouser_xattr,lruresize,lazystatfs,nouser_fid2path,verbose,noencrypt)
