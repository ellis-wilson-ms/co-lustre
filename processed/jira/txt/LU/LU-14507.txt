--- Description ---
When running "lfs getdirstripe -D" on a directory that has a default stripe "lfs setdirstripe -D -c -1" it prints the stripe count as an unsigned value:

# mkdir /mnt/testfs/dir
# lfs setdirstripe -D -c -1 /mnt/testfs/dir
# lfs getdirstripe -D /mnt/testfs/dir
lmv_stripe_count: 4294967295 lmv_stripe_offset: -1 lmv_hash_type: fnv_1a_64
