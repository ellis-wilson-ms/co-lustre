--- Environment ---
Development -- the commit validation script disallows "Reviewed-By:" tags at the end of the commit message

--- Description ---
The code needs to be extended to add these tags:

+        readonly INNOCUOUS='^('$(

+                echo \

+                        Acked-by \

+                        Tested-by \

+                        CC \

+                        Reported-by \

+                        Reviewed-by \

+                | sed 's/ /|/g'

+        )'):'

and to be able to add more in a trivial fashion.

Adding to this echo list should be trivial enough.

I have issue with commit-msg script on my MacOS.

It appears MacOS uses BSD version of fmt so there is no --split-only option:

http://www.freebsd.org/cgi/man.cgi?query=fmt&sektion=1

but linux has it: 

http://linux.die.net/man/1/fmt

So I have created this patch to see errors from commit-msg on MacOS:

http://review.whamcloud.com/2192
