--- Description ---
Since "lfs_migrate" is only copying a single file at a time, there is no real benefit from using "rsync" as the only copytool.  Allow the copytool to be replaced by using the RSYNC and RSYNC_OPTS environment variables to be specified, and add a short "--cp" option to complement the "--rsync" option with suitable options.
