--- Description ---
Explore the possibility for l_getauth to start the lsvcgssd daemon if it's not running yet. For SSK, daemon startup time is much longer than with Kerberos, because of sk_speedtest_dh_valid(). So to avoid hitting the upcall timeout, we could use predefined default values to save time in this particular case.
