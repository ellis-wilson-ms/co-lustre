--- Description ---
Please document the options for "lfs pcc add" parameters in a new "lfs-pcc-add.1" man page.  According to the "lfs-pcc.1" man page there are multiple options for the parameters:

For RW-PCC, when a file is being created, a rule-based policy is used to determine               whether it will be cached.  The rule expression supports logical conditional conjunction and disjunction operations among different users, groups, projects, or filenames  including  wildcards.   You  need  to specify auto create caching rule and archive ID in $PARAM.

but they are not actually documented what the option names are or what the syntax is.
