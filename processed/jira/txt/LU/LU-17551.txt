--- Description ---
There are a small number of places that use "ed" in autoMakefile.am and contrib/lbuild/lbuild that should be replaced with sed or some other mechanism (grep -v?) to reduce the build dependencies.

Also, quilt is definitely not needed for client builds since this is only used for ldiskfs, and barely needed for server builds.  It is more of a developer tool, and AFAIR the ldiskfs code can still be built without it.
