--- Description ---
IAM in ldiskfs-osd will always consume some slots in BH LRU (see: fs/buffer.c), if we keep buffer_head reference on root node, we can save one slot in BH LRU and could be somehow helpful for overall performance, I did some tests, LRU hits rate increased 5%-10% while creating files if we always keep this reference.
