--- Description ---
This is a request for improvement. It would be nice to support wildcard with UID rule when using the mixed policy "tbf". Wildcards are only supported with NRS JobID but not UID.

For example, starting a rule like that is not possible today:

start default_uid uid={*} rate=1000

This would allow to set a rpc rate limit per user with rate exemption on some NIDs (like DTNs).
