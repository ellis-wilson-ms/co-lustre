--- Description ---
It would be useful to have a utility to modify the name of an existing filesystem, though the filesystem would likely have to be unmounted from the clients/servers at the time.

This would require changes to:

	change labels of the backing filesystems (MDT/OST)

	change fsname and svname in mountdata (ldiskfs, or setting the equivalent ZFS dataset properties

	rename the config log for this filesystem in /CONFIGS/
{fsname}
-config, rewrite contents with new MDT/OST names on MGS, MDT, OST
