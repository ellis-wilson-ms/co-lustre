--- Comment (adilger, Fri, 30 Aug 2013 17:05:20 +0000) ---
We don't actually have OBD_CONNECT_GRANT_SHRINK enabled on the clients in our version of Lustre.

I agree that it shouldn't be flooding the server with these requests - grant shrinking should only happen occasionally, like on a ping.  The default interval GRANT_SHRINK_INTERVAL is 1200s, but this would be sent from multiple clients if they are idle and not using the grant.  It should only be a one-time event.
