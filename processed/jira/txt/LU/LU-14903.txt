--- Description ---
In LU-13417 a default LMV is set on ROOT to mkdir by space usage. But if user doesn't want this, he/she can't just delete it by "lfs setdirstripe -d..." because it will be added automatically if it's not set. Instead user should run "lfs setdirstripe -D -i-1 -c1 --max-inherit 1 " to limit it under root directory.
