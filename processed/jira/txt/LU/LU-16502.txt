--- Comment (simmonsja, Tue, 24 Feb 2026 15:53:42 +0000) ---
I was looking at this the other day and I think the issue is the python version. LUTF was originally written to work with both python2 and python3.  Some platforms like RHEL8 that I have only have python3. The python detection code doesn't work quite right so LUTF ends up not being built. Try 

export PYTHON_VERSION=3

and see if it builds.
