--- Description ---
The rpms for this build http://newbuild.whamcloud.com/job/lustre-reviews/1180/build_type=client,distro=el5,arch=x86_64,ib_stack=inkernel/? and the server version do not contain the kernel dependancy information as part of the RPM.

I.e. the kernel rpm does not say which kernel it provides, and the client modules does not say which kernel is requires.

It may well be that other information is also missing, but this information certainly is. I've also not check to see if this is a per branch or build thing.
