--- Description ---
Lustre specfile included in Lustre source code specify Lustre licensing information, GPL.

Lustre is licensed partially under GPLv2 and partially under LGPLv2.1 for liblustreapi.

 

The specfile should be adapted to include both license files and both license names.

Following Fedora Packaging Guidelines, the license should be set as

GPLv2 and LGPLv2.1

The LGPLv2.1 license file is available in lustre/utils directory but should be included as %doc likewise the GPL license file.

 

If the RPM specfile authorizes that, only the part containing the liblustreapi should have the 2 licences. If not possible, better to set that globally.
