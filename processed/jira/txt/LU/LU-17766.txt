Things to check:

	almost all new patches should be submitted to fs/lustre-release on the master branch. Backporting to maintenance release will depend on the nature of the patch, but generally a cleanup like this that is only fixing internal comments and test scripts would not be backported to a release branch

	check that you have the Git commit hooks installed, so that your patch has a "Change-Id: line and the commit message format follow the correct style.

# cd .git/hooks
# ln -sf ../../contrib/git-hooks/*commit-msg .

	check that you are pushing to port 29428 on review.whamcloud.com

	check that your SSH key is installed properly to your account. You should be able to run "ssh -p 29418 review.whamcloud.com test" and it Gerrit will reply "fatal: Gerrit Code Review: test: not found" if it is working and another error like "review.whamcloud.com: Permission denied (publickey)." if you are using the wrong username or your SSH key is not working.
