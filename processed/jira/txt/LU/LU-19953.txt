Patch https://review.whamcloud.com/c/fs/lustre-release/+/53169 in atomic_open()" class="issue-link" data-issue-key="LU-17295">LU-17295 avoids the VFS lock and enables parallel creates. It demonstrated about a 20× performance boost for file creation in a single shared directory.

It would be nice to have the same improvement for unlink operations in a shared directory.
