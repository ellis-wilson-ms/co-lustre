--- Description ---
To make the semantics more clear, when "--reset" or "-r" is specified when start the LFSCK manually, then all the non-specified parameters will be set as default values.

If the former LFSCK instance has already completed, then when re-trigger the LFSCK, it will restart from the beginning, and all the non-specified parameters will be set as default values.
