--- Description ---
Under LU-12387, the l_tunedisk utility was modified to recognize and skip tuning MDS and MGS devices. conf-sanity test_125 was added to make sure only OSS devices were tuned. These changes landed prior to the release of 2.12.3. Thus, conf-sanity test 125 should be skipped for all servers with version less than 2.12.3.

We’ve seen this test fail for 2.12.x client and 2.10.y server testing:
