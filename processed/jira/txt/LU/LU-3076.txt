Gentoo Linux

--- Description ---
Different udev versions has different path for rules installation. Especialy after udev was merged with systemd. But this path can be deduced from udev.pc

For example depending on udev version values could be 

/etc/udev/rules.d -> usualy old udev versions

/lib/udev/rules.d -> recent versions (>=199)

and 

/usr/lib/udev/rules.d > some versions in meantime (18x - 197)

--- Comment (adilger, Sat, 9 Oct 2021 06:13:22 +0000) ---
All modern distros have udev as part of systemd in /usr/lib/udev.
