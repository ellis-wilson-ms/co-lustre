--- Description ---
This ticket adds the following usages to extract data directly from last_rcvd or reply_data dump files:

# save raw file from ldiskfs filesystem
debugfs -c -R "dump reply_data /tmp/reply_data" /dev/mapper/mds1_flakey
debugfs -c -R "dump last_rcvd /tmp/last_rcvd" /dev/mapper/mds1_flakey

# read and decode the files
lr_reader -cr -C /tmp/last_rcvd -R /tmp/reply_data
...
