--- Description ---
When probing the file system on a client with lfs fid2path command, if the FID no longer exists the following error message is dropped in kernel log:

LustreError: 10725:0:(lmv_obd.c:854:lmv_iocontrol()) error: iocontrol MDC lustre-MDT0000_UUID on MDTidx 0 cmd c0086696: err = -2

Debug level appears to be set wrong for this operation.
