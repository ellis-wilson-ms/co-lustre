--- Description ---
It has come up several times recently that I wanted to disable lock expansion on all requests to explore possible shared file performance issues.  This is not currently possible without modifying application codes, which is not always an option.

This is primarily for testing, but it may increase performance in some situations.  It would be nice to include this in a standard Lustre distribution so it's easy to test on real hardware.
