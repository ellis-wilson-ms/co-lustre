--- Description ---
Autoconf 2.71 causes multiple deprecation warnings and some errors when used on Lustre.

OpenSUSE-Tumbleweed now uses 2.71 and so cannot build lustre.

 

autoconf provides a program "autoupdate" which can be used to up update configure.am and *.m4 files.  The result appears to still work with older autoconf.
