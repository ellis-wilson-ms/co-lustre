--- Description ---
Enable tracing of every outgoing rx and incoming tx message when net debug level is set. Add printing of IB connection's "CM id" and "QP num" to debug messages for easier matching to mlnx driver. Make printing of debug message on o2iblnd connection refcount increment/decrement events compile-time optional. Add compile-time option for dumping detailed connection state info to net debug log.
