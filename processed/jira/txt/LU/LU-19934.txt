--- Description ---
In kernel 6.17, the zstd and deflate compressors are using acomp API in their implementation.

zstd commit: https://github.com/torvalds/linux/commit/f5ad93ffb54119a8dc5e18f070624d4ead586969

deflate commit: https://github.com/torvalds/linux/commit/08cabc7d3c8638b078e0ac2f755cead1defafe91

We can adapt this on our compressors implementation along with some cleanup and improvements.

In addition we can get rid of `libcfs/include/libcfs/compr_private/scompress.h` and use the one exposed by the kernel `/include/crypto/internal/scompress.h` which will make things simpler in future changes.

https://github.com/torvalds/linux/commit/08cabc7d3c8638b078e0ac2f755cead1defafe91
