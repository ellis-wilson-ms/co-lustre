--- Description ---
module-assistant 0.11.11 attempts to extract the compiler (gcc-12) that was used to build the kernel, however it seems to think the compiler is 'gcc-12 make' which oddly doesn't work very well.

Explicitly set CC and RELAX_CC_CHECK when calling m-a (module-assistant) to avoid the incorrect guessing.
