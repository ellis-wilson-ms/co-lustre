--- Description ---
We should remove the obsolete meta packages, deps-sles and deps-rhel, from the lustre spec file.  That approach nine years old, and the code to enable those packages has been commented out the entire time.  It is just dead code.

This is not an approach that we will want to pursue in the future either.  It would be better to either have multiple spec files or to use distro-specific macros to make various requirements conditional to specific systems.
