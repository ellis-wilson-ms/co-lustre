--- Description ---
Default values for nodemap.squash_uid/gid/projid are currently set to 99, to match user 'nobody'. But on newer systems, nobody has changed to 65534 and 99 no longer exists.

It is safe to use 65534 in all cases, as even on older systems it exists and corresponds to 'nfsnobody'.
