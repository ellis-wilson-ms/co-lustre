--- Description ---
We need make OST and MDT online addition testing automated for both ldiskfs and zfs

OST online addition test steps as follow:

1 mount lustre with one OST/MDT

2 set quota for user quota_usr(disk limit for OST addition and inode limit for MDT addtion)

3 use up the space or inodes to achieve quota limit

4 keep the system on, format and mount another OST/MDT with index +1

5 check if quota is enabled on the new OST/MDT

6 create new file/dir, make sure it is on the new OST/MDT

7 should expect quota exceed error

8 remove the files create on the first OST/MDT and write data to the second OST/MDT, should expect success
