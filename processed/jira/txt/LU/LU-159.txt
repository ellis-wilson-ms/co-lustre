--- Description ---
lustre-diagnostics is a nice tool MJMac wrote to quickly, easily and efficiently profile a lustre machine and collecting the results in a well defined format.

This tool currently supports attaching a machine profile to automatically to a Bugzilla bug id - but it cannot attach to Jira issues.

lustre-diagnostics should be improved to support Jira.

--- Comment (rhenwood, Mon, 24 Sep 2012 17:09:25 +0000) ---
It seems that it is not possible to disable download of attachments without modifying the code.

Downloading attachments is necessary as the browser needs to maintain common domain security principle. Currently, Jira insists on downloading an attachment, so the browser thinks the attached xml is actually on localhost - and because of common domain security, it won't apply the xsl.

I have found this page that proposed code to alter this behaviour:

https://confluence.atlassian.com/pages/viewpage.action?pageId=215484007
